# Security

## Reporting a vulnerability

If you discover a potential security issue in this project we ask that you notify AWS/Amazon Security via our [vulnerability reporting page](http://aws.amazon.com/security/vulnerability-reporting/). Please do **not** create a public GitHub issue.

## Security features of generated projects

Projects scaffolded by the Nx Plugin for AWS include security controls out of the box, including infrastructure security scanning with Checkov, container image scanning with Trivy, credential scanning with git-secrets, IAM (SigV4) authentication by default, and encryption in transit and at rest for vended infrastructure.

See the [Security guide](https://awslabs.github.io/nx-plugin-for-aws/guides/security/) for a complete overview.
