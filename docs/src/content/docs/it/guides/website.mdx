Il generatore `ts#website` crea una nuova applicazione web. Puoi scegliere quale framework frontend utilizzare tramite l'opzione `framework` — attualmente è supportato [React](https://react.dev/).

Per informazioni dettagliate sulla struttura del progetto generato, la configurazione e l'utilizzo, consulta la guida specifica del framework:

- <Link path="/guides/react-website">React Website</Link>

### Generare un Website
