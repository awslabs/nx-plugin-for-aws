Il generatore `connection` collega un'<Link path="guides/ts-smithy-api">API Smithy</Link> a un progetto <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link>, configurando lo sviluppo locale in modo che entrambi si avviino automaticamente insieme.

## Prerequisiti

Prima di utilizzare questo generatore, assicurati di avere:

1. Un progetto <Link path="guides/ts-smithy-api">Smithy TypeScript API</Link> (generato con `ts#api` usando `--framework=smithy`)
2. Un progetto <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link>

## Utilizzo

### Eseguire il Generatore

## Output del Generatore

Il generatore aggiorna il file `project.json` della tua API Smithy per aggiungere una dipendenza dal suo target `dev` al target `dev` del progetto DynamoDB. Nessun file sorgente viene modificato.

## Utilizzare DynamoDB nelle Operazioni

Importa le factory di entità dal pacchetto DynamoDB e utilizzale all'interno delle implementazioni delle tue operazioni:

```ts title="packages/api/src/operations/list-examples.ts"
import { createExampleEntity } from '@my-scope/my-table';
import {
  ListExamplesOperationInput,
  ListExamplesOperationOutput,
} from '../generated/ssdk/index.js';
import { ServiceContext } from '../context.js';

export const listExamples = async (
  _input: ListExamplesOperationInput,
  _ctx: ServiceContext,
): Promise<ListExamplesOperationOutput> => {
  const entity = await createExampleEntity();
  const result = await entity.scan.go();
  return { items: result.data };
};
```
