Il generatore `connection` collega un <Link path="guides/py-mcp-server">Python MCP Server</Link> a un progetto <Link path="guides/py-dynamodb">Python DynamoDB</Link>, configurando lo sviluppo locale in modo che entrambi si avviino automaticamente insieme.

## Prerequisiti

Prima di utilizzare questo generatore, assicurati di avere:

1. Un progetto <Link path="guides/py-mcp-server">`py#mcp-server`</Link>
2. Un progetto <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Utilizzo

### Eseguire il Generatore

<RunGenerator generator="connection" />

Seleziona il tuo progetto MCP server come sorgente e il tuo progetto DynamoDB come destinazione. Se il progetto contiene più componenti MCP server, specifica `sourceComponent` per disambiguare.

### Opzioni

<GeneratorParameters generator="connection" />

## Output del Generatore

Il generatore aggiorna il target `<mcp-server-name>-serve-local` del MCP server in `project.json` per dipendere dal target `serve-local` del progetto DynamoDB. Nessun file sorgente viene modificato.
