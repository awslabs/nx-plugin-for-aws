Il generatore `connection` collega una <Link path="guides/fastapi">FastAPI</Link> a un progetto <Link path="guides/py-dynamodb">Python DynamoDB</Link>, configurando lo sviluppo locale in modo che entrambi si avviino automaticamente insieme.

## Prerequisiti

Prima di utilizzare questo generatore, assicurati di avere:

1. Un progetto FastAPI (generato con `py#api --framework=fastapi`), vedi la guida <Link path="guides/fastapi">`py#api`</Link>
2. Un progetto <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Utilizzo

### Esegui il Generatore

## Output del Generatore

Il generatore aggiorna il `project.json` della FastAPI per aggiungere una dipendenza dal suo target `dev` al target `dev` del progetto DynamoDB, e aggiunge il pacchetto DynamoDB come dipendenza del workspace. Nessun file sorgente viene modificato.

## Utilizzo di DynamoDB nei Route Handler

Importa le classi entity dal pacchetto DynamoDB e usale all'interno dei tuoi route handler:

```python title="packages/my_api/my_api/api.py"
from my_scope.my_table.entities.example import ExampleModel

@app.get("/examples")
def list_examples():
    return list(ExampleModel.scan())
```
