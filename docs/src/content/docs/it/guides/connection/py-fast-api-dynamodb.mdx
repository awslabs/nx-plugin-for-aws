Il generatore `connection` collega una <Link path="guides/fastapi">FastAPI</Link> a un progetto <Link path="guides/py-dynamodb">Python DynamoDB</Link>, configurando lo sviluppo locale in modo che entrambi si avviino automaticamente insieme.

## Prerequisiti

Prima di utilizzare questo generatore, assicurati di avere:

1. Un progetto <Link path="guides/fastapi">`py#fast-api`</Link>
2. Un progetto <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Utilizzo

### Esegui il Generatore

<RunGenerator generator="connection" />

Seleziona il tuo progetto FastAPI come sorgente e il tuo progetto DynamoDB come destinazione.

### Opzioni

<GeneratorParameters generator="connection" />

## Output del Generatore

Il generatore aggiorna il `project.json` della FastAPI per aggiungere una dipendenza dal suo target `serve-local` al target `serve-local` del progetto DynamoDB. Nessun file sorgente viene modificato.
