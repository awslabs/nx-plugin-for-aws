Il generatore `connection` collega un <Link path="guides/py-agent">Python Agent</Link> a un progetto <Link path="guides/py-dynamodb">Python DynamoDB</Link>, configurando lo sviluppo locale in modo che entrambi si avviino automaticamente insieme.

## Prerequisiti

Prima di utilizzare questo generatore, assicurati di avere:

1. Un progetto <Link path="guides/py-agent">`py#agent`</Link>
2. Un progetto <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Utilizzo

### Esegui il Generatore

<RunGenerator generator="connection" />

Seleziona il tuo progetto Agent come sorgente e il tuo progetto DynamoDB come destinazione. Se il progetto contiene più componenti agent, specifica `sourceComponent` per disambiguare.

## Output del Generatore

Il generatore aggiorna il target `<agent-name>-serve-local` dell'agent in `project.json` per dipendere dal target `serve-local` del progetto DynamoDB, e aggiunge il pacchetto DynamoDB come dipendenza del workspace. Nessun file sorgente viene modificato.

## Utilizzo di DynamoDB negli Agent

Importa le classi di entità dal pacchetto DynamoDB e usale all'interno dei tuoi strumenti agent:

```python title="packages/my_project/my_project/my_agent/agent.py"
from my_scope.my_table.entities.example import ExampleModel
from strands import tool

@tool
def list_examples() -> list:
    """List all example items."""
    return [item.attribute_values for item in ExampleModel.scan()]
```

## Infrastruttura

Per consentire alla funzione Lambda dell'agent di accedere alla tabella DynamoDB, concedi le autorizzazioni necessarie nella tua infrastruttura.

<Infrastructure>
<Fragment slot="cdk">

```ts title="packages/infra/src/stacks/application-stack.ts"
import { MyTable } from ':my-scope/common-constructs';

const table = new MyTable(this, 'Table');
const myAgent = new MyAgent(this, 'MyAgent');

table.grantReadWriteData(myAgent);
```

`grantReadWriteData` concede sia le autorizzazioni DynamoDB che KMS al ruolo di esecuzione dell'agent.
</Fragment>
<Fragment slot="terraform">

```hcl title="packages/infra/src/main.tf"
module "my_table" {
  source = "../../common/terraform/src/app/dynamodb/my-table"
}

resource "aws_iam_role_policy" "dynamodb_access" {
  role = module.my_agent.lambda_role_name

  policy = jsonencode({
    Version = "2012-10-17"
    Statement = [
      {
        Effect = "Allow"
        Action = [
          "dynamodb:GetItem", "dynamodb:PutItem", "dynamodb:UpdateItem",
          "dynamodb:DeleteItem", "dynamodb:Query", "dynamodb:Scan",
          "dynamodb:BatchGetItem", "dynamodb:BatchWriteItem",
        ]
        Resource = [
          module.my_table.table_arn,
          "${module.my_table.table_arn}/index/*",
        ]
      },
      {
        Effect = "Allow"
        Action = [
          "kms:Encrypt",
          "kms:Decrypt",
          "kms:ReEncrypt*",
          "kms:GenerateDataKey*",
          "kms:DescribeKey"
        ]
        Resource = [module.my_table.kms_key_arn]
      },
    ]
  })
}
```
