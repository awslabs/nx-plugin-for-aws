Il generatore TypeScript API crea un nuovo progetto API con il framework di tua scelta. Supporta [tRPC](https://trpc.io/) per API type-safe end-to-end e [Smithy](https://smithy.io/) per lo sviluppo di API model-driven.

## Utilizzo

### Generare un'API TypeScript
