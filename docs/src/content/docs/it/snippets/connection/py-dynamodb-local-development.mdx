---
title: Sviluppo Locale Python DynamoDB
---

Il generatore `connection` configura il target `serve-local` del tuo progetto per dipendere dal target `serve-local` del progetto DynamoDB. DynamoDB Local si avvierà automaticamente insieme al tuo progetto quando esegui `serve-local`.

La variabile d'ambiente `SERVE_LOCAL=true` viene impostata automaticamente, quindi `is_local()` restituisce `True` e le tue entità PynamoDB si connettono all'istanza DynamoDB locale invece che ad AWS.
