import NxCommands from '@components/nx-commands.astro';

Il generatore configura automaticamente un target `bundle` che utilizza [Rolldown](https://rolldown.rs/) per creare un pacchetto di distribuzione:

<NxCommands commands={['bundle <project-name>']} />

La configurazione di Rolldown si trova in `rolldown.config.ts`, con un'entry per bundle da generare. Rolldown gestisce la creazione di più bundle in parallelo se definiti.

:::tip[Custom Configuration]
Puoi modificare la configurazione di Rolldown in `rolldown.config.ts`, ad esempio se hai dipendenze che richiedono percorsi file specifici e non possono essere incluse nel bundle, puoi aggiungerle alla lista dei moduli `external`.
:::
