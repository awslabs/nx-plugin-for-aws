#### WAF

Per le REST API, il costrutto CDK generato associa un AWS WAFv2 Web ACL allo stage predefinito dell'API per impostazione predefinita. Il Web ACL utilizza il set di regole predefinito gestito da AWS (`AWSManagedRulesCommonRuleSet` e `AWSManagedRulesKnownBadInputsRuleSet`), fornendo protezione contro exploit web comuni inclusi gli OWASP Top 10.

Per disattivarlo (ad esempio, per collegare il proprio Web ACL), impostare `enableWaf` su `false`:

```ts {4}
const api = new MyApi(this, 'MyApi', {
  integrations: MyApi.defaultIntegrations(this).build(),
  enableWaf: false,
});
```
