title: "Scegliere tra REST e HTTP"
---



:::note
Puoi scegliere tra una [REST API](https://docs.aws.amazon.com/apigateway/latest/developerguide/apigateway-rest-api.html) o una [HTTP API](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api.html) per distribuire la tua API. Fai riferimento alla [API Gateway Developer Guide](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api-vs-rest.html) per aiutarti a decidere.
:::
