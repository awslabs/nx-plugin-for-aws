#### WAF

Per le REST API, il modulo Terraform generato associa un AWS WAFv2 Web ACL allo stage dell'API Gateway per impostazione predefinita. Il Web ACL utilizza il set di regole predefinito gestito da AWS (`AWSManagedRulesCommonRuleSet` e `AWSManagedRulesKnownBadInputsRuleSet`), fornendo protezione contro exploit web comuni inclusi gli OWASP Top 10.

Per disattivarlo (ad esempio, per collegare il proprio Web ACL), impostare `enable_waf` su `false`:

```hcl {4}
module "my_api" {
  source = "../../common/terraform/src/app/apis/my-api"

  enable_waf = false
}
```
