---
title: "Prerequisiti"
---



- Installa [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git).
- Installa [Node >= 22](https://nodejs.org/en/download) (Consigliamo di utilizzare [NVM](https://github.com/nvm-sh/nvm) per gestire le versioni di Node)
  - Per verificare la versione, esegui `node --version` nella finestra del terminale.
- Installa [PNPM >= 10](https://pnpm.io/installation#using-npm) (Puoi anche usare [Yarn >= 4](https://yarnpkg.com/getting-started/install), [Bun >= 1](https://bun.sh/docs/installation) o [NPM >= 10](https://nodejs.org/en/learn/getting-started/an-introduction-to-the-npm-package-manager))
  - Per verificare la versione, esegui `pnpm --version`, `yarn --version`, `bun --version` o `npm --version` nel terminale.
- Installa [UV >= 0.5.29](https://docs.astral.sh/uv/getting-started/installation/)
  1. Per installare Python 3.12, esegui `uv python install 3.12.0`.
  2. Verifica con `uv python list --only-installed`
- Configura le tue [AWS Credentials](https://docs.aws.amazon.com/sdkref/latest/guide/access.html) per l'account AWS di destinazione (dove verr√† distribuita la tua applicazione).
- Se utilizzi [VSCode](https://code.visualstudio.com/), consigliamo di installare il [Nx Console VSCode Plugin](https://marketplace.visualstudio.com/items?itemName=nrwl.angular-console).
