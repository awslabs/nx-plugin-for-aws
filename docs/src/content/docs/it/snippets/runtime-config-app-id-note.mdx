:::note[Configurazione runtime]
Questo si basa sulla variabile d'ambiente `RUNTIME_CONFIG_APP_ID` per recuperare la configurazione da AWS AppConfig a runtime. I progetti costruiti con questo plugin (API, Agents, server MCP) hanno già questa variabile configurata automaticamente. Per altri progetti, assicurati che `RUNTIME_CONFIG_APP_ID` sia impostata nell'ambiente runtime con l'ID applicazione AppConfig fornito dalla tua infrastruttura. Per ulteriori informazioni, consulta la <Link path="guides/runtime-config">guida alla Configurazione Runtime</Link>.
:::
