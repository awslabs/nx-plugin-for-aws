[AWS Arch](https://aws.github.io/aws-pdk/developer_guides/aws-arch/index.html) forniva le mappature tra le risorse di CloudFormation e le relative icone architetturali associate per CDK Graph sopra menzionato.

Fai riferimento alla [pagina delle icone architetturali AWS](https://aws.amazon.com/architecture/icons/) per le risorse relative alle icone. [Diagrams](https://diagrams.mingrammer.com/) fornisce anche un modo per creare diagrammi come codice.

Se stavi utilizzando direttamente questo progetto, valuta di effettuare un fork del progetto e prendertene carico!
