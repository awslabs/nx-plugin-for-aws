[PDK Nag](https://aws.github.io/aws-pdk/developer_guides/pdk-nag/index.html) incapsula [CDK Nag](https://github.com/cdklabs/cdk-nag) e fornisce una serie di [regole specifiche per la creazione di prototipi](https://github.com/aws/aws-pdk/blob/mainline/packages/pdk-nag/src/packs/aws-prototyping-rules.ts).

Per migrare da PDK Nag, utilizza direttamente CDK Nag. Se hai bisogno dello stesso set di regole, puoi creare un tuo "pack" personale seguendo la [documentazione qui disponibile](https://github.com/cdklabs/cdk-nag/blob/main/docs/NagPack.md).
