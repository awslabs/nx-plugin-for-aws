### File di Configurazione

La maggior parte degli assistenti AI che supportano MCP utilizza un approccio di configurazione simile. Dovrai creare o aggiornare un file di configurazione con i dettagli del tuo server MCP:

```json
{
  "mcpServers": {
    "your-mcp-server": {
      "command": "npx",
      "args": ["tsx", "/path/to/your-mcp-server/stdio.ts"]
    }
  }
}
```

:::caution
Se ricevi un errore come `ENOENT npx` durante la connessione al server, potresti dover specificare il percorso completo di `npx`, che puoi ottenere eseguendo `which npx` nel tuo terminale.
:::

### Ricarica a Caldo

Durante lo sviluppo del tuo server MCP, potresti voler configurare il flag `--watch` in modo che l'assistente AI visualizzi sempre le versioni pi√π recenti di strumenti/risorse:

```json {5}
{
  "mcpServers": {
    "your-mcp-server": {
      "command": "npx",
      "args": ["tsx", "--watch", "/path/to/your-mcp-server/stdio.ts"]
    }
  }
}
```

:::note
Se aggiungi nuovi strumenti o risorse, potresti dover aggiornare il server MCP nell'assistente AI.
:::
