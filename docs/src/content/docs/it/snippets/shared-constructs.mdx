Poiché questo generatore fornisce infrastruttura come codice basata sul tuo `iacProvider` selezionato, creerà un progetto in `packages/common` che include i relativi costrutti CDK o moduli Terraform.

Il progetto comune di infrastruttura come codice è strutturato come segue:

<Infrastructure>
<Fragment slot="cdk">
<FileTree>
  - packages/common/constructs
    - src
      - app/ Construct per l'infrastruttura specifica di un progetto/generatore
      - core/ Construct generici riutilizzati dai construct in `app`
      - index.ts Punto di ingresso che esporta i construct da `app`
    - project.json Target di build e configurazione del progetto
</FileTree>

:::note
Questo progetto è generato utilizzando il <Link path="/guides/typescript-project">`ts#project`</Link> generator e quindi configura gli stessi target di build.
:::
</Fragment>
<Fragment slot="terraform">
<FileTree>
  - packages/common/terraform
    - src
      - app/ Moduli Terraform per l'infrastruttura specifica di un progetto/generatore
      - core/ Moduli generici riutilizzati dai moduli in `app`
    - project.json Target di build e configurazione del progetto
</FileTree>

:::note
Questo progetto è generato utilizzando il <Link path="/guides/terraform-project">`terraform#project`</Link> generator e quindi configura gli stessi target di build.
:::
</Fragment>
</Infrastructure>
