## Passo 3: Utilizza i Generator per Scaffoldare il Tuo Progetto

Per questa guida rapida, aggiungeremo un'API tRPC, un sito web React, l'autenticazione Cognito e un'infrastruttura CDK al progetto.  
A seconda del tipo di progetto, puoi scegliere qualsiasi combinazione di generatori. Per l'elenco completo, consulta la sezione __Guide__ nella barra di navigazione a sinistra.

### Aggiungi un'API tRPC

<RunGenerator generator="ts#trpc-api" requiredParameters={{ apiName: 'demo-api' }} />

### Aggiungi un Sito Web React

<RunGenerator generator="ts#cloudscape-website" requiredParameters={{ name: 'demo-website' }} />

Questo scaffolda un nuovo sito web React in `packages/demo-website`.

### Aggiungi l'Autenticazione Cognito

<RunGenerator generator="ts#cloudscape-website#auth" requiredParameters={{ project: '@my-project/demo-website', cognitoDomain: 'my-demo' }} />

Configura l'infrastruttura necessaria e il codice React per aggiungere l'autenticazione Cognito al tuo sito web.

### Collega il Frontend al Backend

<RunGenerator generator="api-connection" requiredParameters={{ sourceProject: '@my-project/demo-website', targetProject: '@my-project/demo-api-backend' }} />

Configura i provider necessari per consentire al sito web di chiamare la tua API tRPC.

### Aggiungi l'Infrastruttura CDK

<RunGenerator generator="ts#infra" requiredParameters={{ name: 'infra' }} />

Configura un'App CDK che puoi utilizzare per distribuire la tua infrastruttura su AWS.

## Passo 4: Definisci le Risorse Cloud e Distribuisci su AWS

Apri `packages/infra/src/stacks/application-stack.ts` e aggiungi il seguente codice:

```typescript
import * as cdk from 'aws-cdk-lib';
import { DemoApi, DemoWebsite, UserIdentity } from ':my-project/common-constructs';
import { Construct } from 'constructs';

export class ApplicationStack extends cdk.Stack {
  constructor(scope: Construct, id: string, props?: cdk.StackProps) {
    super(scope, id, props);

    const identity = new UserIdentity(this, 'identity');
    const api = new DemoApi(this, 'api');
    api.grantInvokeAccess(identity.identityPool.authenticatedRole);

    new DemoWebsite(this, 'website');
  }
}
```
__Nota__: Questo Ã¨ tutto il codice CDK necessario per distribuire la nostra applicazione full-stack.

### Compila e Distribuisci l'Infrastruttura

Per compilare il tuo progetto, esegui il comando:

<NxCommands commands={['run-many --target build --all']} />

:::tip
Se incontri errori di lint, esegui questo comando per correggerli automaticamente:

<NxCommands commands={['run-many --target lint --configuration=fix --all']} />
:::

## Passo 5: Esegui il Frontend in Locale

<Steps>
1. Recupera il file `runtime-config.json`

    <NxCommands commands={['run @demo/demo-website:load:runtime-config']} />

2. Avvia il server web locale

    <NxCommands commands={['run @demo/demo-website:serve']} />
</Steps>

Puoi visualizzare il sito web all'indirizzo `http://localhost:4200` nel browser.
