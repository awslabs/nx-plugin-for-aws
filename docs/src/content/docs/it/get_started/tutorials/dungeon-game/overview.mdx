## Panoramica

Durante questo tutorial costruirai un gioco di avventura dungeon alimentato da IA. Questo tutorial non presuppone alcuna conoscenza pregressa di `@aws/nx-plugin` o tecnologie correlate. Le tecniche che apprenderai sono fondamentali per costruire qualsiasi applicazione basata su `@aws/nx-plugin`, e comprenderle a fondo ti darà una conoscenza approfondita del plugin e una comprensione di base sull'uso di [NX](https://nx.dev/).

<Aside>Questo tutorial è pensato per chi preferisce imparare facendo e vuole provare rapidamente a creare qualcosa di tangibile. Se preferisci apprendere ogni concetto passo dopo passo, consulta le singole guide dei componenti</Aside>

Al termine del tutorial, gli utenti acquisiranno le competenze per:

- utilizzare `@aws/nx-plugin` per creare nuove applicazioni
- usare NX per gestire/costruire il codice
- costruire API usando sia tRPC che FastAPI
- utilizzare Tanstack router per creare nuove pagine
- usare Tanstack query per chiamare API backend
- modificare il codice generato per adattarlo allo scopo
- creare e distribuire infrastrutture CDK

### Componenti dell'applicazione

L'architettura del gioco utilizza i seguenti componenti:

<Image class="white-bg" src={dungeonAdventureArchitecturePng} alt="dungeon-game-architecture.png" width="800" height="600" />

- API [tRPC](https://trpc.io/) che utilizza [ElectroDB](https://electrodb.dev/)/[DynamoDB](https://aws.amazon.com/dynamodb/) per gestire lo stato del gioco
- Funzione Lambda Python/[FastAPI](https://fastapi.tiangolo.com/) con risposte in streaming per la generazione della storia (alimentata da [Amazon Bedrock](https://aws.amazon.com/bedrock/))
- Frontend React/Vite che utilizza:
  - Amazon Cognito/Identity Pools per chiamate API sicure
  - [Tanstack router](https://tanstack.com/router/latest) con routing basato su file type-safe
  - SDK generati per chiamare le API di Gioco e Storia (con [Tanstack query](https://tanstack.com/query/latest))
- Infrastruttura [CDK](https://docs.aws.amazon.com/cdk/v2/guide/home.html) per il deployment
