---
title: "Metriche di Utilizzo"
description: "Come @aws/nx-plugin raccoglie le metriche"
---



Per aiutarci a migliorare il Plugin Nx per AWS, raccogliamo metriche sull'utilizzo dei generatori. Questo processo non comporta richieste di rete aggiuntive, ci limitiamo a sfruttare le metriche di utilizzo esistenti di AWS.

Per mantenere la trasparenza, aggiungiamo un ID, una versione e un elenco di ID dei generatori alle descrizioni degli Stack CloudFormation. Questa operazione viene eseguita nell'`MetricsAspect` in `common/constructs` per CDK o nello stack CloudFormation `metrics.tf` per Terraform.
