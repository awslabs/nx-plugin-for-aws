---
title: "Metriche di Utilizzo"
description: "Come @aws/nx-plugin raccoglie le metriche"
---



Per aiutarci a migliorare il Nx Plugin for AWS, raccogliamo metriche sull'utilizzo dei generatori. Questo processo non comporta richieste di rete aggiuntive, utilizziamo semplicemente le metriche di utilizzo esistenti di AWS.

Per garantire trasparenza, aggiungiamo un ID, una versione e un elenco di ID generatori alle descrizioni degli Stack CloudFormation. Questa operazione viene eseguita nel costrutto `MetricsAspect` presente in `common/constructs`.
