---
title: "Metriche di Utilizzo"
description: "Come @aws/nx-plugin raccoglie le metriche"
---



Per aiutarci a migliorare il Plugin Nx per AWS, raccogliamo metriche sull'utilizzo dei generatori. Questo processo non richiede richieste di rete aggiuntive, sfruttiamo semplicemente le metriche di utilizzo esistenti di AWS.

Per mantenere la trasparenza, aggiungiamo un ID, una versione e un elenco degli ID dei generatori alle descrizioni degli Stack CloudFormation. Questa operazione viene eseguita nello `MetricsAspect` in `common/constructs` per CDK o nello stack Cloudformation `metrics.tf` per Terraform.
