---
title: "Metriche di Utilizzo"
description: "Come @aws/nx-plugin raccoglie le metriche"
---



Per aiutarci a migliorare il Nx Plugin per AWS, raccogliamo metriche sull'utilizzo dei generatori. Questo processo non comporta richieste di rete aggiuntive, ma utilizza semplicemente le metriche di utilizzo esistenti di AWS.

Per garantire trasparenza, aggiungiamo un ID, una versione e un elenco di ID dei generatori alle descrizioni degli stack CloudFormation. Questa operazione viene eseguita tramite il `MetricsAspect` in `common/constructs`.
