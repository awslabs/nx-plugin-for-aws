## Nx Daemon

Il [Nx Daemon](https://nx.dev/docs/concepts/nx-daemon) è un processo che monitora continuamente le modifiche all'area di lavoro per ricalcolare il grafo dei progetti. Questo garantisce un miglioramento delle prestazioni, specialmente per progetti di grandi dimensioni, ma potrebbe occasionalmente causare problemi a seconda dell'ambiente.

Questo può manifestarsi in diversi modi, uno dei quali è il seguente errore:

```bash
 NX   Impossibile trovare il progetto "@scope/project"

Passa --verbose per visualizzare lo stacktrace.
```

Per questo e per qualsiasi altro errore imprevisto, è sempre utile resettare Nx e riprovare in caso di problemi con il demone. Puoi utilizzare il seguente comando:

<NxCommands commands={['reset']} />

Se il problema persiste, puoi provare a [disattivare completamente il demone](https://nx.dev/docs/concepts/nx-daemon#turning-it-off).

:::warning
Il comando `nx watch` richiede il demone, quindi alcuni target di hot-reloading potrebbero non funzionare se lo disattivi.
:::
