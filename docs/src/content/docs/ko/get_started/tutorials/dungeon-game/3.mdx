## 작업 1: Story 에이전트 구현

Story 에이전트는 <Link path="get_started/tutorials/dungeon-game/1">모듈 1</Link>에서 `--protocol=AG-UI`로 생성된 [Strands](https://strandsagents.com/) 에이전트로, UI가 CopilotKit을 통해 [Agent-User Interaction protocol](https://docs.copilotkit.ai/aws-strands/protocol)을 통해 스트리밍할 수 있습니다. 이 에이전트는 Inventory MCP Server를 사용하여 플레이어의 아이템을 관리하고, Strands의 내장 [`S3SessionManager`](https://strandsagents.com/latest/documentation/docs/user-guide/concepts/agents/sessions/)를 사용하여 대화 기록을 모듈 2에서 프로비저닝한 세션 버킷에 저장합니다.

### 에이전트 구현

`packages/story/dungeon_adventure_story/agent`의 다음 파일들을 업데이트하세요:

<Tabs>
  <TabItem label="main.py">
<E2EDiff lang="python" before="dungeon-adventure/3/main.py.old.template" after="dungeon-adventure/3/main.py.template" />
  </TabItem>
  <TabItem label="agent.py">
<E2EDiff lang="python" before="dungeon-adventure/3/agent.py.old.template" after="dungeon-adventure/3/agent.py.template" />
  </TabItem>
</Tabs>

변경 사항은 다음과 같습니다:

- `main.py`는 배포 시 `thread_id`당 `S3SessionManager`를 생성하는 `session_manager_provider`를 추가하고, `agent-serve-local`에서 실행할 때(`SERVE_LOCAL=true`) `/tmp/strands-sessions` 아래의 온디스크 `FileSessionManager`로 폴백합니다. 배포 시 S3 버킷은 Game API의 `queryActions`가 읽는 것과 동일한 버킷이므로 브라우저가 재방문 시 대화 기록을 재구성할 수 있습니다. 로컬에서는 에이전트가 세션을 디스크에 저장하고 배포 없이 로컬 MCP 서버와 통신합니다.
- `agent.py`는 샘플 `subtract` 도구를 제거하고 시스템 프롬프트를 던전 마스터 프롬프트로 교체합니다. 이 프롬프트는 첫 번째 사용자 메시지에서 플레이어의 이름과 장르를 명시하도록 유도하며, Inventory MCP Server의 도구를 사용합니다.

## 작업 2: 로컬에서 에이전트 테스트

코드를 빌드하려면:

<PackageManagerShortCommand commands={["lint"]} />
</Aside>

### 에이전트와 대화하기

생성된 `agent-chat` 타겟을 사용하여 로컬에서 제공되는 AG-UI 에이전트의 복사본에 대해 대화형 REPL을 엽니다:

<NxCommands commands={["agent-chat story"]} />

첫 번째 메시지는 에이전트에게 영웅의 이름과 장르를 알려야 합니다(예: `My name is Alice. Start my zombie adventure.`). 그러면 스토리가 스트리밍되어 돌아옵니다.

<Aside type="caution" title="AWS 자격 증명 필요">
에이전트가 로컬에서 실행되지만 LLM을 위해 [Amazon Bedrock](https://aws.amazon.com/bedrock/)을 호출합니다. [Strands 기본 모델](https://strandsagents.com/latest/documentation/docs/user-guide/concepts/model-providers/amazon-bedrock/)에 액세스할 수 있는 AWS 자격 증명이 환경에 필요합니다. 자세한 내용은 [Strands 모델 공급자 문서](https://strandsagents.com/latest/documentation/docs/user-guide/concepts/model-providers/amazon-bedrock/)를 참조하세요.
</Aside>
