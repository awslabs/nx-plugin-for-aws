## 마무리

축하합니다! `@aws/nx-plugin`에서 제공하는 다양한 생성기를 활용한 던전 어드벤처 게임을 성공적으로 만들었습니다. 🎉🎉🎉

## 다음 단계

다음 기능들을 직접 구현해보며 코드베이스를 확장해 보는 것을 추천합니다:

1. 프로젝트에 <Link path="guides/license">라이선스 생성기</Link>를 추가하여 LICENSE 파일과 소스 코드 헤더를 자동으로 관리합니다.
2. 새로운 tRPC API 추가를 단순화하기 위해 <Link path="get_started/tutorials/create-generator">사용자 정의 생성기</Link>를 개발합니다.
3. `GameAPI`에 새로운 `resetToAction` API를 생성(위에서 만든 생성기 활용)하여 특정 시점으로 되돌리고 이후 액션을 삭제할 수 있도록 합니다. UI를 업데이트하여 채팅 버블에 되돌리기 버튼을 추가하고 새 API를 호출합니다.
4. 다양한 가이드 문서를 탐색합니다

### 리소스 삭제

생성된 AWS 리소스를 삭제하려면 다음 명령어를 실행하세요:

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

이 명령은 삭제할 스택 목록(`waf` 및 `dungeon-adventure-infra-sandbox`)을 물어보면 `Y`를 입력하면 Cloudformation이 스택을 삭제합니다.
