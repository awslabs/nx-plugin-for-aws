## 마무리

축하합니다! `@aws/nx-plugin`에서 제공하는 다양한 제너레이터를 활용한 던전 어드벤처 게임을 성공적으로 생성했습니다. 🎉🎉🎉

## 다음 단계?

다음 기능들을 직접 구현해보며 코드베이스를 확장해 볼 것을 권장합니다:

1. 프로젝트에 <Link path="guides/license">라이선스 제너레이터</Link>를 추가하여 워크스페이스의 LICENSE 파일과 소스 코드 헤더를 자동으로 관리합니다.
2. 새로운 tRPC API 추가를 단순화하기 위해 <Link path="get_started/tutorials/contribute-generator">나만의 제너레이터를 개발</Link>합니다.
3. GameAPI에 새로운 `resetToAction` API를 생성(위에서 만든 제너레이터 사용)하여 특정 시점부터의 모든 액션을 삭제하고 시간을 되돌릴 수 있도록 합니다. 그런 다음 UI에 대화 버블에 버튼을 추가하여 이 기능을 호출합니다.
4. 제공되는 다양한 가이드를 탐색해 봅니다.

### 리소스 삭제

생성된 AWS 리소스를 삭제하려면 다음 명령어를 실행하세요:

<NxCommands commands={['run @dungeon-adventure/infra:destroy dungeon-adventure-infra-sandbox/*']} />

이 명령은 삭제할 스택 목록(`dungeon-adventure-infra-sandbox/Application/GameUI/waf` 및 `dungeon-adventure-infra-sandbox/Application`)을 물어보면 `Y`를 입력하면 Cloudformation이 스택을 삭제합니다.
