## 마무리

축하합니다! `@aws/nx-plugin`에서 제공하는 다양한 생성기를 활용한 에이전틱 AI 던전 어드벤처 게임을 성공적으로 생성했습니다. 🎉🎉🎉

## 다음 단계

다음 기능들을 직접 구현해보며 코드베이스를 확장해 볼 것을 권장합니다:

1. 프로젝트에 <Link path="guides/license">라이선스 생성기</Link>를 추가하여 작업 공간의 LICENSE 파일과 소스 코드 헤더를 자동으로 관리합니다.
2. 새로운 tRPC API 추가를 단순화하기 위해 <Link path="get_started/tutorials/contribute-generator">직접 생성기를 개발</Link>해 보세요.
3. GameAPI에 새로운 `resetToAction` API를 생성기로 추가한 후, 특정 시점부터의 모든 액션을 삭제하고 시간을 되돌릴 수 있도록 구현합니다. 이후 대화 버블에 버튼을 추가하여 이 API를 호출하는 기능으로 UI를 업데이트하세요.
4. 왼쪽 네비게이션을 사용하여 다양한 컴포넌트 가이드를 탐색해 보세요.

## 리소스 삭제

1. 생성된 AWS 리소스를 삭제하려면 다음 명령어를 실행하세요:

<NxCommands commands={['run @dungeon-adventure/infra:destroy dungeon-adventure-infra-sandbox/*']} />

    이 명령은 삭제할 스택 목록(`dungeon-adventure-infra-sandbox/Application/GameUI/waf` 및 `dungeon-adventure-infra-sandbox/Application`)을 표시합니다.

2. 계속하려면 `Y`를 입력하세요. CloudFormation이 해당 스택들을 삭제합니다.
