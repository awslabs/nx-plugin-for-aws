## 마무리

축하합니다! `@aws/nx-plugin` 내에서 찾을 수 있는 많은 생성기를 활용하는 던전 모험 게임을 만들었습니다! 🎉🎉🎉

## 다음 단계는?

다음과 같은 기능으로 코드베이스를 직접 확장해 보는 것을 추천합니다:

1. <Link path="guides/license">라이선스 생성기</Link>를 프로젝트에 추가하여 작업 공간의 LICENSE 파일 및 소스 코드 헤더를 자동으로 관리합니다.
2. 새로운 tRPC API 추가를 간소화하기 위해 <Link path="get_started/tutorials/create-generator">자신만의 생성기를 만듭니다</Link>.
3. GameAPI에서 특정 시점의 모든 액션을 삭제할 수 있는 새로운 `resetToAction` API를 (위에서 만든 생성기를 사용하여) 추가합니다. 그런 다음 대화 버블에 이 시점으로 되돌리는 버튼을 추가하고 새로 생성된 API를 호출합니다.
4. 다양한 가이드 탐색

### 리소스 삭제

생성된 AWS 리소스를 삭제하려면 다음 명령을 실행하세요:

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

이렇게 하면 삭제할 스택 목록을 묻는 메시지가 표시되며, `waf`와 `dungeon-adventure-infra-sandbox`가 포함되어야 합니다. `Y`를 입력하면 Cloudformation이 스택을 삭제합니다.
