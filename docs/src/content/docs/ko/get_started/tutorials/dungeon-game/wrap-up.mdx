## 마무리

축하합니다! 여러분은 `@aws/nx-plugin`에서 제공하는 다양한 생성기(generator)를 활용한 던전 어드벤처 게임을 성공적으로 생성했습니다. 🎉🎉🎉

## 다음 단계

다음 기능들을 직접 구현해보며 프로젝트를 확장해보는 것을 추천합니다:

1. <Link path="guides/license">라이선스 생성기</Link>를 프로젝트에 추가하여 작업 공간의 LICENSE 파일과 소스 코드 헤더를 자동으로 관리해보세요.
2. <Link path="get_started/tutorials/contribute-generator">나만의 생성기를 개발</Link>하여 새로운 tRPC API 추가 과정을 단순화해보세요.
3. 새로운 `resetToAction` API를 GameAPI에 추가하고(위에서 만든 생성기 사용), 특정 시점부터의 모든 액션을 삭제하며 시간을 되돌릴 수 있도록 UI에 버튼을 추가해보세요(새로 생성된 API 호출).
4. 제공되는 다양한 가이드 문서를 탐색해보세요

### 리소스 삭제

생성된 AWS 리소스를 삭제하려면 다음 명령어를 실행하세요:

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

이 명령은 `waf`와 `dungeon-adventure-infra-sandbox`로 구성된 삭제할 스택 목록을 입력하라는 메시지가 표시됩니다. `Y`를 입력하면 Cloudformation이 해당 스택들을 삭제합니다.
