## 마무리

축하합니다! `@aws/nx-plugin`에서 제공하는 다양한 제너레이터를 활용한 던전 어드벤처 게임을 성공적으로 생성했습니다. 🎉🎉🎉

## 다음 단계

다음 기능들을 직접 추가해보며 코드베이스를 확장해 볼 것을 권장합니다:

1. 프로젝트에 <Link path="guides/license">라이선스 제너레이터</Link>를 추가하여 LICENSE 파일과 소스 코드 헤더를 자동으로 관리
2. 새로운 tRPC API 추가를 간소화하기 위해 <Link path="get_started/tutorials/create-generator">나만의 제너레이터 생성</Link>
3. GameAPI에 새로운 `resetToAction` API 생성(위에서 만든 제너레이터 사용)하여 특정 시점으로 되돌아가 해당 시점 이후의 모든 액션 삭제 기능 추가. 이후 대화 버블에 버튼을 추가해 이 시점으로 리셋하는 기능 구현(새로 생성된 API 호출)
4. 제공되는 다양한 가이드 살펴보기

### 리소스 삭제

생성된 AWS 리소스를 삭제하려면 다음 명령어를 실행하세요:

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

이 명령은 삭제할 스택 목록(`waf`와 `dungeon-adventure-infra-sandbox`로 구성)을 확인하라는 프롬프트를 표시합니다. `Y`를 입력하면 Cloudformation이 해당 스택들을 삭제합니다.
