## 개요

이 튜토리얼에서는 AI 기반 던전 어드벤처 게임을 구축하게 됩니다. 본 튜토리얼은 `@aws/nx-plugin` 또는 관련 기술에 대한 사전 지식을 요구하지 않습니다. 여기서 학습할 기술들은 모든 `@aws/nx-plugin` 기반 애플리케이션 구축에 필수적인 핵심 내용이며, 튜토리얼 완료 시 `@aws/nx-plugin`에 대한 깊은 이해와 [NX](https://nx.dev/) 사용법의 기초를 습득하게 됩니다.

<Aside>이 튜토리얼은 직접 만들어보며 학습하는 것을 선호하고 빠르게 유형화된 결과물을 만들고자 하는 분들을 위해 설계되었습니다. 단계별 개념 학습을 선호한다면 왼쪽 탐색의 개별 구성 요소 가이드를 참조하세요.</Aside>

튜토리얼을 완료하면 다음을 학습하게 됩니다:

- `@aws/nx-plugin`을 사용한 새로운 애플리케이션 생성
- 코드베이스 관리/빌드를 위한 NX 활용
- tRPC와 FastAPI를 모두 사용한 API 구축
- 타입 안전 파일 기반 라우팅을 지원하는 TanStack 라우터로 페이지 생성
- 백엔드 API 호출을 위한 TanStack 쿼리 사용
- 목적에 맞게 생성된 코드 수정
- CDK 인프라 생성 및 배포

### 애플리케이션 구성 요소

다음 컴포넌트 아키텍처를 사용해 AI 기반 던전 어드벤처 게임을 구축합니다.

<Image class="white-bg" src={dungeonAdventureArchitecturePng} alt="dungeon-game-architecture.png" width="800" height="600" />

1. 게임 상태 관리를 위해 [ElectroDB](https://electrodb.dev/)/[DynamoDB](https://aws.amazon.com/dynamodb/)를 사용하는 [tRPC](https://trpc.io/) API
2. 스토리 생성용 Python/[FastAPI](https://fastapi.tiangolo.com/) Lambda 함수 (스트리밍 응답 지원, [Amazon Bedrock](https://aws.amazon.com/bedrock/) 기반)
3. React/Vite 프론트엔드 웹사이트:
  - 안전한 API 호출을 위한 Amazon Cognito/Identity Pools
  - 타입 안전 파일 기반 라우팅을 지원하는 [TanStack 라우터](https://tanstack.com/router/latest)
  - Game 및 Story API 호출용 생성된 SDK ([TanStack 쿼리](https://tanstack.com/query/latest) 활용)

애플리케이션 컴포넌트 배포를 위한 [CDK](https://docs.aws.amazon.com/cdk/v2/guide/home.html) 인프라
