## 개요

이 튜토리얼에서는 AI 기반 던전 모험 게임을 구축할 것입니다. 이 튜토리얼은 `@aws/nx-plugin` 또는 관련 기술에 대한 기존 지식을 가정하지 않습니다. 이 튜토리얼에서 배울 기술은 모든 `@aws/nx-plugin` 기반 애플리케이션을 구축하는 데 근본적이며, 이를 완전히 이해하면 `@aws/nx-plugin`에 대한 깊은 이해와 [NX](https://nx.dev/)를 사용하는 기본적인 방법을 알 수 있습니다.

<Aside>이 튜토리얼은 실습을 통해 배우고 빠르게 무언가를 만들어보고 싶어하는 사람들을 위해 설계되었습니다. 각 개념을 단계별로 배우는 것을 선호하는 경우, 개별 구성 요소 가이드를 참조하세요.</Aside>

이 튜토리얼을 완료한 사용자는 다음과 같은 지식을 얻을 수 있습니다:

- `@aws/nx-plugin`을 사용하여 새 애플리케이션 생성
- NX를 사용하여 코드베이스 관리/빌드
- tRPC와 FastAPI를 사용하여 API 구축
- Tanstack 라우터를 사용하여 새 페이지 생성
- Tanstack 쿼리를 사용하여 백엔드 API 호출
- 생성된 코드를 목적에 맞게 수정
- CDK 인프라 생성 및 배포

### 애플리케이션 구성 요소

AI 기반 던전 어드벤처 게임은 다음과 같은 컴포넌트 아키텍처를 사용하여 구축됩니다:

<Image class="white-bg" src={dungeonAdventureArchitecturePng} alt="dungeon-game-architecture.png" width="800" height="600" />

- 게임 상태 관리를 위해 [ElectroDB](https://electrodb.dev/)/[DynamoDB](https://aws.amazon.com/dynamodb/)를 사용하는 [tRPC](https://trpc.io/) API
- 스토리 생성을 위한 Python/[FastAPI](https://fastapi.tiangolo.com/) Lambda 함수와 스트리밍 응답 ([Amazon Bedrock](https://aws.amazon.com/bedrock/) 기반)
- 다음을 활용하는 React/Vite 프론트엔드 웹사이트:
  - 안전한 API 호출을 위한 Amazon Cognito/Identity Pools
  - 타입 안전한 파일 기반 라우팅을 지원하는 [Tanstack router](https://tanstack.com/router/latest)
  - 게임 및 스토리 API 호출을 위한 생성된 SDK ([Tanstack query](https://tanstack.com/query/latest) 활용)
- 애플리케이션 배포를 위한 [CDK](https://docs.aws.amazon.com/cdk/v2/guide/home.html) 인프라
