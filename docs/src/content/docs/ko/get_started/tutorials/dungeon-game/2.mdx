## 모듈 2: 게임 API 구현

게임 API를 구현하기 시작하겠습니다. 총 4개의 API를 생성해야 합니다:

1. `createGame` - 새 게임 인스턴스를 생성합니다.
2. `queryGames` - 이전에 저장된 게임의 페이지화된 목록을 반환합니다.
3. `saveAction` - 특정 게임에 대한 액션을 저장합니다.
4. `queryActions` - 게임과 관련된 모든 액션의 페이지화된 목록을 반환합니다.
