## Kiro Power

Nx Plugin for AWS는 MCP 서버와 문서 및 워크플로 가이드를 번들로 제공하는 [Kiro Power](https://kiro.dev/docs/powers/)와 함께 제공되어, Kiro에게 사용 가능한 제너레이터와 사용 방법에 대한 풍부한 컨텍스트를 제공합니다. 이것이 Kiro를 시작하는 권장 방법입니다.

### Power 설치하기

<Steps>

1. 사이드바에서 Kiro Powers 패널 열기
2. `+` 버튼을 클릭하여 커스텀 power 추가
3. 다음 GitHub URL 붙여넣기:

   ```
   https://github.com/awslabs/nx-plugin-for-aws/tree/main/powers/nx-plugin-for-aws
   ```

4. install 클릭

</Steps>

설치가 완료되면 Kiro는 자동으로 `aws-nx-mcp` MCP 서버와 해당 도구에 액세스할 수 있습니다. 추가 MCP 구성이 필요하지 않습니다.

### Power가 제공하는 기능

이 power는 Kiro에게 다음 MCP 도구에 대한 액세스를 제공합니다:

- `create_workspace_command` — AWS 플러그인으로 사전 구성된 새 Nx 작업 공간을 생성하는 명령 생성
- `list_generators` — 사용 가능한 모든 제너레이터와 해당 매개변수 검색
- `generator_guide` — 매개변수, 파일 구조 및 모범 사례를 포함한 특정 제너레이터에 대한 상세 문서 가져오기
- `general_guidance` — 플러그인 사용에 대한 일반 가이드 가져오기

또한 Kiro가 사용 가능한 제너레이터의 전체 세트, 일반적인 워크플로 및 모범 사례를 이해하는 데 도움이 되는 가이드 문서도 포함되어 있습니다.

### 예제 프롬프트

power가 설치되면 Kiro와 대화하여 빌드를 시작할 수 있습니다:

- *"Create a new Nx workspace called my-app using pnpm"* — Kiro는 적절한 작업 공간 생성 명령을 생성하고 실행합니다
- *"Scaffold a React website with a tRPC API backend and CDK infrastructure"* — Kiro는 각 프로젝트를 스캐폴딩하고 함께 연결합니다
- *"What generators are available in the Nx Plugin for AWS?"* — Kiro는 설명과 예제 사용법과 함께 모든 제너레이터를 나열합니다
- *"Add a Lambda function to my existing TypeScript project"* — Kiro는 적절한 제너레이터를 찾아 스캐폴딩 프로세스를 안내합니다
- *"Build my project"* - Kiro는 nx build 명령을 실행합니다

## MCP 서버 구성

Nx Plugin for AWS는 [MCP Server](https://modelcontextprotocol.io/introduction)를 포함하고 있어 AI 어시스턴트가 Nx Plugin for AWS와 협업할 수 있도록 합니다. MCP Server를 활용하면 AI를 통해 개발 워크플로를 가속화할 수 있으며, 프로젝트가 결정론적인 방식으로 구성되어 주요 컴포넌트 설정에 소요되는 시간과 컨텍스트 윈도우를 절약할 수 있습니다.

  </TabItem>
  <TabItem label="Claude Code">
    터미널에서 다음 명령을 실행하세요:

    ```bash
    claude mcp add nx-plugin-for-aws -- npx -y @aws/nx-plugin-mcp
    ```

    자세한 내용은 [Claude Code MCP docs](https://code.claude.com/docs/en/mcp)를 참조하세요.
  </TabItem>
  <TabItem label="Cursor">
    `~/.cursor/mcp.json`에 다음을 추가하세요:

    <Snippet name="mcp/config" />

AI 어시스턴트에게 `nx-plugin-for-aws`를 사용하여 무언가를 구축하도록 요청하세요! 일반적으로 AI 어시스턴트는 작업 공간 생성, 적절한 제너레이터를 통한 스캐폴딩, 비즈니스 로직 채우기 순서로 진행합니다.
