[PDK Nag](https://aws.github.io/aws-pdk/developer_guides/pdk-nag/index.html)는 [CDK Nag](https://github.com/cdklabs/cdk-nag)을 래핑하며 [프로토타입 구축에 특화된 규칙 집합](https://github.com/aws/aws-pdk/blob/mainline/packages/pdk-nag/src/packs/aws-prototyping-rules.ts)을 제공합니다.

PDK Nag에서 마이그레이션하려면 CDK Nag을 직접 사용하세요. 동일한 규칙 집합이 필요한 경우 [여기 문서](https://github.com/cdklabs/cdk-nag/blob/main/docs/NagPack.md)를 참고하여 자체 "팩"을 생성할 수 있습니다.
