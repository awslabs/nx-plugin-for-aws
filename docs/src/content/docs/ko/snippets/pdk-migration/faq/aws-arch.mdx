[AWS Arch](https://aws.github.io/aws-pdk/developer_guides/aws-arch/index.html)는 CDK Graph용 CloudFormation 리소스와 관련 아키텍처 아이콘 간 매핑을 제공합니다.

아이콘 관련 리소스는 [AWS Architecture Icons 페이지](https://aws.amazon.com/architecture/icons/)를 참조하세요. [Diagrams](https://diagrams.mingrammer.com/)도 코드로 다이어그램을 구축하는 방법을 제공합니다.

직접 사용하시는 경우 프로젝트를 포크(fork)하여 소유권을 가져가는 것을 고려해 보세요!
