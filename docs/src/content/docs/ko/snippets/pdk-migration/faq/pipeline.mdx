PDK는 [`PDKPipelineProject`](https://aws.github.io/aws-pdk/developer_guides/pipeline/index.html)를 제공하여 CDK 인프라 프로젝트를 설정하고 [CDK Pipelines](https://docs.aws.amazon.com/cdk/api/v2/docs/aws-cdk-lib.pipelines-readme.html) 리소스를 래핑한 CDK 구문을 활용했습니다.

이것에서 마이그레이션하려면 CDK Pipelines 구문을 직접 사용할 수 있습니다. 그러나 실제로는 [CDK Stages](https://docs.aws.amazon.com/cdk/v2/guide/stages.html)를 정의하고 적절한 스테이지에 대한 deploy 명령을 직접 실행하는 GitHub actions나 GitLab CI/CD와 같은 것을 사용하는 것이 더 간편할 수 있습니다.
