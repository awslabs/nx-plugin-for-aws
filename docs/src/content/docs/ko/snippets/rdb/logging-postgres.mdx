---
title: PostgreSQL 쿼리 로깅
---

`postgresql` 로그는 [Aurora PostgreSQL](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraPostgreSQL.CloudWatch.html)에 대해 내보내지며, 로깅은 DDL 문으로만 범위가 지정됩니다(`log_statement=ddl`). 따라서 각 문이 개별적으로 전송되는 한 문 매개변수 값은 절대 로깅되지 않습니다. `log_statement=ddl`은 다중 문 배치(예: 단일 `psql -c "a;b;c"` 호출)의 전체 원시 텍스트를 그대로 로깅하며, 배치 내 어떤 문이라도 DDL인 경우 동일한 배치의 모든 DML 값을 포함합니다.
