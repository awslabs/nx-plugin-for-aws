---
title: Serve Local Windows 주의사항
---

`serve-local`을 중지하면 (예: `Ctrl+C` 사용) DynamoDB Local 컨테이너가 자동으로 제거되지만, 명명된 볼륨은 보존되므로 재시작 시에도 데이터가 유지됩니다.

:::caution[Windows]
Windows의 신호 처리 제한으로 인해 `serve-local`이 중지될 때 컨테이너가 자동으로 제거되지 않습니다. 수동으로 제거해야 합니다:

```bash
<engine> rm -f <scope>-dynamodb
```

`<engine>`을 컨테이너 엔진(`docker` 또는 `finch`)으로, `<scope>`를 Nx 워크스페이스 스코프(예: `proj`)로 바꾸세요.
:::
