---
title: 로컬 DynamoDB 시작하기
---
import NxCommands from '@components/nx-commands.astro';

제너레이터는 [DynamoDB Local](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/DynamoDBLocal.html) 인스턴스를 시작하고 테이블을 생성하는 `serve-local` 타겟을 구성합니다. 프로젝트의 `dev` 타겟(`serve-local`의 별칭)을 사용하세요:

<NxCommands commands={['run <project-name>:dev']} />

이 명령은 자동으로 다음을 수행합니다:
1. DynamoDB Local 이미지를 가져옵니다 (`pull-image` 타겟)
2. 컨테이너를 시작합니다
3. `config.json`에 정의된 인덱스로 로컬 테이블을 생성합니다
