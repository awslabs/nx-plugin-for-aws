---
title: WAF 구성 CDK
---

:::note[REST API WAF 구성]
REST API의 경우, 생성된 CDK 구성 요소는 기본적으로 AWS WAFv2 Web ACL을 API의 기본 스테이지와 연결합니다. Web ACL은 AWS 관리형 기본 규칙 세트(`AWSManagedRulesCommonRuleSet` 및 `AWSManagedRulesKnownBadInputsRuleSet`)를 사용하여 OWASP Top 10을 포함한 일반적인 웹 공격으로부터 보호합니다.

옵트아웃이 필요한 경우(예: 자체 Web ACL을 연결하려는 경우), `enableWaf`를 `false`로 설정하세요:

```ts {4}
const api = new MyApi(this, 'MyApi', {
  integrations: MyApi.defaultIntegrations(this).build(),
  // Disable the default WAF (not recommended for production)
  enableWaf: false,
});
```
:::
