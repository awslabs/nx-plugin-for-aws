#### WAF

REST API의 경우, 생성된 Terraform 모듈은 기본적으로 AWS WAFv2 Web ACL을 API Gateway 스테이지와 연결합니다. Web ACL은 AWS 관리형 기본 규칙 세트(`AWSManagedRulesCommonRuleSet` 및 `AWSManagedRulesKnownBadInputsRuleSet`)를 사용하여 OWASP Top 10을 포함한 일반적인 웹 취약점 공격으로부터 보호합니다.

옵트아웃하려면(예: 자체 Web ACL을 연결하려는 경우), `enable_waf`를 `false`로 설정하세요:

```hcl {4}
module "my_api" {
  source = "../../common/terraform/src/app/apis/my-api"

  enable_waf = false
}
```
