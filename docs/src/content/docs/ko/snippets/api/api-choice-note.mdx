---



:::note
API를 배포할 때 [REST API](https://docs.aws.amazon.com/apigateway/latest/developerguide/apigateway-rest-api.html) 또는 [HTTP API](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api.html) 중 선택할 수 있습니다. 결정에 도움이 필요하면 [API Gateway Developer Guide](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api-vs-rest.html)를 참조하세요.
:::
