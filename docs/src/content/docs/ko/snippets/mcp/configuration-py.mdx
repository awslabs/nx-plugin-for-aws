### 설정 파일

MCP를 지원하는 대부분의 AI 어시스턴트는 유사한 설정 방식을 사용합니다. MCP 서버 세부 정보를 포함한 설정 파일을 생성하거나 업데이트해야 합니다:

```json
{
  "mcpServers": {
    "your-mcp-server": {
      "command": "uv",
      "args": [
        "run",
        "python",
        "-m",
        "my_module.mcp_server.stdio"
      ],
      "env": {
        "VIRTUAL_ENV": "/path/to/your/project/.venv"
      }
    }
  }
}
```

:::caution
서버에 연결할 때 `ENOENT uv`와 같은 오류가 발생한다면, 터미널에서 `which uv`를 실행하여 얻을 수 있는 `uv`의 전체 경로를 지정해야 할 수 있습니다.
:::
