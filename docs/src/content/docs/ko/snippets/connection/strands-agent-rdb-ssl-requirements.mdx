---
title: RDS Proxy 없이 연결할 때 SSL 요구사항
---
import Infrastructure from '@components/infrastructure.astro';

연결 생성기는 Dockerfile을 업데이트하여 Amazon RDS CA 번들을 `/usr/local/share/ca-certificates/rds-bundle.crt`에 설치합니다. RDS Proxy 없이 연결할 때 Node.js가 인증서를 신뢰하도록 `NODE_EXTRA_CA_CERTS`를 해당 경로로 설정하세요:

<Infrastructure>
<Fragment slot="cdk">

```ts title="packages/infra/src/stacks/application-stack.ts"
new MyAgent(this, 'MyAgent', {
  ...
  environmentVariables: {
    NODE_EXTRA_CA_CERTS: '/usr/local/share/ca-certificates/rds-bundle.crt',
  },
});
```

</Fragment>
<Fragment slot="terraform">

```hcl title="packages/infra/src/main.tf"
module "my_agent" {
  ...
  environment_variables = {
    NODE_EXTRA_CA_CERTS = "/usr/local/share/ca-certificates/rds-bundle.crt"
  }
}
```

</Fragment>
</Infrastructure>

자세한 내용은 [Amazon RDS SSL/TLS 문서](https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/UsingWithRDS.SSL.html)를 참조하세요. RDS Proxy를 사용하는 경우 `NODE_EXTRA_CA_CERTS`를 구성할 필요가 없습니다.
