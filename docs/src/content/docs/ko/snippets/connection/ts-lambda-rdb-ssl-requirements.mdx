---
title: RDS Proxy 없이 연결할 때의 SSL 요구사항
---
import Infrastructure from '@components/infrastructure.astro';

Node.js 20 이상의 Lambda 런타임에서 Aurora 클러스터에 직접 연결하는 경우, `NODE_EXTRA_CA_CERTS`를 설정하여 Amazon RDS CA 번들을 로드하세요:

<Infrastructure>
<Fragment slot="cdk">

```ts title="packages/infra/src/stacks/application-stack.ts"
const api = new Api(this, 'Api', {
  integrations: Api.defaultIntegrations(this)
    .withDefaultOptions({
      environment: {
        NODE_EXTRA_CA_CERTS: '/var/runtime/ca-cert.pem',
      },
    })
    .build(),
});
```

</Fragment>
<Fragment slot="terraform">

```hcl title="packages/infra/src/main.tf"
module "api" {
  source = "..."
  ...

  environment_variables = {
    NODE_EXTRA_CA_CERTS = "/var/runtime/ca-cert.pem"
  }
}
```

</Fragment>
</Infrastructure>

자세한 내용은 AWS Lambda [Amazon RDS 연결을 위한 SSL/TLS 요구사항](https://docs.aws.amazon.com/lambda/latest/dg/services-rds.html#services-rds-tls) 및 Amazon RDS Proxy [TLS 문서](https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/rds-proxy.howitworks.html#rds-proxy-security.tls)를 참조하세요. RDS Proxy를 사용하는 경우 Lambda 함수에서 RDS CA 번들을 구성할 필요가 없습니다.
