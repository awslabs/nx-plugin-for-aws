### 필수 요구사항

- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
- [Node >= 22](https://nodejs.org/en/download) (버전 관리를 위해 [NVM](https://github.com/nvm-sh/nvm) 사용을 권장합니다)
  - 설치 확인: `node --version`
- [PNPM >= 10](https://pnpm.io/installation#using-npm) ([Yarn >= 4](https://yarnpkg.com/getting-started/install), [Bun >= 1](https://bun.sh/docs/installation), 또는 [NPM >= 10](https://nodejs.org/en/learn/getting-started/an-introduction-to-the-npm-package-manager) 사용도 가능)
  - 설치 확인: `pnpm --version`, `yarn --version`, `bun --version` 또는 `npm --version`
- [UV >= 0.5.29](https://docs.astral.sh/uv/getting-started/installation/)
  1. Python 3.12 설치: `uv python install 3.12.0`
  2. 설치 확인: `uv python list --only-installed`
- 애플리케이션 배포 대상 AWS 계정에 구성된 [AWS Credentials](https://docs.aws.amazon.com/sdkref/latest/guide/access.html)

### 권장 사항

- 일부 생성기 사용 시 [Docker](https://www.docker.com/) 필요
- CDK 대신 인프라 코드로 [Terraform >= 1.12](https://developer.hashicorp.com/terraform/install) 사용 시 필수
  - 설치 확인: `terraform --version`
- [VSCode](https://code.visualstudio.com/) 사용 시 [Nx Console VSCode Plugin](https://marketplace.visualstudio.com/items?itemName=nrwl.angular-console) 설치 권장
