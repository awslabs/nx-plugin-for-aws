---
title: "필수 조건"
---



- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git) 설치
- [Node >= 22](https://nodejs.org/en/download) 설치 (노드 버전 관리를 위해 [NVM](https://github.com/nvm-sh/nvm) 사용을 권장합니다)
  - 설치된 버전을 확인하려면 터미널 창에서 `node --version`을 실행하세요
- [PNPM >= 10](https://pnpm.io/installation#using-npm) 설치 ([Yarn >= 4](https://yarnpkg.com/getting-started/install), [Bun >= 1](https://bun.sh/docs/installation) 또는 [NPM >= 10](https://nodejs.org/en/learn/getting-started/an-introduction-to-the-npm-package-manager)도 사용 가능)
  - 설치된 버전을 확인하려면 터미널 창에서 `pnpm --version`, `yarn --version`, `bun --version` 또는 `npm --version`을 실행하세요
- [UV >= 0.5.29](https://docs.astral.sh/uv/getting-started/installation/) 설치
  1. Python 3.1을 설치하려면 `uv python install 3.12.0`을 실행하세요
  2. `uv python list --only-installed`로 확인하세요
- [AWS Credentials](https://docs.aws.amazon.com/sdkref/latest/guide/access.html)을 대상 AWS 계정(애플리케이션이 배포될 계정)에 맞게 구성하세요
- [VSCode](https://code.visualstudio.com/)를 사용하는 경우 [Nx Console VSCode Plugin](https://marketplace.visualstudio.com/items?itemName=nrwl.angular-console) 설치를 권장합니다
