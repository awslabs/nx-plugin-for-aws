import NxCommands from '@components/nx-commands.astro';

제너레이터는 [Rolldown](https://rolldown.rs/)을 사용하여 배포 패키지를 생성하는 `bundle` 타겟을 자동으로 구성합니다:

<NxCommands commands={['run <project-name>:bundle']} />

Rolldown 구성은 `rolldown.config.ts`에서 확인할 수 있으며, 생성할 각 번들별로 엔트리가 존재합니다. 정의된 경우 Rolldown은 여러 번들을 병렬로 생성하는 작업을 관리합니다.

:::tip
`rolldown.config.ts`에서 Rolldown 구성을 조정할 수 있습니다. 예를 들어 파일 경로에 의존하는 종속성이 있어 번들링이 불가능한 경우, `external` 모듈 목록에 추가할 수 있습니다.
:::
