이 생성기는 선택한 `iacProvider` 기반으로 인프라를 코드 형태로 제공하므로, `packages/common` 디렉터리에 관련 CDK 구축 요소 또는 Terraform 모듈을 포함하는 프로젝트를 생성합니다.

공통 인프라스트럭처 코드 프로젝트의 구조는 다음과 같습니다:

<Infrastructure>
<Fragment slot="cdk">
<FileTree>
  - packages/common/constructs
    - src
      - app/ 특정 프로젝트/생성기에 종속적인 인프라를 위한 구축 요소
      - core/ `app` 내 구축 요소에서 재사용되는 일반적 구축 요소
      - index.ts `app`의 구축 요소를 익스포트하는 진입점
    - project.json 프로젝트 빌드 대상 및 구성
</FileTree>

:::note
이 프로젝트는 <Link path="/guides/typescript-project">`ts#project`</Link> 생성기를 사용하여 생성되었으므로 동일한 빌드 대상이 구성됩니다.
:::
</Fragment>
<Fragment slot="terraform">
<FileTree>
  - packages/common/terraform
    - src
      - app/ 특정 프로젝트/생성기 전용 Terraform 모듈
      - core/ `app` 내 모듈에서 재사용되는 일반적 모듈
    - project.json 프로젝트 빌드 대상 및 구성
</FileTree>

:::note
이 프로젝트는 <Link path="/guides/terraform-project">`terraform#project`</Link> 생성기를 사용하여 생성되었으므로 동일한 빌드 대상이 구성됩니다.
:::
</Fragment>
</Infrastructure>
