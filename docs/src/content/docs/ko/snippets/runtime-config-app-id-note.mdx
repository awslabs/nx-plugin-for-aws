:::note[런타임 구성]
이것은 런타임에 AWS AppConfig에서 구성을 가져오기 위해 `RUNTIME_CONFIG_APP_ID` 환경 변수에 의존합니다. 이 플러그인으로 빌드된 프로젝트(API, Agent, MCP 서버)는 이미 이 변수가 자동으로 구성되어 있습니다. 다른 프로젝트의 경우, 인프라에서 프로비저닝한 AppConfig 애플리케이션 ID로 런타임 환경에 `RUNTIME_CONFIG_APP_ID`가 설정되어 있는지 확인하세요. 자세한 내용은 <Link path="guides/runtime-config">런타임 구성 가이드</Link>를 참조하세요.
:::
