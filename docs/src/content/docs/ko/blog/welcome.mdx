이제 라이브입니다! 🚀

AWS용 Nx 플러그인은 AWS에서 풀스택 애플리케이션을 구축하고 배포하는 과정을 간소화하는 도구 키트를 제공하는 [Nx](https://nx.dev/) 플러그인입니다. 개발자에게 애플리케이션 및 IaC 코드에 대한 사전 구성된 템플릿을 제공하여 설정 및 구성에 소요되는 시간을 크게 줄입니다. 이 플러그인은 사용자 지정의 유연성을 유지하면서 AWS 서비스 통합의 복잡성을 처리합니다.

사용자는 간단히 사용 가능한 생성기 목록에서 원하는 구성 요소를 선택하고, 구성 옵션을 제공하면 `@aws/nx-plugin`이 필요한 스타터 코드를 생성합니다. 이 도구 키트에는 API, 웹사이트, 인프라를 생성할 수 있는 여러 생성기가 있으며, 타입 안전 클라이언트와 함께 프론트엔드를 백엔드에 통합하는 것과 같은 더 정교한 작업도 수행할 수 있습니다.
