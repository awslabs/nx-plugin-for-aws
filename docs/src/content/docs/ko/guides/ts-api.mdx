TypeScript API 생성기는 선택한 프레임워크로 새로운 API 프로젝트를 생성합니다. 엔드투엔드 타입 안전한 API를 위한 [tRPC](https://trpc.io/)와 모델 기반 API 개발을 위한 [Smithy](https://smithy.io/)를 지원합니다.

## 사용법
