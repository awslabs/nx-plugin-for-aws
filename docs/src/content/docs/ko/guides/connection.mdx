이 생성기는 웹사이트가 API를 호출하는 것과 같이 프로젝트들을 서로 연결하는 데 사용됩니다. 소스 프로젝트(예: API를 호출할 프로젝트)와 타겟 프로젝트(예: API 프로젝트)를 선택하기만 하면 두 프로젝트의 통합을 자동으로 처리해줍니다.

### 지원되는 연결 유형

연결 생성기는 다음 유형의 연결을 지원합니다:

:::note[런타임 구성]
연결 생성기는 <Link path="guides/runtime-config">런타임 구성</Link>을 활용하여 배포 시점의 값들(예: API URL, Cognito 설정, 에이전트 런타임 ARN)을 생성된 프로젝트와 컴포넌트 간에 런타임에 전달함으로써 서로를 발견하고 연결할 수 있도록 합니다.
