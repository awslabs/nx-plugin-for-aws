이 생성기는 웹사이트가 API를 호출하는 것과 같이 프로젝트들을 서로 연결하는 데 사용됩니다. 소스 프로젝트(예: API를 호출할 프로젝트)와 타겟 프로젝트(예: API 프로젝트)를 선택하기만 하면 이 생성기가 두 프로젝트의 통합을 처리합니다.

## 사용 방법

### 제너레이터 실행

<RunGenerator generator="connection" />

### 옵션

<GeneratorParameters generator="connection" />

:::note[런타임 구성]
연결 생성기는 <Link path="guides/runtime-config">런타임 구성</Link>을 사용하여 배포 시점의 값들(예: API URL, Cognito 설정, 에이전트 런타임 ARN)을 생성된 프로젝트와 컴포넌트 간에 런타임에 전달함으로써 서로를 발견하고 연결할 수 있도록 합니다.
:::

:::tip[로컬 개발]
연결된 프로젝트는 `serve` 및 `dev` 타겟을 사용하여 로컬 머신에서 실행할 수 있습니다. 자세한 내용은 <Link path="guides/local-development">로컬 개발</Link> 가이드를 참조하세요.
:::
