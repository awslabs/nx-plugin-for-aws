`ts#website` 생성기는 새로운 웹사이트 애플리케이션을 생성합니다. `framework` 옵션을 통해 사용할 프론트엔드 프레임워크를 선택할 수 있습니다 — 현재 [React](https://react.dev/)가 지원됩니다.

생성된 프로젝트 구조, 설정 및 사용법에 대한 자세한 내용은 프레임워크별 가이드를 참조하세요:
