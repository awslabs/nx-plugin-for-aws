`connection` 생성기는 <Link path="guides/trpc">tRPC API</Link>를 <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link> 프로젝트에 연결하여, 로컬 개발 시 두 프로젝트가 자동으로 함께 시작되도록 구성합니다.

## 사전 요구 사항

이 생성기를 사용하기 전에 다음이 필요합니다:

1. <Link path="guides/trpc">tRPC API</Link> 프로젝트 (`ts#api`로 생성됨)
2. <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link> 프로젝트

## 사용법

### 생성기 실행

<RunGenerator generator="connection" />

tRPC API 프로젝트를 소스로, DynamoDB 프로젝트를 타겟으로 선택하세요.

### 옵션

<GeneratorParameters generator="connection" />

## 생성기 출력

생성기는 tRPC API의 `project.json`을 업데이트하여 `dev` 타겟에서 DynamoDB 프로젝트의 `dev` 타겟으로의 의존성을 추가합니다. 소스 파일은 수정되지 않습니다.

## 프로시저에서 DynamoDB 사용

DynamoDB 패키지에서 엔티티 팩토리를 가져와 tRPC 프로시저 내에서 사용하세요:

```ts title="packages/api/src/procedures/example.ts"
import { createExampleEntity } from ':my-scope/my-table';
import { publicProcedure } from '../init.js';

export const listExamples = publicProcedure
  .query(async () => {
    const entity = await createExampleEntity();
    const result = await entity.scan.go();
    return result.data;
  });
```
