`connection` 생성기는 <Link path="guides/py-agent">Python Agent</Link>를 <Link path="guides/py-dynamodb">Python DynamoDB</Link> 프로젝트에 연결하여 로컬 개발 환경에서 두 프로젝트가 자동으로 함께 시작되도록 구성합니다.

## 사전 요구 사항

이 생성기를 사용하기 전에 다음이 필요합니다:

1. <Link path="guides/py-agent">`py#agent`</Link> 프로젝트
2. <Link path="guides/py-dynamodb">`py#dynamodb`</Link> 프로젝트

## 사용법

### 생성기 실행

<RunGenerator generator="connection" />

Agent 프로젝트를 소스로 선택하고 DynamoDB 프로젝트를 타겟으로 선택합니다. 프로젝트에 여러 에이전트 컴포넌트가 포함되어 있는 경우 `sourceComponent`를 지정하여 명확히 구분합니다.

### 옵션

<GeneratorParameters generator="connection" />

## 생성기 출력

생성기는 에이전트의 `<agent-name>-serve-local` 타겟을 `project.json`에서 업데이트하여 DynamoDB 프로젝트의 `serve-local` 타겟에 의존하도록 합니다. 소스 파일은 수정되지 않습니다.
