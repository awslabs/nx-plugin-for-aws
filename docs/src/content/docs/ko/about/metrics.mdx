---
title: "사용 메트릭"
description: "@aws/nx-plugin이 메트릭을 수집하는 방법"
---



Nx Plugin for AWS 개선을 돕기 위해 생성기 사용 현황에 대한 메트릭을 수집합니다. 이 과정에는 추가 네트워크 요청이 수반되지 않으며, 기존 AWS 사용 현황 메트릭을 활용합니다.

투명성을 유지하기 위해 CloudFormation Stack 설명에 ID, 버전 및 생성기 ID 목록을 추가합니다. 이는 CDK의 경우 `common/constructs` 내 `MetricsAspect`에서, Terraform의 경우 `metrics.tf` CloudFormation 스택에서 수행됩니다.

## 옵트아웃

메트릭 코드는 전적으로 워크스페이스 내에 존재하므로, 이를 제거하여 옵트아웃할 수 있습니다:

- **CDK**: `packages/common/constructs/src/core/app.ts`에서 `MetricsAspect` 클래스와 `Aspects.of(this).add(new MetricsAspect())` 호출을 제거하세요.
- **Terraform**: `packages/common/terraform/src/metrics/metrics.tf`를 삭제하세요.

생성기는 제거된 메트릭 코드를 다시 추가하지 않습니다.
