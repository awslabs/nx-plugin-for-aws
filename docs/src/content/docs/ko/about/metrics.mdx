---
title: "사용 메트릭"
description: "@aws/nx-plugin이 메트릭을 수집하는 방법"
---



Nx Plugin for AWS 개선을 지원하기 위해 생성기 사용 현황에 대한 메트릭을 수집합니다. 이 프로세스에는 추가 네트워크 요청이 필요하지 않으며, 기존 AWS 사용 현황 메트릭을 활용합니다.

투명성을 유지하기 위해 CloudFormation Stack 설명에 ID, 버전 및 생성기 ID 목록을 추가합니다. 이 작업은 `common/constructs`의 `MetricsAspect`에서 수행됩니다.
