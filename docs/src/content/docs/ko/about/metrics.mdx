---
title: "사용 메트릭"
description: "@aws/nx-plugin이 메트릭을 수집하는 방법"
---



Nx Plugin for AWS를 개선하는 데 도움을 주기 위해 생성기 사용 현황에 대한 메트릭을 수집합니다. 이 과정에는 추가 네트워크 요청이 수반되지 않으며, 기존 AWS 사용 현황 메트릭을 활용합니다.

투명성을 유지하기 위해 CloudFormation 스택 설명에 ID, 버전 및 생성기 ID 목록을 추가합니다. 이는 CDK의 경우 `common/constructs`에 있는 `MetricsAspect`에서, Terraform의 경우 `metrics.tf` CloudFormation 스택에서 수행됩니다.
