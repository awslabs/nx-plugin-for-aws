import NxCommands from '@components/nx-commands.astro';

生成器会自动配置一个使用 [Rolldown](https://rolldown.rs/) 创建部署包的 `bundle` 目标：

<NxCommands commands={['run <project-name>:bundle']} />

Rolldown 配置位于 `rolldown.config.ts` 文件中，每个要生成的包都有对应的入口配置。如果定义了多个包，Rolldown 会并行管理这些包的创建过程。

:::tip
您可以通过修改 `rolldown.config.ts` 来调整 Rolldown 的配置。例如，如果有依赖项需要依赖文件路径且无法被打包，可以将它们添加到 `external` 模块列表中。
:::
