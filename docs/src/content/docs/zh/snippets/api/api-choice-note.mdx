---
title: "选择 REST 和 HTTP 之间的方式"
---



:::note
您可以选择使用 [REST API](https://docs.aws.amazon.com/apigateway/latest/developerguide/apigateway-rest-api.html) 或 [HTTP API](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api.html) 来部署 API，具体选择可参考 [API Gateway 开发人员指南](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api-vs-rest.html) 进行决策。
:::
