#### WAF

对于 REST API，生成的 Terraform 模块默认会将 AWS WAFv2 Web ACL 与 API Gateway 阶段关联。Web ACL 使用 AWS 托管的默认规则集（`AWSManagedRulesCommonRuleSet` 和 `AWSManagedRulesKnownBadInputsRuleSet`），提供针对常见 Web 漏洞（包括 OWASP Top 10）的保护。

要选择退出（例如，附加您自己的 Web ACL），请将 `enable_waf` 设置为 `false`：

```hcl {4}
module "my_api" {
  source = "../../common/terraform/src/app/apis/my-api"

  enable_waf = false
}
```
