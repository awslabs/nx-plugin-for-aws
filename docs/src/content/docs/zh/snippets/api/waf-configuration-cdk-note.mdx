#### WAF

对于 REST API，生成的 CDK 构造默认会将 AWS WAFv2 Web ACL 与 API 的默认阶段关联。Web ACL 使用 AWS 托管的默认规则集（`AWSManagedRulesCommonRuleSet` 和 `AWSManagedRulesKnownBadInputsRuleSet`），提供针对常见 Web 漏洞（包括 OWASP Top 10）的保护。

要选择退出（例如，附加您自己的 Web ACL），请将 `enableWaf` 设置为 `false`：

```ts {4}
const api = new MyApi(this, 'MyApi', {
  integrations: MyApi.defaultIntegrations(this).build(),
  enableWaf: false,
});
```
