---
title: PostgreSQL 查询日志
---

`postgresql` 日志会导出到 [Aurora PostgreSQL](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraPostgreSQL.CloudWatch.html)，日志记录范围仅限于 DDL 语句（`log_statement=ddl`），因此永远不会记录语句参数值——只要每个语句单独发送。`log_statement=ddl` 会逐字记录多语句批处理的整个原始文本（例如单个 `psql -c "a;b;c"` 调用），如果其中任何语句是 DDL，则包括同一批处理中的任何 DML 值。
