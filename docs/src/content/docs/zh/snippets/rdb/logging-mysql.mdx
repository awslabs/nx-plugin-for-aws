---
title: MySQL 查询日志
---

为 [Aurora MySQL](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraMySQL.Integrating.CloudWatch.html) 导出了 `audit` 和 `error` 日志——故意排除了 `general` 和 `slowquery`，因为它们会记录完整的语句文本，包括 DML 值。[高级审计](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraMySQL.Auditing.html)的范围限定为连接和 DDL（`server_audit_events=CONNECT,QUERY_DDL`），因此永远不会记录语句参数值。
