---
title: 启动本地 DynamoDB
---
import NxCommands from '@components/nx-commands.astro';

生成器配置了一个 `serve-local` 目标，用于启动 [DynamoDB Local](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/DynamoDBLocal.html) 实例并创建表。使用项目的 `dev` 目标（`serve-local` 的别名）：

<NxCommands commands={['run <project-name>:dev']} />

这将自动：
1. 拉取 DynamoDB Local 镜像（`pull-image` 目标）
2. 启动容器
3. 使用 `config.json` 中定义的索引创建本地表
