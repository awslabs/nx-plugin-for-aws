:::note[运行时配置]
这依赖于 `RUNTIME_CONFIG_APP_ID` 环境变量在运行时从 AWS AppConfig 获取配置。使用此插件构建的项目（API、Agent、MCP 服务器）已经自动配置了此变量。对于其他项目，请确保在运行时环境中设置 `RUNTIME_CONFIG_APP_ID`，其值为基础设施配置的 AppConfig 应用程序 ID。有关更多信息，请参阅<Link path="guides/runtime-config">运行时配置指南</Link>。
:::
