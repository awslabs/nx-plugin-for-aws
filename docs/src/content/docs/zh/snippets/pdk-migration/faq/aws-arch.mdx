[AWS Arch](https://aws.github.io/aws-pdk/developer_guides/aws-arch/index.html) 为 CDK Graph 提供了 CloudFormation 资源与其对应架构图标的映射关系。

有关图标资源，请参考 [AWS Architecture Icons 页面](https://aws.amazon.com/architecture/icons/)。[Diagrams](https://diagrams.mingrammer.com/) 也提供了一种通过代码构建架构图的方式。

如果您需要直接使用这些资源，建议您 fork 该项目并自行维护！
