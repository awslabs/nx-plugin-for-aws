[PDK Nag](https://aws.github.io/aws-pdk/developer_guides/pdk-nag/index.html) 封装了 [CDK Nag](https://github.com/cdklabs/cdk-nag)，并提供了一套[专门用于构建原型的安全规则](https://github.com/aws/aws-pdk/blob/mainline/packages/pdk-nag/src/packs/aws-prototyping-rules.ts)。

如需从 PDK Nag 迁移，请直接使用 CDK Nag。若需要相同的规则集，您可以通过[此文档](https://github.com/cdklabs/cdk-nag/blob/main/docs/NagPack.md)创建自己的"规则包"。
