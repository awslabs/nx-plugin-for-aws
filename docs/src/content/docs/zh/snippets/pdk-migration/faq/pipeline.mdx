PDK 提供了一个 [`PDKPipelineProject`](https://aws.github.io/aws-pdk/developer_guides/pipeline/index.html)，它用于设置 CDK 基础设施项目，并利用了一个封装了部分 [CDK Pipelines](https://docs.aws.amazon.com/cdk/api/v2/docs/aws-cdk-lib.pipelines-readme.html) 资源的 CDK 构造体。

若需从此迁移，您可以直接使用 CDK Pipelines 的构造体。然而在实际操作中，采用如 GitHub Actions 或 GitLab CI/CD 这类工具可能更为简便。您只需定义 [CDK Stages](https://docs.aws.amazon.com/cdk/v2/guide/stages.html) 并直接运行对应阶段的部署命令即可。
