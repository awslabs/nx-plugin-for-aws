### 配置文件

大多数支持 MCP 的 AI 助手都采用相似的配置方式。您需要创建或更新包含 MCP 服务器详细信息的配置文件：

```json
{
  "mcpServers": {
    "your-mcp-server": {
      "command": "npx",
      "args": ["tsx", "/path/to/your-mcp-server/stdio.ts"]
    }
  }
}
```

:::caution
如果连接服务器时出现类似 `ENOENT npx` 的错误，可能需要指定 `npx` 的完整路径。您可以通过在终端运行 `which npx` 获取该路径。
:::

### 热重载

在开发 MCP 服务器时，建议配置 `--watch` 参数以确保 AI 助手始终获取工具/资源的最新版本：

```json {5}
{
  "mcpServers": {
    "your-mcp-server": {
      "command": "npx",
      "args": ["tsx", "--watch", "/path/to/your-mcp-server/stdio.ts"]
    }
  }
}
```

:::note
添加新工具或资源后，可能需要在 AI 助手中刷新 MCP 服务器。
:::
