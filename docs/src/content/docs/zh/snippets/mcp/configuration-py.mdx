### 配置文件

大多数支持 MCP 的 AI 助手都采用相似的配置方式。您需要创建或更新包含 MCP 服务器详细信息的配置文件：

```json
{
  "mcpServers": {
    "your-mcp-server": {
      "command": "uv",
      "args": [
        "run",
        "python",
        "-m",
        "my_module.mcp_server.stdio"
      ],
      "env": {
        "VIRTUAL_ENV": "/path/to/your/project/.venv"
      }
    }
  }
}
```

:::caution
如果在连接服务器时遇到类似 `ENOENT uv` 的错误，可能需要指定 `uv` 的完整路径。您可以通过在终端运行 `which uv` 命令获取该路径。
:::
