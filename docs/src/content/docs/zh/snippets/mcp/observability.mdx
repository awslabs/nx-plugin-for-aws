---
title: "MCP 可观测性"
---



您的MCP服务器通过配置`Dockerfile`中的自动检测功能，使用[AWS Distro for Open Telemetry](https://aws.amazon.com/otel/) (ADOT) 实现了开箱即用的可观测性。

您可以在CloudWatch AWS控制台的菜单中选择"GenAI Observability"查看追踪数据。注意：要使追踪数据正常显示，需先启用[Transaction Search](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-Transaction-Search.html)功能。

更多详细信息，请参考[AgentCore可观测性配置文档](https://docs.aws.amazon.com/bedrock-agentcore/latest/devguide/observability-configure.html)。
