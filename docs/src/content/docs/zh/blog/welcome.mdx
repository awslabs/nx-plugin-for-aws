Nx Plugin for AWS 是一款面向 [Nx](https://nx.dev/) 的插件工具包，旨在简化全栈应用在 AWS 上的构建和部署流程。它为开发者提供了预配置的应用程序和基础设施即代码（IaC）模板，大幅减少了环境搭建与配置的时间。该插件在保持定制灵活性的同时，妥善处理了 AWS 服务集成的复杂性。

用户只需从现有生成器列表中选择所需组件，配置必要选项，`@aws/nx-plugin` 便会自动生成对应的初始代码。工具包内包含多种生成器，可创建 API、网站、基础设施，甚至能通过 AST 转换实现前端与后端的深度集成（包括自动更新现有文件！）并生成类型安全的客户端。
