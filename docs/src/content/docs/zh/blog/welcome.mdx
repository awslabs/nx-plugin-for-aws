Nx Plugin for AWS 是一款 [Nx](https://nx.dev/) 插件，提供在 AWS 上构建和部署全栈应用的开发工具包。它为开发者提供应用代码和基础设施即代码（IaC）的预配置模板，大幅减少搭建和配置所需时间。该插件处理了 AWS 服务集成的复杂性，同时保留了定制化的灵活性。

用户只需从可用生成器列表中选择所需组件，提供配置选项后即可通过 `@aws/nx-plugin` 生成所需初始代码。工具包内包含多种生成器，可创建 API、网站、基础设施，甚至通过抽象语法树转换实现前端与后端的深度集成（包括更新现有文件！）并生成类型安全客户端。
