`connection` 生成器将 <Link path="guides/ts-smithy-api">Smithy API</Link> 连接到 <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link> 项目，配置本地开发环境使两者自动一起启动。

## 前置条件

在使用此生成器之前，请确保您已有：

1. 一个 <Link path="guides/ts-smithy-api">Smithy TypeScript API</Link> 项目（使用 `ts#api` 和 `--framework=smithy` 生成）
2. 一个 <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link> 项目

## 使用方法

### 运行生成器

<RunGenerator generator="connection" />

## 生成器输出

生成器会更新您的 Smithy API 的 `project.json`，添加从其 `serve-local` 目标到 DynamoDB 项目的 `serve-local` 目标的依赖关系。不会修改源文件。

## 在操作中使用 DynamoDB

从 DynamoDB 包中导入实体工厂，并在操作实现中使用它们：

```ts title="packages/api/src/operations/list-examples.ts"
import { createExampleEntity } from ':my-scope/my-table';
import {
  ListExamplesOperationInput,
  ListExamplesOperationOutput,
} from '../generated/ssdk/index.js';
import { ServiceContext } from '../context.js';

export const listExamples = async (
  _input: ListExamplesOperationInput,
  _ctx: ServiceContext,
): Promise<ListExamplesOperationOutput> => {
  const entity = await createExampleEntity();
  const result = await entity.scan.go();
  return { items: result.data };
};
```
