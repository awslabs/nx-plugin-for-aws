`connection` 生成器将 <Link path="guides/trpc">tRPC API</Link> 连接到 <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link> 项目，配置本地开发环境使两者自动一起启动。

## 前置条件

在使用此生成器之前，请确保您拥有：

1. 一个 <Link path="guides/trpc">tRPC API</Link> 项目（使用 `ts#api` 生成）
2. 一个 <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link> 项目

## 使用方法

### 运行生成器

<RunGenerator generator="connection" />

<GeneratorParameters generator="connection" />

## 生成器输出

生成器会更新您的 tRPC API 的 `project.json`，添加从其 `serve-local` 目标到 DynamoDB 项目的 `serve-local` 目标的依赖关系。不会修改源文件。

## 在 Procedures 中使用 DynamoDB

从 DynamoDB 包中导入实体工厂，并在 tRPC procedures 中使用它们：

```ts title="packages/api/src/procedures/example.ts"
import { createExampleEntity } from ':my-scope/my-table';
import { publicProcedure } from '../init.js';

export const listExamples = publicProcedure
  .query(async () => {
    const entity = await createExampleEntity();
    const result = await entity.scan.go();
    return result.data;
  });
```
