`connection` 生成器将 <Link path="guides/fastapi">FastAPI</Link> 连接到 <Link path="guides/py-dynamodb">Python DynamoDB</Link> 项目，配置本地开发环境使两者自动一起启动。

## 前置条件

在使用此生成器之前，请确保您具备：

1. 一个 FastAPI 项目（使用 `py#api --framework=fastapi` 生成），请参阅 <Link path="guides/fastapi">`py#api`</Link> 指南
2. 一个 <Link path="guides/py-dynamodb">`py#dynamodb`</Link> 项目

### 运行生成器

<RunGenerator generator="connection" />

<GeneratorParameters generator="connection" />

## 生成器输出

该生成器会更新 FastAPI 的 `project.json`，从其 `dev` 目标添加对 DynamoDB 项目的 `dev` 目标的依赖，并将 DynamoDB 包添加为工作区依赖项。不会修改源文件。

## 在路由处理器中使用 DynamoDB

从 DynamoDB 包导入实体类并在路由处理器中使用它们：

```python title="packages/my_api/my_api/api.py"
from my_scope.my_table.entities.example import ExampleModel

@app.get("/examples")
def list_examples():
    return list(ExampleModel.scan())
```
