`connection` 生成器将 <Link path="guides/fastapi">FastAPI</Link> 连接到 <Link path="guides/py-dynamodb">Python DynamoDB</Link> 项目，配置本地开发环境使两者自动一起启动。

## 前置条件

在使用此生成器之前，请确保您具备：

1. 一个 <Link path="guides/fastapi">`py#fast-api`</Link> 项目
2. 一个 <Link path="guides/py-dynamodb">`py#dynamodb`</Link> 项目

## 使用方法

### 运行生成器

<RunGenerator generator="connection" />

选择您的 FastAPI 项目作为源，选择您的 DynamoDB 项目作为目标。

### 选项

<GeneratorParameters generator="connection" />

## 生成器输出

该生成器会更新 FastAPI 的 `project.json`，从其 `serve-local` 目标添加对 DynamoDB 项目的 `serve-local` 目标的依赖。不会修改源文件。
