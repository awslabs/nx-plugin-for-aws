该生成器用于将项目连接在一起，例如网站调用 API。只需选择源项目（例如将调用您的 API 的项目）和目标项目（例如您的 API 项目），该生成器将处理两者的集成。

:::note[运行时配置]
连接生成器利用<Link path="guides/runtime-config">运行时配置</Link>在生成的项目和组件之间传递部署时值（例如 API URL、Cognito 设置和 agent 运行时 ARN），以便它们可以在运行时相互发现和连接。
:::

:::tip[本地开发]
已连接的项目可以使用 `serve` 和 `serve-local` 目标在您的机器上运行。有关详细信息，请参阅<Link path="guides/local-development">本地开发</Link>指南。
:::
