该生成器用于将项目连接在一起，例如网站调用API。只需选择源项目（例如将调用您的API的项目）和目标项目（例如您的API项目），该生成器将处理两者的集成。

### 支持的连接

连接生成器支持以下连接：

:::note[运行时配置]
连接生成器利用<Link path="guides/runtime-config">运行时配置</Link>在生成的项目和组件之间传递部署时值（例如API URL、Cognito设置和agent运行时ARN），以便它们可以在运行时相互发现和连接。
:::
