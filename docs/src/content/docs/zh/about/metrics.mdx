---
title: "使用指标"
description: "@aws/nx-plugin 如何收集指标"
---



为了帮助我们改进 Nx Plugin for AWS，我们会收集生成器的使用指标。此过程不涉及任何额外的网络请求，我们仅利用现有的 AWS 使用指标。

为保持透明度，我们会在 CloudFormation 堆栈描述中附加 ID、版本和生成器 ID 列表。该操作通过 CDK 的 `common/constructs` 中的 `MetricsAspect` 或 Terraform 的 `metrics.tf` CloudFormation 堆栈完成。

## 选择退出

指标代码完全存在于您的工作区中，因此您可以通过删除它来选择退出：

- **CDK**：从 `packages/common/constructs/src/core/app.ts` 中删除 `MetricsAspect` 类和 `Aspects.of(this).add(new MetricsAspect())` 调用。
- **Terraform**：删除 `packages/common/terraform/src/metrics/metrics.tf`。

生成器在删除后不会重新添加指标代码。
