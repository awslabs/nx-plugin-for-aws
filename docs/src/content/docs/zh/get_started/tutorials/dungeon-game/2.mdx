## 任务 1：实现游戏 API

我们将在本节中实现以下 API：

1. `saveGame` - 创建或更新游戏。
2. `queryGames` - 返回分页的历史游戏列表。
3. `saveAction` - 保存指定游戏的操作记录。
4. `queryActions` - 返回分页的指定游戏所有操作记录。
5. `queryInventory` - 返回分页的玩家库存物品列表。

### 部署应用
要部署应用，运行以下命令：

<NxCommands commands={['deploy infra "dungeon-adventure-infra-sandbox/*"']} />

:::caution[Bootstrap Required]
要先初始化 AWS 账户，运行：

<NxCommands commands={['bootstrap infra']} />
:::

首次部署约需 8 分钟，后续部署约需 2 分钟。

:::tip[Fast Iteration]
若仅修改 Lambda 函数代码，构建后使用 `--hotswap` 标志部署可大幅缩短时间（2-3 秒）：

<NxCommands commands={['deploy infra "dungeon-adventure-infra-sandbox/*" --hotswap']} />
:::
