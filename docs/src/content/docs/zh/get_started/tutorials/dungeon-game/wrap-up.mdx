## 总结

恭喜！您已成功创建了一个 Agentic AI 地牢冒险游戏，该游戏使用了 `@aws/nx-plugin` 中提供的多种生成器。🎉🎉🎉

## 后续步骤

我们建议您尝试自行扩展代码库，添加以下功能：

1. 为项目添加<Link path="guides/license">许可证生成器</Link>，自动管理工作区中的 LICENSE 文件与源代码头部信息。
2. <Link path="get_started/tutorials/contribute-generator">贡献自定义生成器</Link>来简化新 tRPC API 的添加流程。
3. 在 GameAPI 中新增 `resetToAction` API（使用上述生成器），允许回溯时间并删除指定时间点后的所有操作。更新 UI，在对话气泡添加重置按钮来调用该 API。
4. 使用左侧导航栏探索各种组件指南。

## 销毁资源

1. 要销毁已创建的 AWS 资源，请运行以下命令：

<NxCommands commands={['run @dungeon-adventure/infra:destroy dungeon-adventure-infra-sandbox/*']} />

    该命令将提示选择要删除的堆栈列表，通常包含 `dungeon-adventure-infra-sandbox/Application/GameUI/waf` 和 `dungeon-adventure-infra-sandbox/Application`。

2. 输入 `Y` 继续。CloudFormation 将销毁您的堆栈。
