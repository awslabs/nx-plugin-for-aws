## 下一步建议？

我们建议您尝试自行扩展代码库，添加以下功能：

1. 为项目添加<Link path="guides/license">许可证生成器</Link>，自动管理工作区中的LICENSE文件及源代码头信息。
2. <Link path="get_started/tutorials/create-generator">创建自定义生成器</Link>，简化新增tRPC API的过程。
3. 在GameAPI中新增`resetToAction` API（使用上述生成器），允许回溯时间并删除指定时间点后的所有操作。随后更新UI，在对话气泡中添加按钮以触发此API实现状态回退。
4. 探索更多可用指南

### 销毁资源

如需销毁已创建的AWS资源，请运行以下命令：

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

此命令将列出待删除的堆栈清单，通常包括`waf`和`dungeon-adventure-infra-sandbox`。输入`Y`后，Cloudformation将开始销毁相关堆栈。
