## 后续步骤

我们建议您尝试自行扩展代码库，添加以下功能：

1. 为项目添加<Link path="guides/license">许可证生成器</Link>，自动管理工作区中的LICENSE文件和源代码头部
2. <Link path="get_started/tutorials/contribute-generator">贡献自己的生成器</Link>来简化新tRPC API的添加
3. 在GameAPI中新增`resetToAction` API（使用上述生成器），允许回溯时间并删除指定时间点后的所有操作。然后更新UI，在对话气泡添加按钮实现该功能（并调用新建的API）
4. 探索其他可用指南

### 销毁资源

要销毁已创建的AWS资源，请运行以下命令：

<NxCommands commands={['run @dungeon-adventure/infra:destroy dungeon-adventure-infra-sandbox/*']} />

该命令将提示您输入要删除的堆栈列表，通常包括`dungeon-adventure-infra-sandbox/Application/GameUI/waf`和`dungeon-adventure-infra-sandbox/Application`。输入`Y`后，Cloudformation将开始销毁这些堆栈。
