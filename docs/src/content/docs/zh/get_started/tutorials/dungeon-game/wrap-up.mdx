## 下一步建议

我们建议你尝试自行扩展代码库，实现以下功能：

1. 为项目添加<Link path="guides/license">许可证生成器</Link>，自动管理工作区内的LICENSE文件及源码头部信息。
2. <Link path="get_started/tutorials/contribute-generator">贡献自定义生成器</Link>，简化新增tRPC API的流程。
3. 在GameAPI中新增`resetToAction` API（使用上述生成器），允许回退时间线并删除指定时间点后的所有操作。随后更新UI，在对话气泡添加按钮以触发此API。
4. 探索更多可用指南。

### 资源清理

要销毁已创建的AWS资源，请执行以下命令：

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

此命令将列出待删除的堆栈列表，通常包含`waf`和`dungeon-adventure-infra-sandbox`。输入`Y`后，Cloudformation将开始销毁这些堆栈。
