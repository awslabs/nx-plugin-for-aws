## 后续步骤

我们建议你尝试独立扩展代码库，添加以下功能：

1. 为项目添加<Link path="guides/license">许可证生成器</Link>，自动管理工作区中的 LICENSE 文件和源代码头部信息。
2. <Link path="get_started/tutorials/create-generator">创建自定义生成器</Link>以简化新 tRPC API 的添加流程。
3. 在 GameAPI 中新增 `resetToAction` API（使用上述生成器），实现时间回溯功能，删除指定时间点后的所有操作。随后更新 UI，在对话气泡添加按钮调用该 API。
4. 探索更多可用指南

### 销毁资源

如需销毁已创建的 AWS 资源，运行以下命令：

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

此操作将提示选择待删除的堆栈列表，通常包含 `waf` 和 `dungeon-adventure-infra-sandbox`。输入 `Y` 后 Cloudformation 将销毁相关堆栈。
