Esta página ofrece orientación para solucionar problemas que puedas encontrar al usar Nx.

## Nx Daemon

El [Nx Daemon](https://nx.dev/docs/concepts/nx-daemon) es un proceso que monitorea continuamente los cambios en el espacio de trabajo para recalcular su gráfico de proyectos. Esto mejora el rendimiento, especialmente en proyectos grandes, pero en algunos entornos podría causar problemas.

Esto puede manifestarse de diferentes formas, una de ellas es el siguiente error:

```bash
 NX   Could not find project "@scope/project"

Pass --verbose to see the stacktrace.
```

Para este y cualquier otro error inesperado, siempre vale la pena reiniciar Nx y reintentar por si hay un problema del daemon. Puedes usar el siguiente comando:

<NxCommands commands={['reset']} />

Si esto no funciona, puedes intentar [desactivar el daemon completamente](https://nx.dev/docs/concepts/nx-daemon#turning-it-off).

:::warning
El comando `nx watch` requiere el daemon, por lo que algunos de tus objetivos de recarga en caliente podrían dejar de funcionar si lo desactivas.
:::
