## Resumen

En este tutorial construirás un juego de aventuras tipo mazmorra impulsado por IA. Este tutorial no asume ningún conocimiento previo del `@aws/nx-plugin` o tecnologías relacionadas. Las técnicas que aprenderás son fundamentales para construir cualquier aplicación basada en `@aws/nx-plugin`, y comprenderlo completamente te dará un conocimiento profundo de esta herramienta y una comprensión básica de cómo usar [NX](https://nx.dev/).

<Aside>Este tutorial está diseñado para personas que prefieren aprender haciendo y quieren crear algo tangible rápidamente. Si prefieres aprender cada concepto paso a paso, consulta las guías individuales de componentes en la navegación izquierda.</Aside>

Los usuarios que completen este tutorial aprenderán a:

- Usar el `@aws/nx-plugin` para crear nuevas aplicaciones
- Utilizar NX para gestionar/construir su base de código
- Construir APIs usando tRPC y FastAPI
- Usar TanStack Router para crear nuevas páginas
- Emplear TanStack Query para llamar a APIs backend
- Hacer modificaciones al código generado para adaptarlo a propósitos específicos
- Crear e implementar infraestructura CDK

### Componentes de la aplicación

Construiremos el juego usando la siguiente arquitectura de componentes:

<Image class="white-bg" src={dungeonAdventureArchitecturePng} alt="dungeon-game-architecture.png" width="800" height="600" />

1. La API [tRPC](https://trpc.io/) usa [ElectroDB](https://electrodb.dev/)/[DynamoDB](https://aws.amazon.com/dynamodb/) para gestionar el estado del juego
2. Función Lambda Python/[FastAPI](https://fastapi.tiangolo.com/) con respuestas en streaming para generación de historias (con [Amazon Bedrock](https://aws.amazon.com/bedrock/))
3. Frontend React/Vite que utiliza:
  - Amazon Cognito/Identity Pools para llamadas API seguras
  - [TanStack Router](https://tanstack.com/router/latest) con enrutado basado en archivos y tipado seguro
  - SDKs generados para consumir las APIs de Juego e Historia (usando [TanStack Query](https://tanstack.com/query/latest))

Una infraestructura [CDK](https://docs.aws.amazon.com/cdk/v2/guide/home.html) para implementar los componentes de la aplicación.
