## ¿Qué sigue?

Te recomendamos intentar extender el código con las siguientes capacidades por tu cuenta:

1. Agrega el <Link path="guides/license">generador de licencias</Link> a tu proyecto, el cual administrará automáticamente archivos LICENSE y encabezados en el código fuente de tu espacio de trabajo.
2. <Link path="get_started/tutorials/create-generator">Crea tu propio generador</Link> para simplificar la adición de nuevas APIs tRPC.
3. Agrega una nueva API `resetToAction` (usando tu generador) en GameAPI que permita retroceder en el tiempo y eliminar todas las acciones desde un punto dado. Luego actualiza la UI para añadir un botón en una burbuja de conversación que reinicie a este punto (y llame a la nueva API).
4. Explora las diversas guías disponibles

### Eliminar recursos

Para destruir los recursos de AWS que se crearon, ejecuta el siguiente comando:

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

Esto te pedirá una lista de pilas para eliminar, que deben incluir `waf` y `dungeon-adventure-infra-sandbox`. Ingresa `Y` y luego Cloudformation eliminará tus pilas.
