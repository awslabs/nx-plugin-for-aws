## ¿Qué sigue?

Te recomendamos intentar extender la base de código con las siguientes capacidades por tu cuenta:

1. Agrega el <Link path="guides/license">generador de licencias</Link> a tu proyecto, el cual administrará automáticamente archivos LICENSE y encabezados en el código fuente de tu espacio de trabajo.
2. <Link path="get_started/tutorials/contribute-generator">Contribuye con tu propio generador</Link> para simplificar la adición de nuevas APIs tRPC.
3. Agrega una nueva API `resetToAction` (usando tu generador anterior) en GameAPI que permita retroceder en el tiempo y eliminar todas las acciones desde un punto dado. Actualiza la UI para añadir un botón en una burbuja de conversación que reinicie a este punto (y llame a la API recién creada).
4. Explora las diversas guías de componentes usando la navegación izquierda.

## Destrucción de recursos

1. Para eliminar los recursos de AWS que se crearon, ejecuta el siguiente comando:

<NxCommands commands={['run @dungeon-adventure/infra:destroy dungeon-adventure-infra-sandbox/*']} />

    Esto te pedirá una lista de pilas para eliminar, que deben incluir `dungeon-adventure-infra-sandbox/Application/GameUI/waf` y `dungeon-adventure-infra-sandbox/Application`. 

2. Ingresa `Y` para continuar. CloudFormation destruirá tus pilas.
