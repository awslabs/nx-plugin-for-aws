## ¿Qué sigue?

Te recomendamos intentar extender la base de código con las siguientes capacidades por tu cuenta:

1. Agrega el <Link path="guides/license">generador de licencias</Link> a tu proyecto para gestionar automáticamente archivos LICENSE y cabeceras en el código fuente.
2. <Link path="get_started/tutorials/create-generator">Crea tu propio generador</Link> para simplificar la adición de nuevas APIs de tRPC.
3. Añade una nueva API `resetToAction` (usando tu generador) en GameAPI que permita retroceder en el tiempo y borrar acciones desde un punto dado. Luego actualiza la UI para agregar un botón en los diálogos que active esta función.
4. Explora las diversas guías disponibles

### Eliminar recursos

Para destruir los recursos de AWS creados, ejecuta este comando:

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

Esto mostrará una lista de stacks para eliminar, que incluirán `waf` y `dungeon-adventure-infra-sandbox`. Ingresa `Y` y CloudFormation eliminará tus pilas.
