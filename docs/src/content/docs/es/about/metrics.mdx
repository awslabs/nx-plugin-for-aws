---
title: "Métricas de Uso"
description: "Cómo @aws/nx-plugin recopila métricas"
---



Para ayudarnos a mejorar el Nx Plugin para AWS, recopilamos métricas sobre el uso de los generadores. Este proceso no involucra solicitudes de red adicionales, simplemente aprovechamos las métricas de uso existentes de AWS.

Para mantener la transparencia, agregamos un ID, versión y lista de IDs de generadores a las descripciones de Stack de CloudFormation. Esto se realiza en el `MetricsAspect` dentro de `common/constructs`.
