---
title: Registro de Consultas PostgreSQL
---

El registro `postgresql` se exporta para [Aurora PostgreSQL](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraPostgreSQL.CloudWatch.html), con el registro limitado solo a sentencias DDL (`log_statement=ddl`) para que los valores de los parámetros de las sentencias nunca se registren, siempre y cuando cada sentencia se envíe por separado. `log_statement=ddl` registra el texto completo sin procesar de un lote de múltiples sentencias (por ejemplo, una sola llamada `psql -c "a;b;c"`) de forma literal si alguna sentencia en él es DDL, incluyendo cualquier valor DML en ese mismo lote.
