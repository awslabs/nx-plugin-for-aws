---
title: Registro de Consultas MySQL
---

Los registros `audit` y `error` se exportan para [Aurora MySQL](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraMySQL.Integrating.CloudWatch.html) — `general` y `slowquery` se excluyen deliberadamente ya que registran el texto completo de las sentencias, incluyendo valores DML. [Advanced Auditing](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraMySQL.Auditing.html) está limitado a conexiones y DDL (`server_audit_events=CONNECT,QUERY_DDL`), por lo que los valores de los parámetros de las sentencias nunca se registran.
