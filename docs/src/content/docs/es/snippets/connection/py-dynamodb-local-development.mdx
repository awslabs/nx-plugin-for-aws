---
title: Desarrollo Local de Python DynamoDB
---

El generador `connection` configura el target `serve-local` de tu proyecto para que dependa del target `serve-local` del proyecto DynamoDB. DynamoDB Local se iniciará automáticamente junto con tu proyecto al ejecutar `serve-local`.

La variable de entorno `SERVE_LOCAL=true` se establece automáticamente, por lo que `is_local()` devuelve `True` y tus entidades PynamoDB se conectan a la instancia local de DynamoDB en lugar de AWS.
