---
title: Desarrollo Local de DynamoDB
---

El generador `connection` configura el target `dev` de tu proyecto para que dependa del target `dev` del proyecto DynamoDB. DynamoDB Local se iniciará automáticamente junto con tu proyecto al ejecutar `dev`.

La variable de entorno `LOCAL_DEV=true` se establece automáticamente, por lo que `getDynamoDBClient()` y `resolveTableName()` se conectan a la instancia local de DynamoDB Local en lugar de AWS.
