#### WAF

Para las REST APIs, el constructo CDK generado asocia un AWS WAFv2 Web ACL con el stage predeterminado de la API por defecto. El Web ACL utiliza el conjunto de reglas predeterminado administrado por AWS (`AWSManagedRulesCommonRuleSet` y `AWSManagedRulesKnownBadInputsRuleSet`), proporcionando protección contra exploits web comunes, incluyendo el OWASP Top 10.

Para desactivarlo (por ejemplo, para adjuntar tu propio Web ACL), establece `enableWaf` en `false`:

```ts {4}
const api = new MyApi(this, 'MyApi', {
  integrations: MyApi.defaultIntegrations(this).build(),
  enableWaf: false,
});
```
