---
title: Configuración de WAF en Terraform
---

:::note[Configuración de WAF para REST API]
Para las REST APIs, el módulo de Terraform generado asocia un AWS WAFv2 Web ACL con el stage de API Gateway de forma predeterminada. El Web ACL utiliza el conjunto de reglas predeterminado administrado por AWS (`AWSManagedRulesCommonRuleSet` y `AWSManagedRulesKnownBadInputsRuleSet`), proporcionando protección contra exploits web comunes, incluidos los del OWASP Top 10.

Si necesita desactivarlo (por ejemplo, para adjuntar su propio Web ACL), establezca `enable_waf` en `false`:

```hcl {4}
module "my_api" {
  source = "../../common/terraform/src/app/apis/my-api"

  enable_waf = false # Disable the default WAF (not recommended for production)
}
```
:::
