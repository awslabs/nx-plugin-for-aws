[AWS Arch](https://aws.github.io/aws-pdk/developer_guides/aws-arch/index.html) proporciona mapeos entre recursos de CloudFormation y sus iconos de arquitectura asociados para CDK Graph.

Consulta la [página de iconos de arquitectura de AWS](https://aws.amazon.com/architecture/icons/) para recursos relacionados con iconos. [Diagrams](https://diagrams.mingrammer.com/) también ofrece una forma de construir diagramas como código.

Si estabas usando esto directamente, ¡considera hacer un fork del proyecto y tomar posesión del mismo!
