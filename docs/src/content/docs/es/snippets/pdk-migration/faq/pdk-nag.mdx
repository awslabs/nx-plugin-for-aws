[PDK Nag](https://aws.github.io/aws-pdk/developer_guides/pdk-nag/index.html) envuelve [CDK Nag](https://github.com/cdklabs/cdk-nag) y proporciona un conjunto de [reglas específicas para la creación de prototipos](https://github.com/aws/aws-pdk/blob/mainline/packages/pdk-nag/src/packs/aws-prototyping-rules.ts).

Para migrar desde PDK Nag, utiliza CDK Nag directamente. Si necesitas el mismo conjunto de reglas, puedes crear tu propio "pack" siguiendo la [documentación aquí](https://github.com/cdklabs/cdk-nag/blob/main/docs/NagPack.md).
