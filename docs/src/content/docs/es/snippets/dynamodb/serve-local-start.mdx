---
title: Iniciando DynamoDB Local
---
import NxCommands from '@components/nx-commands.astro';

El generador configura un target `serve-local` que inicia una instancia de [DynamoDB Local](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/DynamoDBLocal.html) y crea la tabla. Usa el target `dev` del proyecto (un alias para `serve-local`):

<NxCommands commands={['run <project-name>:dev']} />

Esto automáticamente:
1. Descarga la imagen de DynamoDB Local (target `pull-image`)
2. Inicia un contenedor
3. Crea una tabla local con los índices definidos en `config.json`
