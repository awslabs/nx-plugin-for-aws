Dado que este generador proporciona infraestructura como código basada en tu proveedor de `iacProvider` seleccionado, creará un proyecto en `packages/common` que incluye los constructos CDK o módulos de Terraform correspondientes.

El proyecto común de infraestructura como código tiene la siguiente estructura:

<Infrastructure>
<Fragment slot="cdk">
<FileTree>
  - packages/common/constructs
    - src
      - app/ Constructos para infraestructura específica de un proyecto/generador
      - core/ Constructos genéricos reutilizados por los constructos en `app`
      - index.ts Punto de entrada que exporta los constructos de `app`
    - project.json Objetivos de compilación y configuración del proyecto
</FileTree>

:::note
Este proyecto se genera utilizando el <Link path="/guides/typescript-project">`ts#project`</Link> generator y por lo tanto configura los mismos objetivos de compilación.
:::
</Fragment>
<Fragment slot="terraform">
<FileTree>
  - packages/common/terraform
    - src
      - app/ Módulos de Terraform para infraestructura específica de un proyecto/generador
      - core/ Módulos genéricos reutilizados por los módulos en `app`
    - project.json Objetivos de compilación y configuración del proyecto
</FileTree>

:::note
Este proyecto se genera utilizando el <Link path="/guides/terraform-project">`terraform#project`</Link> generator y por lo tanto configura los mismos objetivos de compilación.
:::
</Fragment>
</Infrastructure>
