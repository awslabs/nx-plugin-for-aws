---
title: "Observabilidad de MCP"
---



Tu servidor MCP se configura automáticamente con observabilidad utilizando la [AWS Distro for Open Telemetry](https://aws.amazon.com/otel/) (ADOT), mediante la configuración de auto-instrumentación en tu `Dockerfile`.

Puedes encontrar trazas en la consola de AWS CloudWatch seleccionando "GenAI Observability" en el menú. Ten en cuenta que para que las trazas se rellenen, deberás habilitar [Transaction Search](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-Transaction-Search.html).

Para más detalles, consulta la [documentación de AgentCore sobre observabilidad](https://docs.aws.amazon.com/bedrock-agentcore/latest/devguide/observability-configure.html).
