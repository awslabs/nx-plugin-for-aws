---
title: "Requisitos previos"
---



- Instala [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git).
- Instala [Node >= 22](https://nodejs.org/en/download) (Recomendamos usar [NVM](https://github.com/nvm-sh/nvm) para gestionar tus versiones de Node)
  - Para verificar tu versión, ejecuta `node --version` en la ventana de terminal.
- Instala [PNPM >= 10](https://pnpm.io/installation#using-npm) (También puedes usar [Yarn >= 4](https://yarnpkg.com/getting-started/install), [Bun >= 1](https://bun.sh/docs/installation) o [NPM >= 10](https://nodejs.org/en/learn/getting-started/an-introduction-to-the-npm-package-manager))
  - Para verificar tu versión, ejecuta `pnpm --version`, `yarn --version`, `bun --version` o `npm --version` en tu ventana de terminal.
- Instala [UV >= 0.5.29](https://docs.astral.sh/uv/getting-started/installation/)
  1. Para instalar Python 3.1, ejecuta `uv python install 3.12.0`.
  2. Verifica usando `uv python list --only-installed`
- Configura tus [AWS Credentials](https://docs.aws.amazon.com/sdkref/latest/guide/access.html) para tu cuenta de AWS objetivo (donde se desplegará tu aplicación).
- Si usas [VSCode](https://code.visualstudio.com/), recomendamos instalar el [Nx Console VSCode Plugin](https://marketplace.visualstudio.com/items?itemName=nrwl.angular-console).
