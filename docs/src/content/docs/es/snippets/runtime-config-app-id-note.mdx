:::note[Configuración en tiempo de ejecución]
Esto depende de la variable de entorno `RUNTIME_CONFIG_APP_ID` para obtener la configuración de AWS AppConfig en tiempo de ejecución. Los proyectos construidos con este plugin (APIs, Agents, servidores MCP) ya tienen esta variable configurada automáticamente. Para otros proyectos, asegúrese de que `RUNTIME_CONFIG_APP_ID` esté establecida en el entorno de ejecución con el ID de aplicación de AppConfig aprovisionado por su infraestructura. Para más información, consulte la <Link path="guides/runtime-config">guía de Configuración en tiempo de ejecución</Link>.
:::
