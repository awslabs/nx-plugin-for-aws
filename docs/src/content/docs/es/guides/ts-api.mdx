El generador de TypeScript API crea un nuevo proyecto de API con el framework de su elección. Soporta [tRPC](https://trpc.io/) para APIs con seguridad de tipos de extremo a extremo y [Smithy](https://smithy.io/) para desarrollo de APIs dirigido por modelos.
