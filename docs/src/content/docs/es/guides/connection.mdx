Este generador se utiliza para conectar proyectos entre sí, como sitios web que llaman a APIs. Simplemente selecciona el proyecto origen (por ejemplo, el proyecto que llamará a tu API) y el proyecto destino (por ejemplo, tu proyecto de API), y este generador se encargará de integrar ambos.

### Conexiones admitidas

El generador de Conexión admite las siguientes conexiones:

:::note[Configuración en Tiempo de Ejecución]
El generador de conexión hace uso de <Link path="guides/runtime-config">Configuración en Tiempo de Ejecución</Link> para pasar valores en tiempo de despliegue (como URLs de API, configuraciones de Cognito y ARNs de tiempo de ejecución de agentes) entre proyectos y componentes generados en tiempo de ejecución para que puedan descubrirse y conectarse entre sí.
:::
