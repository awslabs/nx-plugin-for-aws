El generador `ts#website` crea una nueva aplicación de sitio web. Puedes elegir qué framework de frontend utilizar mediante la opción `framework` — actualmente se soporta [React](https://react.dev/).

Para información detallada sobre la estructura del proyecto generado, configuración y uso, consulta la guía específica del framework:

- <Link path="/guides/react-website">React Website</Link>

## Uso

### Generar un Website

Puedes generar un nuevo sitio web de dos maneras:
