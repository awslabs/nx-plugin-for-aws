El generador `connection` conecta una <Link path="guides/fastapi">FastAPI</Link> a un proyecto <Link path="guides/py-dynamodb">Python DynamoDB</Link>, configurando el desarrollo local para que ambos se inicien juntos automáticamente.

## Requisitos previos

Antes de usar este generador, asegúrate de tener:

1. Un proyecto <Link path="guides/fastapi">`py#fast-api`</Link>
2. Un proyecto <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Uso

### Ejecutar el generador

<RunGenerator generator="connection" />

Selecciona tu proyecto FastAPI como origen y tu proyecto DynamoDB como destino.

### Opciones

<GeneratorParameters generator="connection" />

## Salida del generador

El generador actualiza el `project.json` de FastAPI para agregar una dependencia desde su objetivo `serve-local` al objetivo `serve-local` del proyecto DynamoDB. No se modifican archivos de código fuente.
