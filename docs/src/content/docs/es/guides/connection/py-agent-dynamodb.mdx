El generador `connection` conecta un <Link path="guides/py-agent">Python Agent</Link> a un proyecto <Link path="guides/py-dynamodb">Python DynamoDB</Link>, configurando el desarrollo local para que ambos se inicien juntos automáticamente.

## Requisitos previos

Antes de usar este generador, asegúrate de tener:

1. Un proyecto <Link path="guides/py-agent">`py#agent`</Link>
2. Un proyecto <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Uso

### Ejecutar el generador

<RunGenerator generator="connection" />

Selecciona tu proyecto Agent como origen y tu proyecto DynamoDB como destino. Si el proyecto contiene múltiples componentes de agente, especifica `sourceComponent` para desambiguar.

### Opciones

<GeneratorParameters generator="connection" />

## Salida del generador

El generador actualiza el target `<agent-name>-serve-local` del agente en `project.json` para que dependa del target `serve-local` del proyecto DynamoDB, y agrega el paquete DynamoDB como una dependencia del workspace. No se modifican archivos de código fuente.

## Uso de DynamoDB en Agents

Importa las clases de entidad del paquete DynamoDB y úsalas dentro de las herramientas de tu agente:

```python title="packages/my_project/my_project/my_agent/agent.py"
from my_scope.my_table.entities.example import ExampleModel
from strands import tool

@tool
def list_examples() -> list:
    """List all example items."""
    return [item.attribute_values for item in ExampleModel.scan()]
```

## Infraestructura

Para permitir que la función Lambda del agente acceda a la tabla DynamoDB, otorga los permisos necesarios en tu infraestructura.

<Infrastructure>
<Fragment slot="cdk">

```ts title="packages/infra/src/stacks/application-stack.ts"
import { MyTable } from ':my-scope/common-constructs';

const table = new MyTable(this, 'Table');
const myAgent = new MyAgent(this, 'MyAgent');

table.grantReadWriteData(myAgent);
```

`grantReadWriteData` otorga tanto los permisos de DynamoDB como de KMS al rol de ejecución del agente.
</Fragment>
<Fragment slot="terraform">

```hcl title="packages/infra/src/main.tf"
module "my_table" {
  source = "../../common/terraform/src/app/dynamodb/my-table"
}

resource "aws_iam_role_policy" "dynamodb_access" {
  role = module.my_agent.lambda_role_name

  policy = jsonencode({
    Version = "2012-10-17"
    Statement = [
      {
        Effect = "Allow"
        Action = [
          "dynamodb:GetItem", "dynamodb:PutItem", "dynamodb:UpdateItem",
          "dynamodb:DeleteItem", "dynamodb:Query", "dynamodb:Scan",
          "dynamodb:BatchGetItem", "dynamodb:BatchWriteItem",
        ]
        Resource = [
          module.my_table.table_arn,
          "${module.my_table.table_arn}/index/*",
        ]
      },
      {
        Effect = "Allow"
        Action = [
          "kms:Encrypt",
          "kms:Decrypt",
          "kms:ReEncrypt*",
          "kms:GenerateDataKey*",
          "kms:DescribeKey"
        ]
        Resource = [module.my_table.kms_key_arn]
      },
    ]
  })
}
```
