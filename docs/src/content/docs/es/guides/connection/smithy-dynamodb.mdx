El generador `connection` conecta una <Link path="guides/ts-smithy-api">API Smithy</Link> a un proyecto de <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link>, configurando el desarrollo local para que ambos se inicien juntos automáticamente.

## Requisitos previos

Antes de usar este generador, asegúrate de tener:

1. Un proyecto <Link path="guides/ts-smithy-api">Smithy TypeScript API</Link> (generado con `ts#api` usando `--framework=smithy`)
2. Un proyecto <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link>

## Uso

### Ejecutar el generador

<RunGenerator generator="connection" />

Selecciona tu proyecto de backend de API Smithy como origen y tu proyecto de DynamoDB como destino.

## Salida del generador

El generador actualiza el `project.json` de tu API Smithy para agregar una dependencia desde su target `dev` al target `dev` del proyecto de DynamoDB. No se modifican archivos de código fuente.

## Usar DynamoDB en operaciones

Importa las factories de entidades desde el paquete de DynamoDB y úsalas dentro de las implementaciones de tus operaciones:

```ts title="packages/api/src/operations/list-examples.ts"
import { createExampleEntity } from ':my-scope/my-table';
import {
  ListExamplesOperationInput,
  ListExamplesOperationOutput,
} from '../generated/ssdk/index.js';
import { ServiceContext } from '../context.js';

export const listExamples = async (
  _input: ListExamplesOperationInput,
  _ctx: ServiceContext,
): Promise<ListExamplesOperationOutput> => {
  const entity = await createExampleEntity();
  const result = await entity.scan.go();
  return { items: result.data };
};
```
