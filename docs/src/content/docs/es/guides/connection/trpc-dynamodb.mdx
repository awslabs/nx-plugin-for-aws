El generador `connection` conecta una <Link path="guides/trpc">API tRPC</Link> a un proyecto <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link>, configurando el desarrollo local para que ambos se inicien juntos automáticamente.

## Requisitos previos

Antes de usar este generador, asegúrate de tener:

1. Un proyecto <Link path="guides/trpc">tRPC API</Link> (generado con `ts#api`)
2. Un proyecto <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link>

## Uso

### Ejecutar el generador

<RunGenerator generator="connection" />

<GeneratorParameters generator="connection" />

## Salida del generador

El generador actualiza el `project.json` de tu API tRPC para agregar una dependencia desde su objetivo `dev` al objetivo `dev` del proyecto DynamoDB. No se modifican archivos de código fuente.

## Usar DynamoDB en procedimientos

Importa las factories de entidades desde el paquete DynamoDB y úsalas dentro de tus procedimientos tRPC:

```ts title="packages/api/src/procedures/example.ts"
import { createExampleEntity } from ':my-scope/my-table';
import { publicProcedure } from '../init.js';

export const listExamples = publicProcedure
  .query(async () => {
    const entity = await createExampleEntity();
    const result = await entity.scan.go();
    return result.data;
  });
```
