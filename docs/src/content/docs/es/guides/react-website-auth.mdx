El generador de autenticación para sitios web React añade autenticación a tu sitio web React usando [Amazon Cognito](https://aws.amazon.com/cognito/).

Este generador configura la infraestructura CDK para crear un User Pool de Cognito y un Identity Pool asociado, así como una interfaz de usuario alojada para manejar flujos de inicio de sesión y su integración con tu sitio web React.

## Uso

### Añadir autenticación a tu sitio web React

Puedes añadir autenticación a tu sitio web React de dos formas:

<RunGenerator generator="ts#react-website#auth" />

### Opciones

<GeneratorParameters generator="ts#react-website#auth" />

## Uso de la infraestructura

Necesitarás añadir el constructo `UserIdentity` a tu stack, declarándolo _antes_ del constructo del sitio web:

```ts title="packages/infra/src/stacks/application-stack.ts" {3,9}
import { Stack } from 'aws-cdk-lib';
import { Construct } from 'constructs';
import { MyWebsite, UserIdentity } from ':my-scope/common-constructs';

export class ApplicationStack extends Stack {
  constructor(scope: Construct, id: string) {
    super(scope, id);

    new UserIdentity(this, 'Identity');

    new MyWebsite(this, 'MyWebsite');
  }
}
```

El constructo `UserIdentity` añade automáticamente la <Link path="guides/react-website#runtime-configuration">Configuración en tiempo de ejecución</Link> necesaria para garantizar que tu sitio web pueda apuntar al User Pool de Cognito correcto para la autenticación.

### Otorgar acceso a usuarios autenticados

Para permitir que usuarios autenticados realicen ciertas acciones, como conceder permisos para invocar una API, puedes añadir políticas IAM al rol autenticado del identity pool:

```ts title="packages/infra/src/stacks/application-stack.ts" {12}
import { Stack } from 'aws-cdk-lib';
import { Construct } from 'constructs';
import { MyWebsite, UserIdentity, MyApi } from ':my-scope/common-constructs';

export class ApplicationStack extends Stack {
  constructor(scope: Construct, id: string) {
    super(scope, id);

    const identity = new UserIdentity(this, 'Identity');
    const api = new MyApi(this, 'MyApi');

    api.grantInvokeAccess(identity.identityPool.authenticatedRole);

    new MyWebsite(this, 'MyWebsite');
  }
}
```
