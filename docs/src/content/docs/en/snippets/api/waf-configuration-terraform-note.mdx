#### WAF

For REST APIs, the generated Terraform module associates an AWS WAFv2 Web ACL with the API Gateway stage by default. The Web ACL uses the AWS managed default ruleset (`AWSManagedRulesCommonRuleSet` and `AWSManagedRulesKnownBadInputsRuleSet`), providing protection against common web exploits including the OWASP Top 10.

To opt out (for example, to attach your own Web ACL), set `enable_waf` to `false`:

```hcl {4}
module "my_api" {
  source = "../../common/terraform/src/app/apis/my-api"

  enable_waf = false
}
```
