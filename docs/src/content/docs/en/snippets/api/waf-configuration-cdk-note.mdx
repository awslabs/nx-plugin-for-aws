#### WAF

For REST APIs, the generated CDK construct associates an AWS WAFv2 Web ACL with the API's default stage by default. The Web ACL uses the AWS managed default ruleset (`AWSManagedRulesCommonRuleSet` and `AWSManagedRulesKnownBadInputsRuleSet`), providing protection against common web exploits including the OWASP Top 10.

To opt out (for example, to attach your own Web ACL), set `enableWaf` to `false`:

```ts {4}
const api = new MyApi(this, 'MyApi', {
  integrations: MyApi.defaultIntegrations(this).build(),
  enableWaf: false,
});
```
