import ProjectBuilder from '@components/project-builder/ProjectBuilder.tsx';

Design your project visually by dragging generators onto the canvas, configuring their options, and connecting them together. When you're done, copy the generated CLI commands to scaffold your project.

**How to use:**
1. **Drag** generators from the left panel onto the canvas (or click to add)
2. **Configure** each generator's name and options directly in the node
3. **Connect** generators by dragging from a right handle to a left handle (only valid connections are allowed)
4. **Adjust** global settings (workspace name, package manager, IaC provider) in the top bar
5. **Copy** the generated CLI commands with the button below the canvas
