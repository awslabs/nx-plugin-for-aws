The `ts#website` generator creates a new website application. You can choose which frontend framework to use via the `framework` option — currently [React](https://react.dev/) is supported.

For detailed information about the generated project structure, configuration, and usage, see the framework-specific guide:

- <Link path="/guides/react-website">React Website</Link>
