## Tarefa 1: Implementar a API do Jogo

Vamos implementar as seguintes APIs nesta seção:

1. `saveGame` - criar ou atualizar um jogo.
2. `queryGames` - retornar uma lista paginada de jogos salvos anteriormente.
3. `saveAction` - salvar uma ação para um determinado jogo.
4. `queryActions` - retornar uma lista paginada de todas as ações relacionadas a um jogo.
5. `queryInventory` - retornar uma lista paginada de itens no inventário de um jogador.

Observe que como cada procedimento é atendido por uma função lambda individual, podemos seguir o princípio de menor privilégio e atribuir apenas as permissões de leitura/escrita necessárias com base na implementação do procedimento.
:::

:::tip[Configuração de Runtime]
O nome da tabela DynamoDB é registrado na <Link path="guides/runtime-config">Configuração de Runtime</Link> sob o namespace `tables`, armazenado no AWS AppConfig. As funções Lambda e o servidor MCP o recuperam em tempo de execução usando Lambda Powertools, em vez de precisar ser passado como uma variável de ambiente. Um benefício dessa abordagem é que, à medida que o número de tabelas ou outros identificadores de infraestrutura cresce, você precisa apenas de uma única variável de ambiente `RUNTIME_CONFIG_APP_ID` ao executar localmente.
:::

### Implantar sua aplicação
Para implantar sua aplicação, execute o seguinte comando:

<NxCommands commands={['deploy infra "dungeon-adventure-infra-sandbox/*"']} />

:::caution[Bootstrap necessário]
Para fazer o bootstrap da sua conta AWS primeiro, execute:

<NxCommands commands={['bootstrap infra']} />
:::

Sua primeira implantação levará cerca de 8 minutos para ser concluída. Implantações subsequentes levarão cerca de 2 minutos.

:::tip[Iteração rápida]
Se estiver iterando em alterações de código de funções Lambda, você pode implantar com a flag `--hotswap` após construir a base de código para um tempo de implantação muito menor (2-3 segundos).

<NxCommands commands={['deploy infra "dungeon-adventure-infra-sandbox/*" --hotswap']} />
:::

Uma vez concluída a implantação, você verá saídas semelhantes às seguintes _(alguns valores foram omitidos)_:
