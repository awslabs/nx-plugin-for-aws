import Link from '@components/link.astro';

:::note[Configuração em tempo de execução]
Isso depende da variável de ambiente `RUNTIME_CONFIG_APP_ID` para buscar a configuração do AWS AppConfig em tempo de execução. Projetos construídos com este plugin (APIs, Agents, servidores MCP) já têm essa variável configurada automaticamente. Para outros projetos, certifique-se de que `RUNTIME_CONFIG_APP_ID` esteja definida no ambiente de execução com o ID da aplicação AppConfig provisionado pela sua infraestrutura. Para mais informações, consulte o <Link path="guides/runtime-config">guia de Configuração em Tempo de Execução</Link>.
:::
