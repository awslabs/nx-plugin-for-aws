---
title: Desenvolvimento Local com Python DynamoDB
---

O gerador `connection` configura o target `dev` do seu projeto para depender do target `dev` do projeto DynamoDB. O DynamoDB Local será iniciado automaticamente junto com seu projeto ao executar `dev`.

A variável de ambiente `LOCAL_DEV=true` é definida automaticamente, então `is_local()` retorna `True` e suas entidades PynamoDB se conectam à instância local do DynamoDB em vez da AWS.
