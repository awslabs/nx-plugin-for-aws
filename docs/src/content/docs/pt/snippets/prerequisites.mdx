---
title: "Pré-requisitos"
---



- Instale o [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git).
- Instale o [Node >= 22](https://nodejs.org/en/download) (Recomendamos usar o [NVM](https://github.com/nvm-sh/nvm) para gerenciar versões do Node)
  - Para verificar sua versão, execute `node --version` na janela do terminal.
- Instale o [PNPM >= 10](https://pnpm.io/installation#using-npm) (Você também pode usar [Yarn >= 4](https://yarnpkg.com/getting-started/install), [Bun >= 1](https://bun.sh/docs/installation) ou [NPM >= 10](https://nodejs.org/en/learn/getting-started/an-introduction-to-the-npm-package-manager))
  - Para verificar sua versão, execute `pnpm --version`, `yarn --version`, `bun --version` ou `npm --version` no terminal.
- Instale o [UV >= 0.5.29](https://docs.astral.sh/uv/getting-started/installation/)
  1. Para instalar o Python 3.12, execute `uv python install 3.12.0`.
  2. Verifique usando `uv python list --only-installed`
- Configure suas [Credenciais da AWS](https://docs.aws.amazon.com/sdkref/latest/guide/access.html) para a conta AWS de destino (onde seu aplicativo será implantado).
- Se você usa o [VSCode](https://code.visualstudio.com/), recomendamos instalar o [Plugin Nx Console para VSCode](https://marketplace.visualstudio.com/items?itemName=nrwl.angular-console).
