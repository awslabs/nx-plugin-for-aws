[AWS Arch](https://aws.github.io/aws-pdk/developer_guides/aws-arch/index.html) fornece mapeamentos entre recursos do CloudFormation e seus ícones de arquitetura associados para o CDK Graph acima.

Consulte a [AWS Architecture Icons page](https://aws.amazon.com/architecture/icons/) para recursos relacionados a ícones. [Diagrams](https://diagrams.mingrammer.com/) também oferece uma forma de construir diagramas como código.

Se você estiver usando isso diretamente, considere fazer um fork do projeto e assumir a responsabilidade!
