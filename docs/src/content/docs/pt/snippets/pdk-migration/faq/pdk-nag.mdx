O [PDK Nag](https://aws.github.io/aws-pdk/developer_guides/pdk-nag/index.html) envolve o [CDK Nag](https://github.com/cdklabs/cdk-nag) e fornece um conjunto de [regras específicas para construção de protótipos](https://github.com/aws/aws-pdk/blob/mainline/packages/pdk-nag/src/packs/aws-prototyping-rules.ts).

Para migrar do PDK Nag, use o CDK Nag diretamente. Se você precisa do mesmo conjunto de regras, pode criar seu próprio "pack" seguindo a [documentação aqui](https://github.com/cdklabs/cdk-nag/blob/main/docs/NagPack.md).
