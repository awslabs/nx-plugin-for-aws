#### WAF

Para REST APIs, o módulo Terraform gerado associa um AWS WAFv2 Web ACL com o estágio do API Gateway por padrão. O Web ACL usa o conjunto de regras padrão gerenciado pela AWS (`AWSManagedRulesCommonRuleSet` e `AWSManagedRulesKnownBadInputsRuleSet`), fornecendo proteção contra explorações web comuns, incluindo o OWASP Top 10.

Para desativar (por exemplo, para anexar seu próprio Web ACL), defina `enable_waf` como `false`:

```hcl {4}
module "my_api" {
  source = "../../common/terraform/src/app/apis/my-api"

  enable_waf = false
}
```
