---
title: "Escolha entre REST e HTTP"
---



:::note
Você pode escolher entre uma [API REST](https://docs.aws.amazon.com/apigateway/latest/developerguide/apigateway-rest-api.html) ou uma [API HTTP](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api.html) para implantar sua API. Consulte o [Guia do desenvolvedor do API Gateway](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api-vs-rest.html) para ajudá-lo a decidir.
:::
