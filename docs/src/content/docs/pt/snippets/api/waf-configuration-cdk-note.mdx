---
title: Configuração WAF CDK
---

:::note[Configuração WAF da REST API]
Para REST APIs, o construto CDK gerado associa um AWS WAFv2 Web ACL com o estágio padrão da API por padrão. O Web ACL usa o conjunto de regras padrão gerenciado pela AWS (`AWSManagedRulesCommonRuleSet` e `AWSManagedRulesKnownBadInputsRuleSet`), fornecendo proteção contra explorações web comuns, incluindo o OWASP Top 10.

Se você precisar desativar (por exemplo, para anexar seu próprio Web ACL), defina `enableWaf` como `false`:

```ts {4}
const api = new MyApi(this, 'MyApi', {
  integrations: MyApi.defaultIntegrations(this).build(),
  // Disable the default WAF (not recommended for production)
  enableWaf: false,
});
```
:::
