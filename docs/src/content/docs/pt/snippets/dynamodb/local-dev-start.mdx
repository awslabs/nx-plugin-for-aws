---
title: Iniciando DynamoDB Local
---
import NxCommands from '@components/nx-commands.astro';

O gerador configura um target `dev` que inicia uma instância do [DynamoDB Local](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/DynamoDBLocal.html) e cria a tabela. Use o target `dev` do projeto:

<NxCommands commands={['dev <project-name>']} />

Isso automaticamente:
1. Baixa a imagem do DynamoDB Local (target `pull-image`)
2. Inicia um contêiner
3. Cria uma tabela local com os índices definidos em `config.json`
