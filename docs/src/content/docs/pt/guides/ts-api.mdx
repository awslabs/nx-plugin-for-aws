O gerador TypeScript API cria um novo projeto de API com o framework de sua escolha. Suporta [tRPC](https://trpc.io/) para APIs type-safe de ponta a ponta e [Smithy](https://smithy.io/) para desenvolvimento de APIs orientado a modelos.
