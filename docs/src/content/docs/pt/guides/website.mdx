O gerador `ts#website` cria uma nova aplicação de website. Você pode escolher qual framework de frontend utilizar através da opção `framework` — atualmente [React](https://react.dev/) é suportado.

Para informações detalhadas sobre a estrutura do projeto gerado, configuração e uso, consulte o guia específico do framework:

- <Link path="/guides/react-website">React Website</Link>
