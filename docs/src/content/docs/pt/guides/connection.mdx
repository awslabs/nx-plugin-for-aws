Este gerador é usado para conectar projetos entre si, como sites chamando APIs. Basta selecionar o projeto de origem (por exemplo, o projeto que irá chamar sua API) e o projeto de destino (por exemplo, seu projeto de API), e este gerador cuidará da integração entre os dois.

### Conexões Suportadas

O gerador de Conexão suporta as seguintes conexões:

:::note[Configuração de Runtime]
O gerador de conexão utiliza <Link path="guides/runtime-config">Configuração de Runtime</Link> para passar valores de tempo de implantação (como URLs de API, configurações do Cognito e ARNs de runtime de agentes) entre projetos e componentes gerados em tempo de execução para que possam se descobrir e conectar uns aos outros.
:::

:::tip[Desenvolvimento Local]
Projetos conectados podem ser executados em sua máquina com os alvos `serve` e `dev`. Consulte o guia <Link path="guides/local-development">Desenvolvimento Local</Link> para mais detalhes.
:::
