O gerador `connection` conecta um <Link path="guides/py-mcp-server">Python MCP Server</Link> a um projeto <Link path="guides/py-dynamodb">Python DynamoDB</Link>, configurando o desenvolvimento local para que ambos iniciem juntos automaticamente.

## Pré-requisitos

Antes de usar este gerador, certifique-se de ter:

1. Um projeto <Link path="guides/py-mcp-server">`py#mcp-server`</Link>
2. Um projeto <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Uso

### Execute o Gerador

<RunGenerator generator="connection" />

Selecione seu projeto MCP server como origem e seu projeto DynamoDB como destino. Se o projeto contiver múltiplos componentes MCP server, especifique `sourceComponent` para desambiguar.

### Opções

<GeneratorParameters generator="connection" />

## Saída do Gerador

O gerador atualiza o target `<mcp-server-name>-serve-local` do MCP server em `project.json` para depender do target `serve-local` do projeto DynamoDB. Nenhum arquivo de código-fonte é modificado.
