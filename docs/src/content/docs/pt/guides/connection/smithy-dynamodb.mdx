O gerador `connection` conecta uma <Link path="guides/ts-smithy-api">API Smithy</Link> a um projeto <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link>, configurando o desenvolvimento local para que ambos iniciem juntos automaticamente.

## Pré-requisitos

Antes de usar este gerador, certifique-se de ter:

1. Um projeto <Link path="guides/ts-smithy-api">Smithy TypeScript API</Link> (gerado com `ts#api` usando `--framework=smithy`)
2. Um projeto <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link>

## Saída do Gerador

O gerador atualiza o `project.json` da sua API Smithy para adicionar uma dependência do seu target `serve-local` ao target `serve-local` do projeto DynamoDB. Nenhum arquivo de código-fonte é modificado.

## Usando DynamoDB em Operações

Importe as factories de entidade do pacote DynamoDB e use-as dentro das suas implementações de operação:

```ts title="packages/api/src/operations/list-examples.ts"
import { createExampleEntity } from ':my-scope/my-table';
import {
  ListExamplesOperationInput,
  ListExamplesOperationOutput,
} from '../generated/ssdk/index.js';
import { ServiceContext } from '../context.js';

export const listExamples = async (
  _input: ListExamplesOperationInput,
  _ctx: ServiceContext,
): Promise<ListExamplesOperationOutput> => {
  const entity = await createExampleEntity();
  const result = await entity.scan.go();
  return { items: result.data };
};
```
