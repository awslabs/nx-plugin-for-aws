O gerador `connection` conecta um <Link path="guides/fastapi">FastAPI</Link> a um projeto <Link path="guides/py-dynamodb">Python DynamoDB</Link>, configurando o desenvolvimento local para que ambos iniciem juntos automaticamente.

## Pré-requisitos

Antes de usar este gerador, certifique-se de ter:

1. Um projeto <Link path="guides/fastapi">`py#fast-api`</Link>
2. Um projeto <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Uso

### Executar o Gerador

<RunGenerator generator="connection" />

Selecione seu projeto FastAPI como origem e seu projeto DynamoDB como destino.

### Opções

<GeneratorParameters generator="connection" />

## Saída do Gerador

O gerador atualiza o `project.json` do FastAPI para adicionar uma dependência de seu target `serve-local` ao target `serve-local` do projeto DynamoDB. Nenhum arquivo de código-fonte é modificado.
