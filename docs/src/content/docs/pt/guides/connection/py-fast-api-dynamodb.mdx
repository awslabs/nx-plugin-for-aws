O gerador `connection` conecta um <Link path="guides/fastapi">FastAPI</Link> a um projeto <Link path="guides/py-dynamodb">Python DynamoDB</Link>, configurando o desenvolvimento local para que ambos iniciem juntos automaticamente.

## Pré-requisitos

Antes de usar este gerador, certifique-se de ter:

1. Um projeto <Link path="guides/fastapi">`py#fast-api`</Link>
2. Um projeto <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Uso

<GeneratorParameters generator="connection" />

## Saída do Gerador

O gerador atualiza o `project.json` do FastAPI para adicionar uma dependência de seu target `serve-local` ao target `serve-local` do projeto DynamoDB, e adiciona o pacote DynamoDB como uma dependência do workspace. Nenhum arquivo de código-fonte é modificado.

## Usando DynamoDB em Manipuladores de Rota

Importe classes de entidade do pacote DynamoDB e use-as dentro de seus manipuladores de rota:

```python title="packages/my_api/my_api/api.py"
from my_scope.my_table.entities.example import ExampleModel

@app.get("/examples")
def list_examples():
    return list(ExampleModel.scan())
```
