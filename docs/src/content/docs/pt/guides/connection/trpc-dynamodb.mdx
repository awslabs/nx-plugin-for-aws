O gerador `connection` conecta uma <Link path="guides/trpc">API tRPC</Link> a um projeto <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link>, configurando o desenvolvimento local para que ambos iniciem juntos automaticamente.

## Pré-requisitos

Antes de usar este gerador, certifique-se de ter:

1. Um projeto <Link path="guides/trpc">API tRPC</Link> (gerado com `ts#api`)
2. Um projeto <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link>

## Saída do Gerador

O gerador atualiza o `project.json` da sua API tRPC para adicionar uma dependência do seu target `serve-local` ao target `serve-local` do projeto DynamoDB. Nenhum arquivo de código-fonte é modificado.

## Usando DynamoDB em Procedures

Importe as factories de entidade do pacote DynamoDB e use-as dentro dos seus procedures tRPC:

```ts title="packages/api/src/procedures/example.ts"
import { createExampleEntity } from ':my-scope/my-table';
import { publicProcedure } from '../init.js';

export const listExamples = publicProcedure
  .query(async () => {
    const entity = await createExampleEntity();
    const result = await entity.scan.go();
    return result.data;
  });
```
