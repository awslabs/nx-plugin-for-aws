---
title: "Métricas de Uso"
description: "Como o @aws/nx-plugin coleta métricas"
---



Para nos ajudar a melhorar o Nx Plugin for AWS, coletamos métricas sobre o uso dos geradores. Esse processo não envolve solicitações de rede adicionais, utilizamos apenas as métricas de uso existentes da AWS.

Para manter a transparência, adicionamos um ID, uma versão e uma lista de IDs de geradores às descrições do CloudFormation Stack. Isso é feito no `MetricsAspect` em `common/constructs`.
