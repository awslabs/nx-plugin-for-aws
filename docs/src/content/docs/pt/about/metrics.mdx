---
title: "Métricas de Uso"
description: "Como o @aws/nx-plugin coleta métricas"
---



Para nos ajudar a melhorar o Nx Plugin for AWS, coletamos métricas sobre o uso dos geradores. Este processo não envolve solicitações de rede adicionais, simplesmente aproveitamos as métricas de uso existentes da AWS.

Para manter a transparência, anexamos um ID, uma versão e uma lista de IDs de geradores às descrições da Stack do CloudFormation. Isso é feito no `MetricsAspect` em `common/constructs`.
