[Claude Plugin](https://code.claude.com/docs/en/plugins)、[Kiro Power](https://kiro.dev/docs/powers/)、または[MCPサーバー](https://modelcontextprotocol.io/introduction)をインストールして、AIアシスタントがプラグインと効果的に連携できるようにします。これらを使用することで、AIによる開発ワークフローを加速しながら、一貫した方法でプロジェクトをスキャフォールディングできるため、主要コンポーネントの設定に費やす時間を削減し、プラグインに組み込まれたセキュリティ、可観測性、型安全性、ローカル開発の恩恵を受けることができます。

## AIアシスタントの設定

<Tabs syncKey="ai-assistant">
  <TabItem label="Kiro">
    Nx Plugin for AWSは[Kiro Power](https://kiro.dev/docs/powers/)として提供されており、MCPサーバーとステアリングドキュメント、ワークフローガイドがバンドルされています。

    <Steps>

    1. KiroのPowersパネルを開き、**Add power from GitHub**をクリック
    2. 以下のURLを入力：

       ```
       https://github.com/awslabs/nx-plugin-for-aws/tree/main/powers/nx-plugin-for-aws
       ```

    3. **Install**をクリック

    </Steps>

    Kiroは自動的にMCPサーバーを設定します — 追加のセットアップは不要です。
  </TabItem>
  <TabItem label="Kiro CLI">
    以下のコマンドを実行してください：

    ```bash
    kiro-cli mcp add \
      --name nx-plugin-for-aws \
      --scope global \
      --command npx \
      --args "-y,@aws/nx-plugin-mcp"
    ```

    詳細は[Kiro CLI MCPドキュメント](https://kiro.dev/docs/cli/mcp/)をご覧ください。
  </TabItem>
  <TabItem label="Claude Code">
    Nx Plugin for AWSは[Claude Codeプラグイン](https://code.claude.com/docs/en/plugins)として配布されており、[Skill](https://code.claude.com/docs/en/skills)とMCPサーバーがバンドルされています。

    ターミナルで以下を実行してください：

    ```bash
    npx plugins add awslabs/nx-plugin-for-aws
    ```

    その後、Claude Codeを再起動（または`/reload-plugins`を実行）してプラグインを有効化します。
  </TabItem>
  <TabItem label="Cursor">
    `~/.cursor/mcp.json`（またはプロジェクト内の`.cursor/mcp.json`）に以下を追加してください：

    <Snippet name="mcp/config" />

    詳細は[Cursor MCPドキュメント](https://cursor.com/docs/mcp)をご覧ください。

  <TabItem label="Codex">
    `~/.codex/config.toml`に以下を追加してください：


## Vibe Codingの開始

AIアシスタントに`nx-plugin-for-aws`を使用した構築を指示しましょう！通常、ワークスペースの作成、適用可能なジェネレーターによるスキャフォールディング、ビジネスロジックの実装という流れで作業を進めます。

次のようなプロンプトを試してみてください：

> *"Build a multi-agent application using the Nx Plugin for AWS. I want a React frontend that talks to a backend orchestrator agent which delegates to specialised research and writing agents."*

または、より小規模な出発点として：

> *"Use the Nx Plugin for AWS to scaffold a React website backed by a tRPC API, with Cognito auth and CDK infrastructure."*
