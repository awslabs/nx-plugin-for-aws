## タスク1: ストーリーエージェントの実装

ストーリーエージェントは、<Link path="get_started/tutorials/dungeon-game/1">モジュール1</Link>で`--protocol=AG-UI`を指定して生成された[Strands](https://strandsagents.com/)エージェントです。これにより、UIはCopilotKitを介して[Agent-User Interactionプロトコル](https://docs.copilotkit.ai/aws-strands/protocol)でストリーミングできます。インベントリMCPサーバーを使用してプレイヤーのアイテムを管理し、Strandsの組み込み[`S3SessionManager`](https://strandsagents.com/latest/documentation/docs/user-guide/concepts/agents/sessions/)を使用して、モジュール2でプロビジョニングしたセッションバケットに会話履歴を永続化します。

変更内容は以下の通りです：

- `main.py`は、デプロイ時に`thread_id`ごとに`S3SessionManager`を作成し、`agent-serve-local`（`SERVE_LOCAL=true`）で実行時には`/tmp/strands-sessions`配下のディスク上の`FileSessionManager`にフォールバックする`session_manager_provider`を追加します。デプロイ時、S3バケットはGame APIの`queryActions`が読み取るものと同じであるため、ブラウザは再訪問時にトランスクリプトを再構築できます。ローカルでは、エージェントはAWS呼び出しなしでローカルMCPサーバーに対して完全にオフラインで実行されます。
- `agent.py`は、サンプルの`subtract`ツールを削除し、システムプロンプトをダンジョンマスター用のものに置き換えます。このプロンプトは、最初のユーザーメッセージでプレイヤーの名前とジャンルを述べるよう促し、インベントリMCPサーバーのツールを使用します。

### エージェントとチャットする

生成された`agent-chat`ターゲットを使用して、ローカルで提供されるAG-UIエージェントのコピーに対する対話型REPLを開きます：

<NxCommands commands={["agent-chat story"]} />

`agent-chat`には`dependsOn: ['agent-serve-local']`が設定されているため、エージェントサーバーを別途起動する必要はありません。Nxが`agent-serve-local`を起動し（これによりInventory MCPサーバー、さらにはDynamoDB Localも起動されます）、その後[`agent-chat-cli`](https://www.npmjs.com/package/agent-chat-cli)を使用して`http://localhost:8081/invocations`に対するAG-UI REPLを起動します。エージェントスタック全体がAWS呼び出しなしでローカルで実行されます。

最初のメッセージでは、エージェントにヒーローの名前とジャンルを伝えてください（例：`My name is Alice. Start my zombie adventure.`）。ストーリーがストリーミングで返されます。

<Aside type="tip" title="ローカルセッション履歴">
`agent-serve-local`（`SERVE_LOCAL=true`）では、会話履歴はS3の代わりにStrandsの`FileSessionManager`を介して`/tmp/strands-sessions`に書き込まれます。後続の呼び出しで同じ`threadId`を再利用すると、同じストーリーが続きます。ファイルセッションマネージャーが以前のメッセージを再生します。
</Aside>

<Aside type="tip" title="生のcurl">
ローカルサーバーに対して生のHTTP呼び出しを希望する場合、AG-UIの`RunAgentInput`の形式は以下の通りです：
```bash
curl -N -X POST http://127.0.0.1:8081/invocations \
  -H "Content-Type: application/json" \
  -d '{
    "threadId": "Alice-zombie00000000000000000000000",
    "runId": "run-1",
    "messages": [{ "id": "m1", "role": "user", "content": "My name is Alice. Start my zombie adventure." }],
    "tools": [], "context": [], "state": {}, "forwardedProps": {}
  }'
```
</Aside>

コマンドが正常に実行されると、[AG-UIイベントストリーム](https://docs.copilotkit.ai/aws-strands/protocol/events)（Server-Sent Events）としてストーリーがストリーミングされ始めます。`RUN_STARTED` / `TEXT_MESSAGE_START` / `TEXT_MESSAGE_CONTENT`チャンクが実際のストーリートークンをラップします：

```
data: {"type":"RUN_STARTED","threadId":"Alice-zombie00000000000000000000000",...}
data: {"type":"TEXT_MESSAGE_START","messageId":"...","role":"assistant"}
data: {"type":"TEXT_MESSAGE_CONTENT","messageId":"...","delta":"Greetings, Alice. "}
data: {"type":"TEXT_MESSAGE_CONTENT","messageId":"...","delta":"The moans grow louder beyond the barricaded door..."}
```
