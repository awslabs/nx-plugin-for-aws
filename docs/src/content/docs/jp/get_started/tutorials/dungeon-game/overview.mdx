## 概要

このチュートリアルでは、AIを活用したダンジョン冒険ゲームを構築します。このチュートリアルは、`@aws/nx-plugin`や関連技術に関する既存の知識を前提としていません。このチュートリアルで学ぶ技術は、任意の`@aws/nx-plugin`ベースのアプリケーションを構築する上で基本的なものであり、完全に理解することで`@aws/nx-plugin`に関する深い理解と、[NX](https://nx.dev/)の基本的な使用方法を習得できます。

<Aside>このチュートリアルは、実践を通じて学ぶことを好み、すぐに具体的なものを作りたい人向けに設計されています。各概念を段階的に学びたい場合は、個々のコンポーネントガイドを参照してください</Aside>

このチュートリアルを完了したユーザーは、以下の知識を得ることができます：

- `@aws/nx-plugin`を使用して新しいアプリケーションを作成する
- NXを使用してコードベースを管理/構築する
- tRPCとFastAPIの両方を使用してAPIを構築する
- Tanstack routerを使用して新しいページを作成する
- Tanstack queryを使用してバックエンドAPIを呼び出す
- 生成されたコードを目的に合わせて変更する
- CDKインフラストラクチャを作成およびデプロイする

### アプリケーションのコンポーネント

AI搭載のダンジョン冒険ゲームは、以下のコンポーネントアーキテクチャを使用して構築されます：

<Image class="white-bg" src={dungeonAdventureArchitecturePng} alt="dungeon-game-architecture.png" width="800" height="600" />

- ゲーム状態の管理に[ElectroDB](https://electrodb.dev/)/[DynamoDB](https://aws.amazon.com/dynamodb/)を使用する[tRPC](https://trpc.io/) API
- ストーリー生成のためのストリーミングレスポンスを持つPython/[FastAPI](https://fastapi.tiangolo.com/) Lambda関数（[Amazon Bedrock](https://aws.amazon.com/bedrock/)を使用）
- React/Viteフロントエンドウェブサイト：
  - APIコールを安全に行うためのAmazon Cognito/Identity Pools
  - タイプセーフなファイルベースのルーティングをサポートする[Tanstack router](https://tanstack.com/router/latest)
  - ゲームとストーリーAPIを呼び出すための生成されたSDK（[Tanstack query](https://tanstack.com/query/latest)を活用）
- アプリケーションをデプロイするための[CDK](https://docs.aws.amazon.com/cdk/v2/guide/home.html)インフラストラクチャ
