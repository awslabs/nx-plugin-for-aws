## 概要

このチュートリアルでは、AIを活用したダンジョンアドベンチャーゲームを構築します。`@aws/nx-plugin`や関連技術の事前知識は必要ありません。ここで学ぶテクニックは`@aws/nx-plugin`ベースのアプリケーション開発の基礎となるもので、チュートリアルを完了することで以下の理解が深まります：

- `@aws/nx-plugin`の基本的な仕組み
- [NX](https://nx.dev/)の基本的な使用方法

<Aside>本チュートリアルは、実践を通じて学びたい方や具体的な成果物を素早く作りたい方向けに設計されています。各概念を段階的に学びたい場合は、個別のコンポーネントガイドを参照してください</Aside>

チュートリアル完了後、以下の知識を習得できます：

- `@aws/nx-plugin`を使用した新規アプリケーションの作成
- NXを使ったコードベースの管理/ビルド
- tRPCとFastAPIを使ったAPI構築
- Tanstack routerによるページ作成
- Tanstack queryを使ったバックエンドAPIの呼び出し
- 生成コードの目的に合わせた修正
- CDKインフラの作成とデプロイ

### アプリケーション構成

AIダンジョンアドベンチャーゲームは以下のコンポーネントアーキテクチャで構築されます：

<Image class="white-bg" src={dungeonAdventureArchitecturePng} alt="ゲームアーキテクチャ図" width="800" height="600" />

- ゲーム状態管理のための[ElectroDB](https://electrodb.dev/)/[DynamoDB](https://aws.amazon.com/dynamodb/)を使用する[tRPC](https://trpc.io/) API
- ストーリー生成用のPython/[FastAPI](https://fastapi.tiangolo.com/) Lambda関数（[Amazon Bedrock](https://aws.amazon.com/bedrock/)駆動）
- React/Viteフロントエンド：
  - 安全なAPI呼び出しのためのAmazon Cognito/Identity Pools
  - タイプセーフなファイルベースルーティングをサポートする[Tanstack router](https://tanstack.com/router/latest)
  - ゲームAPI/ストーリーAPI呼び出し用生成SDK（[Tanstack query](https://tanstack.com/query/latest)活用）
- アプリケーション展開用[CDK](https://docs.aws.amazon.com/cdk/v2/guide/home.html)インフラ
