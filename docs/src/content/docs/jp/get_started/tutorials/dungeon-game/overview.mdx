## 概要

このチュートリアルではAI駆動のダンジョンアドベンチャーゲームを構築します。`@aws/nx-plugin`や関連技術の事前知識は必要ありません。本チュートリアルで学ぶ技術は`@aws/nx-plugin`ベースのアプリケーション構築の基礎となるもので、完全に理解することで`@aws/nx-plugin`の深い知識と[NX](https://nx.dev/)の基本的な使用方法を習得できます。

<Aside>このチュートリアルは、実践を通じて学ぶことを好み、具体的なものを素早く作りたい人向けに設計されています。概念ごとにステップバイステップで学びたい場合は、左側のナビゲーションから個別のコンポーネントガイドを参照してください。</Aside>

チュートリアルを完了すると次のスキルが習得できます:

- `@aws/nx-plugin`を使用した新規アプリケーションの作成
- NXを使ったコードベースの管理/ビルド
- tRPCとFastAPIを使ったAPIの構築
- TanStack Routerを使ったページ作成
- TanStack Queryを使ったバックエンドAPIの呼び出し
- 生成コードの目的に合わせた修正
- CDKインフラの作成とデプロイ

### アプリケーション構成

以下のコンポーネントアーキテクチャを使用してAI駆動ダンジョンアドベンチャーゲームを構築します:

<Image class="white-bg" src={dungeonAdventureArchitecturePng} alt="dungeon-game-architecture.png" width="800" height="600" />

1. [tRPC](https://trpc.io/) APIはゲーム状態管理に[ElectroDB](https://electrodb.dev/)/[DynamoDB](https://aws.amazon.com/dynamodb/)を使用
2. ストーリー生成用のストリーミング応答付きPython/[FastAPI](https://fastapi.tiangolo.com/) Lambda関数（[Amazon Bedrock](https://aws.amazon.com/bedrock/)駆動）
3. React/Viteフロントエンドウェブサイト:
  - セキュアなAPI呼び出しにAmazon Cognito/Identity Poolsを採用
  - 型安全なファイルベースルーティングをサポートする[TanStack router](https://tanstack.com/router/latest)
  - Game/Story API呼び出し用生成SDK（[TanStack query](https://tanstack.com/query/latest)活用）

アプリケーションコンポーネントをデプロイする[CDK](https://docs.aws.amazon.com/cdk/v2/guide/home.html)インフラストラクチャ
