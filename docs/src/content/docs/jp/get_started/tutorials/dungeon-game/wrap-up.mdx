## まとめ

おめでとうございます！`@aws/nx-plugin`内にある多くの利用可能なジェネレーターを活用したダンジョンアドベンチャーゲームを作成しました。🎉🎉🎉

## 次のステップ

以下の機能でコードベースを自分自身で拡張してみることをお勧めします：

1. <Link path="guides/license">ライセンスジェネレーター</Link>をプロジェクトに追加して、ワークスペース内のLICENSEファイルとソースコードヘッダーを自動的に管理します。
2. <Link path="get_started/tutorials/create-generator">独自のジェネレーターを作成</Link>して、新しいtRPC APIの追加を簡素化します。
3. GameAPIに新しい`resetToAction` API（上記のジェネレーターを使用）を追加して、特定の時点に戻り、その時点からすべてのアクションを削除できるようにします。次に、UIを更新して会話バブルにボタンを追加し、このポイントにリセットできるようにします（そして新しく作成したAPIを呼び出します）。
4. 利用可能な様々なガイドを探索してみましょう。

### リソースの削除

作成されたAWSリソースを削除するには、次のコマンドを実行します：

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

これにより、削除するスタックのリストの確認が表示されます。これには`waf`と`dungeon-adventure-infra-sandbox`が含まれているはずです。`Y`を入力すると、Cloudformationがスタックを削除します。
