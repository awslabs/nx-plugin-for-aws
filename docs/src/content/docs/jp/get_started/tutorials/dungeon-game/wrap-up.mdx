## まとめ

おめでとうございます！`@aws/nx-plugin` で利用可能な多数のジェネレーターを活用したダンジョンアドベンチャーゲームを作成しました。🎉🎉🎉

## 次のステップ

以下の機能拡張に挑戦して、コードベースをさらに発展させることをお勧めします：

1. ワークスペースのLICENSEファイルとソースコードヘッダーを自動管理する<Link path="guides/license">ライセンスジェネレーター</Link>を追加
2. 新しいtRPC APIの追加を簡素化するために<Link path="get_started/tutorials/create-generator">独自のジェネレーターを作成</Link>
3. GameAPIに新しい`resetToAction` APIを追加（上記のジェネレーターを使用）し、特定の時点に戻ってその時点以降のアクションをすべて削除できる機能を実装。会話バブルにリセットボタンを追加して新しいAPIを呼び出すUIを更新
4. 利用可能な各種ガイドを探索

### リソースの削除

作成されたAWSリソースを削除するには、次のコマンドを実行してください：

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

このコマンドは削除するスタックのリスト（`waf`と`dungeon-adventure-infra-sandbox`）を確認要求します。`Y`を入力するとCloudFormationがスタックの削除を開始します。
