## まとめ

おめでとうございます！`@aws/nx-plugin`に含まれる様々なジェネレーターを活用したダンジョンアドベンチャーゲームを作成できました。🎉🎉🎉

## 次のステップ

以下の機能拡張に挑戦して、コードベースをさらに発展させてみましょう：

1. <Link path="guides/license">ライセンスジェネレーター</Link>をプロジェクトに追加し、ワークスペース内のLICENSEファイルとソースコードヘッダーを自動管理
2. <Link path="get_started/tutorials/contribute-generator">独自ジェネレーターの開発</Link>を行い、新しいtRPC APIの追加を簡素化
3. GameAPIに新しい`resetToAction` APIを追加（上記ジェネレーターを使用）し、特定の時点に戻ってアクションを削除する機能を実装。UIを更新して会話バブルにリセットボタンを追加し、新APIを呼び出す
4. 提供されている各種ガイドを探索

### リソースの削除

作成されたAWSリソースを削除するには、次のコマンドを実行します：

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

削除対象のスタックリスト（`waf`と`dungeon-adventure-infra-sandbox`）が表示されます。`Y`を入力するとCloudFormationがスタックの削除を開始します。
