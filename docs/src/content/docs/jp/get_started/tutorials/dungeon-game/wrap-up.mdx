## まとめ

おめでとうございます！`@aws/nx-plugin` に含まれる様々なジェネレータを活用したダンジョンアドベンチャーゲームを作成できました。🎉🎉🎉

## 次は何をしますか？

以下の機能拡張に挑戦して、コードベースをさらに発展させてみましょう：

1. <Link path="guides/license">ライセンスジェネレータ</Link>をプロジェクトに追加し、ワークスペース内のLICENSEファイルとソースコードヘッダーを自動管理
2. <Link path="get_started/tutorials/contribute-generator">独自のジェネレータを開発</Link>して、新しいtRPC APIの追加を簡素化
3. GameAPIに新しい`resetToAction` APIを追加（上記ジェネレータを使用）し、特定の時点に戻ってアクションを削除できる機能を実装。会話バブルにリセットボタンを追加してこのAPIを呼び出すUIを更新
4. 提供されている各種ガイドを探索

### リソースの削除

作成したAWSリソースを削除するには、次のコマンドを実行します：

<NxCommands commands={['run @dungeon-adventure/infra:destroy dungeon-adventure-infra-sandbox/*']} />

このコマンドは削除するスタックのリスト（`dungeon-adventure-infra-sandbox/Application/GameUI/waf` と `dungeon-adventure-infra-sandbox/Application`）を確認するプロンプトを表示します。`Y`を入力すると、CloudFormationがスタックの削除を開始します。
