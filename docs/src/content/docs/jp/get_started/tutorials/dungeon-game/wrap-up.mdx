## まとめ

おめでとうございます！`@aws/nx-plugin`に含まれる多数のジェネレータを活用したダンジョンアドベンチャーゲームを作成できました。🎉🎉🎉

## 次のステップ

以下の機能拡張に挑戦して、コードベースをさらに発展させることをお勧めします：

1. <Link path="guides/license">ライセンスジェネレータ</Link>を追加して、ワークスペース内のLICENSEファイルとソースコードヘッダーを自動管理
2. <Link path="get_started/tutorials/create-generator">独自ジェネレータの作成</Link>で新しいtRPC APIの追加を簡素化
3. GameAPIに新しい`resetToAction` APIを追加（上記ジェネレータを使用）し、特定の時点に戻ってアクションを削除できる機能を実装。会話バブルにリセットボタンを追加して新しいAPIを呼び出すUIを更新
4. 各種ガイドの探索

### リソースの削除

作成されたAWSリソースを削除するには、次のコマンドを実行してください：

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

このコマンドは削除対象のスタックリスト（`waf`と`dungeon-adventure-infra-sandbox`）を提示します。`Y`を入力するとCloudFormationがスタックの削除を開始します。
