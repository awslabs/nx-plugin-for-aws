## まとめ

おめでとうございます！`@aws/nx-plugin` 内にある多くのジェネレーターを活用したダンジョン冒険ゲームを作成しました。 🎉🎉🎉

## 次は何をしますか？

以下の機能で、コードベースを拡張することをお勧めします：

1. プロジェクトに<Link path="guides/license">ライセンスジェネレーター</Link>を追加し、ワークスペース内のLICENSEファイルとソースコードヘッダーを自動的に管理します。
2. 新しいtRPC APIの追加を簡素化するために、<Link path="get_started/tutorials/create-generator">独自のジェネレーターを作成</Link>します。
3. GameAPIに新しい`resetToAction` APIを追加し（上記のジェネレーターを使用）、特定の時点からすべてのアクションを削除できるようにします。次に、会話バブル上にボタンを追加してこの時点にリセットし、新しく作成されたAPIを呼び出すようにUIを更新します。
4. 利用可能な各種ガイドを探索する

### リソースの破棄

作成されたAWSリソースを破棄するには、次のコマンドを実行します：

<NxCommands commands={['run @dungeon-adventure/infra:destroy --all']} />

これにより、削除するスタックのリストが表示されます。`waf`と`dungeon-adventure-infra-sandbox`が含まれるはずです。`Y`を入力すると、Cloudformationがスタックを破棄します。
