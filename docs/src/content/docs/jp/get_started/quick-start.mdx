## ステップ1: 新しいNxワークスペースの初期化

任意のパッケージマネージャーでNxワークスペースを作成するコマンドを実行：

## ステップ3: クラウドリソースの定義とAWSへのデプロイ

`packages/infra/src/stacks/application-stack.ts`を開き、以下を追加：

```typescript
import * as cdk from 'aws-cdk-lib';
import { DemoApi, DemoWebsite, UserIdentity } from ':my-project/common-constructs';
import { Construct } from 'constructs';

export class ApplicationStack extends cdk.Stack {
  constructor(scope: Construct, id: string, props?: cdk.StackProps) {
    super(scope, id, props);

    const identity = new UserIdentity(this, 'identity');
    const api = new DemoApi(this, 'api', {
      integrations: DemoApi.defaultIntegrations(this).build(),
    });
    api.grantInvokeAccess(identity.identityPool.authenticatedRole);

    new DemoWebsite(this, 'website');
  }
}
```

フルスタックアプリケーションをデプロイするために必要なCDKコードはこれだけです。

### インフラのビルドとデプロイ

プロジェクトをビルドするコマンドを実行：

<NxCommands commands={['run-many --target build --all']} />

:::tip
リンターエラーが発生した場合、自動修正コマンドを実行できます：

<NxCommands commands={['run-many --target lint --configuration=fix --all']} />
:::

## ステップ4: フロントエンドのローカル実行

<Steps>
1. `runtime-config.json`ファイルを取得：

    <NxCommands commands={['run @demo/demo-website:load:runtime-config']} />

2. ローカルウェブサイトサーバーを起動

    <NxCommands commands={['run @demo/demo-website:serve']} />
</Steps>

ウェブサイトは`http://localhost:4200`でアクセス可能です。
