---
title: "前提条件"
---



- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)をインストールします。
- [Node >= 22](https://nodejs.org/en/download)をインストールします（ノードバージョンの管理には[NVM](https://github.com/nvm-sh/nvm)の使用を推奨します）
  - バージョンを確認するには、ターミナルウィンドウで `node --version` を実行します。
- [PNPM >= 10](https://pnpm.io/installation#using-npm)をインストールします（[Yarn >= 4](https://yarnpkg.com/getting-started/install)、[Bun >= 1](https://bun.sh/docs/installation)、または[NPM >= 10](https://nodejs.org/en/learn/getting-started/an-introduction-to-the-npm-package-manager)の使用も可能です）
  - バージョンを確認するには、ターミナルウィンドウで `pnpm --version`、`yarn --version`、`bun --version` または `npm --version` を実行します。
- [UV >= 0.5.29](https://docs.astral.sh/uv/getting-started/installation/)をインストールします
  1. Python 3.12.0をインストールするには、`uv python install 3.12.0` を実行します。
  2. インストールされたバージョンを確認するには `uv python list --only-installed` を実行します
- ターゲットのAWSアカウント（アプリケーションをデプロイする環境）に[AWSクレデンシャル](https://docs.aws.amazon.com/sdkref/latest/guide/access.html)を設定します。
- [VSCode](https://code.visualstudio.com/)を使用している場合は、[Nx Console VSCode Plugin](https://marketplace.visualstudio.com/items?itemName=nrwl.angular-console)のインストールを推奨します。
