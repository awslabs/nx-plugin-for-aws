---
title: PostgreSQL クエリログ
---

`postgresql` ログは [Aurora PostgreSQL](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraPostgreSQL.CloudWatch.html) 用にエクスポートされ、ログは DDL ステートメントのみに限定されています（`log_statement=ddl`）。そのため、各ステートメントが個別に送信される限り、ステートメントのパラメータ値はログに記録されません。`log_statement=ddl` は、複数ステートメントのバッチ（例：単一の `psql -c "a;b;c"` 呼び出し）の生のテキスト全体を、そのバッチ内のいずれかのステートメントが DDL である場合、同じバッチ内の DML 値を含めてそのまま記録します。
