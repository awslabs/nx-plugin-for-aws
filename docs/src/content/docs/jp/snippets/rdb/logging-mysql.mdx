---
title: MySQL クエリログ
---

[Aurora MySQL](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraMySQL.Integrating.CloudWatch.html) では `audit` と `error` ログがエクスポートされます。`general` と `slowquery` は、DML の値を含む完全なステートメントテキストをログに記録するため、意図的に除外されています。[Advanced Auditing](https://docs.aws.amazon.com/AmazonRDS/latest/AuroraUserGuide/AuroraMySQL.Auditing.html) は接続と DDL にスコープされており（`server_audit_events=CONNECT,QUERY_DDL`）、ステートメントパラメータの値はログに記録されません。
