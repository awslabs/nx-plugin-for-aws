[PDK Nag](https://aws.github.io/aws-pdk/developer_guides/pdk-nag/index.html)は[CDK Nag](https://github.com/cdklabs/cdk-nag)をラップし、[プロトタイプ構築向けの特定ルールセット](https://github.com/aws/aws-pdk/blob/mainline/packages/pdk-nag/src/packs/aws-prototyping-rules.ts)を提供します。

PDK Nagからの移行にはCDK Nagを直接使用してください。同じルールセットが必要な場合、[こちらのドキュメント](https://github.com/cdklabs/cdk-nag/blob/main/docs/NagPack.md)に従って独自の「パック」を作成できます。
