### 設定ファイル

MCPをサポートするほとんどのAIアシスタントは類似した設定方法を採用しています。MCPサーバーの詳細を記載した設定ファイルを作成または更新する必要があります：

```json
{
  "mcpServers": {
    "your-mcp-server": {
      "command": "uv",
      "args": [
        "run",
        "python",
        "-m",
        "my_module.mcp_server.stdio"
      ],
      "env": {
        "VIRTUAL_ENV": "/path/to/your/project/.venv"
      }
    }
  }
}
```

:::caution
サーバー接続時に`ENOENT uv`のようなエラーが発生する場合、ターミナルで`which uv`を実行して取得できる`uv`の絶対パスを指定する必要があるかもしれません。
:::
