---
title: ローカル DynamoDB の起動
---
import NxCommands from '@components/nx-commands.astro';

ジェネレーターは、[DynamoDB Local](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/DynamoDBLocal.html) インスタンスを起動してテーブルを作成する `serve-local` ターゲットを設定します：

<NxCommands commands={['run <project-name>:serve-local']} />

これにより自動的に以下が実行されます：
1. DynamoDB Local イメージをプル（`pull-image` ターゲット）
2. コンテナを起動
3. `config.json` で定義されたインデックスを持つローカルテーブルを作成
