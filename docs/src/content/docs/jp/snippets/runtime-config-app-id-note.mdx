---
import Link from '@components/link.astro';

:::note[ランタイム設定]
これは、実行時にAWS AppConfigから設定を取得するために`RUNTIME_CONFIG_APP_ID`環境変数に依存しています。このプラグインでビルドされたプロジェクト（API、Agent、MCPサーバー）には、この変数が自動的に設定されています。その他のプロジェクトの場合は、インフラストラクチャによってプロビジョニングされたAppConfigアプリケーションIDを使用して、ランタイム環境で`RUNTIME_CONFIG_APP_ID`が設定されていることを確認してください。詳細については、<Link path="guides/runtime-config">ランタイム設定ガイド</Link>を参照してください。
:::
