title: "RESTとHTTPの選択"
---



:::note
APIのデプロイには[REST API](https://docs.aws.amazon.com/apigateway/latest/developerguide/apigateway-rest-api.html)または[HTTP API](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api.html)のいずれかを選択できます。選択の参考には[API Gateway Developer Guide](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api-vs-rest.html)をご覧ください。
:::
