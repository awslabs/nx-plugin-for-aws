#### WAF

REST APIの場合、生成されたCDKコンストラクトは、デフォルトでAPIのデフォルトステージにAWS WAFv2 Web ACLを関連付けます。Web ACLはAWSマネージドデフォルトルールセット（`AWSManagedRulesCommonRuleSet`および`AWSManagedRulesKnownBadInputsRuleSet`）を使用し、OWASP Top 10を含む一般的なWebエクスプロイトに対する保護を提供します。

オプトアウトする場合（例えば、独自のWeb ACLをアタッチする場合）、`enableWaf`を`false`に設定してください：

```ts {4}
const api = new MyApi(this, 'MyApi', {
  integrations: MyApi.defaultIntegrations(this).build(),
  enableWaf: false,
});
```
