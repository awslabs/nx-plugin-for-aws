---
title: RDS Proxyを使用せずに接続する場合のSSL要件
---
import Infrastructure from '@components/infrastructure.astro';

接続ジェネレーターは、Dockerfileを更新してAmazon RDS CA バンドルを `/usr/local/share/ca-certificates/rds-bundle.crt` にインストールします。RDS Proxyを使用せずに接続する場合、Node.jsが証明書を信頼できるように `NODE_EXTRA_CA_CERTS` をそのパスに設定してください：

<Infrastructure>
<Fragment slot="cdk">

```ts title="packages/infra/src/stacks/application-stack.ts"
new MyMcpServer(this, 'MyMcpServer', {
  ...
  environmentVariables: {
    NODE_EXTRA_CA_CERTS: '/usr/local/share/ca-certificates/rds-bundle.crt',
  },
});
```

</Fragment>
<Fragment slot="terraform">

```hcl title="packages/infra/src/main.tf"
module "my_mcp_server" {
  ...
  environment_variables = {
    NODE_EXTRA_CA_CERTS = "/usr/local/share/ca-certificates/rds-bundle.crt"
  }
}
```

</Fragment>
</Infrastructure>

詳細については、[Amazon RDS SSL/TLSドキュメント](https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/UsingWithRDS.SSL.html)を参照してください。RDS Proxyを使用する場合は、`NODE_EXTRA_CA_CERTS` を設定する必要はありません。
