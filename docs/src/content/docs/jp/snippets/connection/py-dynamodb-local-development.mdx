---
title: Python DynamoDB ローカル開発
---

`connection` ジェネレーターは、プロジェクトの `serve-local` ターゲットが DynamoDB プロジェクトの `serve-local` ターゲットに依存するように設定します。`serve-local` を実行すると、DynamoDB Local がプロジェクトと一緒に自動的に起動します。

`SERVE_LOCAL=true` 環境変数は自動的に設定されるため、`is_local()` は `True` を返し、PynamoDB エンティティは AWS ではなくローカルの DynamoDB インスタンスに接続します。
