---
title: "必要な前提条件"
---



- [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git)
- [Node >= 22](https://nodejs.org/en/download) (ノードのバージョン管理には[NVM](https://github.com/nvm-sh/nvm)などの使用を推奨します)
  - `node --version` を実行して確認
- [PNPM >= 10](https://pnpm.io/installation#using-npm) (代替として[Yarn >= 4](https://yarnpkg.com/getting-started/install)、[Bun >= 1](https://bun.sh/docs/installation)、または[NPM >= 10](https://nodejs.org/en/learn/getting-started/an-introduction-to-the-npm-package-manager)も使用可能)
  - `pnpm --version`、`yarn --version`、`bun --version` または `npm --version` を実行して確認
- [UV >= 0.5.29](https://docs.astral.sh/uv/getting-started/installation/)
  1. 以下のコマンドを実行して Python 3.12 をインストール: `uv python install 3.12.0`
  2. `uv python list --only-installed` でインストール済みバージョンを確認
- ターゲットとなるAWSアカウント（アプリケーションをデプロイする環境）に設定された[AWS Credentials](https://docs.aws.amazon.com/sdkref/latest/guide/access.html)
