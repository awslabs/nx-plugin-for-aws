`ts#website` ジェネレーターは新しいウェブサイトアプリケーションを作成します。`framework` オプションを使用してフロントエンドフレームワークを選択できます — 現在 [React](https://react.dev/) がサポートされています。

生成されるプロジェクト構造、設定、使用方法の詳細については、フレームワーク固有のガイドを参照してください：

- <Link path="/guides/react-website">React Website</Link>
