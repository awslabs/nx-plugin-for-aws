`connection` ジェネレーターは、<Link path="guides/trpc">tRPC API</Link> を <Link path="guides/ts-dynamodb">DynamoDB</Link> プロジェクトに接続し、両方が自動的に一緒に起動するようにローカル開発を構成します。

## 前提条件

このジェネレーターを使用する前に、以下を確認してください：

1. <Link path="guides/trpc">tRPC API</Link> プロジェクト（`ts#api` で生成）
2. <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link> プロジェクト

## 使用方法

### ジェネレーターの実行

<RunGenerator generator="connection" />

tRPC API プロジェクトをソースとして選択し、DynamoDB プロジェクトをターゲットとして選択します。

### オプション

<GeneratorParameters generator="connection" />

## ジェネレーターの出力

ジェネレーターは、tRPC API の `project.json` を更新して、その `serve-local` ターゲットから DynamoDB プロジェクトの `serve-local` ターゲットへの依存関係を追加します。ソースファイルは変更されません。

## プロシージャでの DynamoDB の使用

DynamoDB パッケージからエンティティファクトリーをインポートし、tRPC プロシージャ内で使用します：

```ts title="packages/api/src/procedures/example.ts"
import { createExampleEntity } from ':my-scope/my-table';
import { publicProcedure } from '../init.js';

export const listExamples = publicProcedure
  .query(async () => {
    const entity = await createExampleEntity();
    const result = await entity.scan.go();
    return result.data;
  });
```
