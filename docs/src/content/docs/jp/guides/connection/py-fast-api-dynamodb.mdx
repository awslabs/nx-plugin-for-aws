`connection`ジェネレーターは、<Link path="guides/fastapi">FastAPI</Link>を<Link path="guides/py-dynamodb">Python DynamoDB</Link>プロジェクトに接続し、両方が自動的に一緒に起動するようにローカル開発環境を設定します。

## 前提条件

このジェネレーターを使用する前に、以下を用意してください：

1. <Link path="guides/fastapi">`py#fast-api`</Link>プロジェクト
2. <Link path="guides/py-dynamodb">`py#dynamodb`</Link>プロジェクト

## 使用方法

### ジェネレーターの実行

<RunGenerator generator="connection" />

FastAPIプロジェクトをソースとして、DynamoDBプロジェクトをターゲットとして選択してください。

### オプション

<GeneratorParameters generator="connection" />

## ジェネレーターの出力

ジェネレーターはFastAPIの`project.json`を更新し、その`serve-local`ターゲットからDynamoDBプロジェクトの`serve-local`ターゲットへの依存関係を追加します。ソースファイルは変更されません。
