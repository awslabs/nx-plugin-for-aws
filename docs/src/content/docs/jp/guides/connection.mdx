このジェネレータは、ウェブサイトがAPIを呼び出すなど、プロジェクト同士を接続するために使用されます。ソースプロジェクト（例：APIを呼び出すプロジェクト）とターゲットプロジェクト（例：APIプロジェクト）を選択するだけで、このジェネレータが2つのプロジェクトの統合を処理します。

### サポートされている接続

:::note[ランタイム設定]
接続ジェネレータは<Link path="guides/runtime-config">ランタイム設定</Link>を使用して、デプロイ時の値（API URL、Cognito設定、エージェントランタイムARNなど）を生成されたプロジェクトとコンポーネント間で実行時に渡し、相互に検出して接続できるようにします。
:::

:::tip[ローカル開発]
接続されたプロジェクトは、`serve` および `serve-local` ターゲットを使用してマシン上で実行できます。詳細については、<Link path="guides/local-development">ローカル開発</Link>ガイドを参照してください。
:::
