---
title: "使用状況メトリクス"
description: "@aws/nx-pluginがメトリクスを収集する方法"
---



Nx Plugin for AWSの改善に役立てるため、ジェネレーターの使用状況に関するメトリクスを収集しています。この処理では追加のネットワークリクエストを発生させず、既存のAWS使用状況メトリクスを活用しています。

透明性を保つため、CloudFormationスタックの説明文にID・バージョン・ジェネレーターIDリストを付加しています。この処理は、CDKの場合は`common/constructs`内の`MetricsAspect`で、Terraformの場合は`metrics.tf` CloudFormationスタックで実施されます。
