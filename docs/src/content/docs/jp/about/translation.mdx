# ドキュメント翻訳

このプロジェクトはAWS BedrockのHaiku 3.5モデルを使用して、ドキュメントを英語から複数言語へ自動翻訳します。翻訳システムは効率的で正確、かつ使いやすいように設計されています。

## 対応言語

現在以下の言語をサポートしています:

- 日本語 (jp)
- フランス語 (fr)
- スペイン語 (es)
- ドイツ語 (de)
- 中国語 (zh)
- 韓国語 (ko)

## 動作原理

翻訳システムは次のように動作します:

1. **h2見出しごとのドキュメント分割** - 効率的な処理と翻訳モデルへの適切なコンテキスト提供を実現
2. **Markdownフォーマットの保持** - 翻訳中にすべてのMarkdown構文、コードブロック、HTMLタグを保持
3. **Frontmatterの特別処理** - YAML frontmatterは構造を保持しつつ翻訳
4. **差分翻訳** - デフォルトで変更されたファイルのみを翻訳し、時間とリソースを節約

## ローカルでの翻訳実行

ドキュメントをローカルで翻訳するには`scripts/translate.ts`スクリプトを使用します:

```bash
# 変更されたファイルのみを日本語に翻訳（デフォルト）
./scripts/translate.ts

# 全ファイルを翻訳
./scripts/translate.ts --all

# 指定言語への翻訳
./scripts/translate.ts --languages jp,fr,es

# 実際の翻訳なしで処理内容を表示
./scripts/translate.ts --dry-run

# 詳細な出力を表示
./scripts/translate.ts --verbose
```

## GitHub ワークフロー

GitHubワークフローはプルリクエストで英語ドキュメントファイルに変更が加えられると自動的に翻訳を実行します。ワークフローは:

1. 英語ドキュメントファイルへの変更を検出
2. AWS Bedrockを使用して変更ファイルを翻訳
3. 翻訳結果をソースブランチにコミット
4. PRに翻訳ステータスを更新

### ワークフローの手動実行

GitHub Actionsタブから手動で翻訳ワークフローをトリガーできます。これは次の場合に有用です:

- 全ドキュメントの完全翻訳を実行
- 特定言語への翻訳
- 翻訳スクリプト変更後の翻訳更新

## AWS設定

翻訳システムはAWS BedrockのHaiku 3.5モデルを使用します。この機能を利用するには以下が必要です:

1. **AWS認証情報** - ローカル開発ではAWS CLIまたは環境変数で認証情報を設定
2. **IAMロール** - GitHub Actionsでは、OIDC認証とAWS Bedrockに必要な権限を持つIAMロールを設定

### 必要な権限

IAMロール/ユーザーには以下の権限が必要です:

```json
{
  "Version": "2012-10-17",
  "Statement": [
    {
      "Effect": "Allow",
      "Action": [
        "bedrock:InvokeModel"
      ],
      "Resource": [
        "arn:aws:bedrock:*::foundation-model/anthropic.claude-3-haiku-20240307-v1:0"
      ]
    }
  ]
}
```

## 翻訳品質

翻訳品質は概ね高い水準ですが、以下の点に注意してください:

- **技術用語** - 技術用語は英語のまま保持されるように設定
- **コードブロック** - コードブロックは元の形式を保持するため翻訳されない
- **コンテキスト認識** - 翻訳モデルはドキュメントのコンテキストを理解し、技術文書の翻訳を支援

## 翻訳のカスタマイズ

`scripts/translate.ts`ファイルを変更することで翻訳プロセスをカスタマイズ可能です。主なカスタマイズ例:

- 新規言語の追加対応
- 翻訳モデルの変更
- 翻訳用プロンプトの調整
- ドキュメント分割・処理方法の変更

## トラブルシューティング

翻訳プロセスで問題が発生した場合:

1. **AWS認証情報を確認** - 正しく設定されているか確認
2. **AWSリージョンを確認** - AWS Bedrockが利用可能なリージョンを使用
3. **詳細出力モードで実行** - `--verbose`フラグを使用して詳細ログを確認
4. **レート制限を確認** - 大規模な翻訳ジョブではAWS Bedrockのレート制限の影響を受ける可能性あり
