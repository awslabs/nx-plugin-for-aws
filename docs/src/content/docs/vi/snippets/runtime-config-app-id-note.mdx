import Link from '@components/link.astro';

:::note[Runtime config]
Điều này dựa vào biến môi trường `RUNTIME_CONFIG_APP_ID` để lấy cấu hình từ AWS AppConfig tại thời điểm chạy. Các dự án được xây dựng với plugin này (APIs, Agents, MCP servers) đã có biến này được cấu hình tự động. Đối với các dự án khác, hãy đảm bảo `RUNTIME_CONFIG_APP_ID` được thiết lập trong môi trường chạy với AppConfig application ID được cung cấp bởi cơ sở hạ tầng của bạn. Để biết thêm thông tin, xem <Link path="guides/runtime-config">hướng dẫn Runtime Configuration</Link>.
:::
