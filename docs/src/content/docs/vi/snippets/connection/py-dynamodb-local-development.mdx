---
title: Phát triển Python DynamoDB Local
---

Trình tạo `connection` cấu hình target `serve-local` của dự án để phụ thuộc vào target `serve-local` của dự án DynamoDB. DynamoDB Local sẽ tự động khởi động cùng với dự án của bạn khi chạy `serve-local`.

Biến môi trường `SERVE_LOCAL=true` được thiết lập tự động, do đó `is_local()` trả về `True` và các entity PynamoDB của bạn kết nối với instance DynamoDB local thay vì AWS.
