---
title: Yêu cầu SSL khi Kết nối không sử dụng RDS Proxy
---
import Infrastructure from '@components/infrastructure.astro';

Trình tạo kết nối cập nhật Dockerfile để cài đặt gói CA của Amazon RDS tại `/usr/local/share/ca-certificates/rds-bundle.crt`. Đặt `NODE_EXTRA_CA_CERTS` vào đường dẫn đó để Node.js tin tưởng chứng chỉ khi kết nối mà không sử dụng RDS Proxy:

<Infrastructure>
<Fragment slot="cdk">

```ts title="packages/infra/src/stacks/application-stack.ts"
new MyAgent(this, 'MyAgent', {
  ...
  environmentVariables: {
    NODE_EXTRA_CA_CERTS: '/usr/local/share/ca-certificates/rds-bundle.crt',
  },
});
```

</Fragment>
<Fragment slot="terraform">

```hcl title="packages/infra/src/main.tf"
module "my_agent" {
  ...
  environment_variables = {
    NODE_EXTRA_CA_CERTS = "/usr/local/share/ca-certificates/rds-bundle.crt"
  }
}
```

</Fragment>
</Infrastructure>

Để biết thêm chi tiết, xem [tài liệu Amazon RDS SSL/TLS](https://docs.aws.amazon.com/AmazonRDS/latest/UserGuide/UsingWithRDS.SSL.html). Khi sử dụng RDS Proxy, bạn không cần cấu hình `NODE_EXTRA_CA_CERTS`.
