---
title: "Lựa chọn giữa REST và HTTP"
---

:::note
Bạn có thể chọn giữa [REST API](https://docs.aws.amazon.com/apigateway/latest/developerguide/apigateway-rest-api.html) hoặc [HTTP API](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api.html) để triển khai API của mình, vui lòng tham khảo [Hướng dẫn Phát triển API Gateway](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api-vs-rest.html) để giúp bạn quyết định.
:::
