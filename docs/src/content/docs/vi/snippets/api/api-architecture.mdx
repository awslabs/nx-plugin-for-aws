```d2 inline=true
}

xray: X-Ray\n(Traces) {
  shape: image
  icon: /nx-plugin-for-aws/icons/aws/xray.svg
  near: bottom-right
}

client -> waf
waf -> apigw
apigw -> lambda
lambda -> cw
lambda -> xray
```

REST API bao gồm một [AWS WAFv2](https://docs.aws.amazon.com/waf/latest/developerguide/waf-chapter.html) Web ACL đặt trước API Gateway stage với bộ quy tắc mặc định do AWS quản lý được kích hoạt.
</TabItem>
