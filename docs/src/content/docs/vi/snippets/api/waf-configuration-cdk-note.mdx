title: Cấu hình WAF CDK
---

:::note[Cấu hình WAF cho REST API]
Đối với REST API, cấu trúc CDK được tạo ra sẽ liên kết một AWS WAFv2 Web ACL với stage mặc định của API theo mặc định. Web ACL sử dụng bộ quy tắc mặc định được quản lý bởi AWS (`AWSManagedRulesCommonRuleSet` và `AWSManagedRulesKnownBadInputsRuleSet`), cung cấp bảo vệ chống lại các lỗ hổng web phổ biến bao gồm OWASP Top 10.

Nếu bạn cần tắt tính năng này (ví dụ, để gắn Web ACL của riêng bạn), hãy đặt `enableWaf` thành `false`:

```ts {4}
const api = new MyApi(this, 'MyApi', {
  integrations: MyApi.defaultIntegrations(this).build(),
  // Disable the default WAF (not recommended for production)
  enableWaf: false,
});
```
:::
