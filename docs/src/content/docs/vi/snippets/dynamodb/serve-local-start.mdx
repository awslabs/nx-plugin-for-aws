title: Khởi động DynamoDB Local
---
import NxCommands from '@components/nx-commands.astro';

Generator cấu hình một target `serve-local` để khởi động một instance [DynamoDB Local](https://docs.aws.amazon.com/amazondynamodb/latest/developerguide/DynamoDBLocal.html) và tạo bảng. Sử dụng target `dev` của dự án (một alias cho `serve-local`):

<NxCommands commands={['run <project-name>:dev']} />

Điều này tự động:
1. Kéo image DynamoDB Local (target `pull-image`)
2. Khởi động một container
3. Tạo một bảng local với các index được định nghĩa trong `config.json`
