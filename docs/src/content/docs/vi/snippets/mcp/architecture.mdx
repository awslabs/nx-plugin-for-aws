<TabItem label="Bedrock AgentCore Runtime" _filter={{ infra: 'agentcore' }}>

Khi triển khai lên [Bedrock AgentCore Runtime](https://aws.amazon.com/bedrock/agentcore/), MCP server được xây dựng thành container image, đẩy lên Amazon ECR, và chạy trong AgentCore Runtime. Các trợ lý AI gọi đến AgentCore Runtime data plane endpoint, endpoint này chuyển tiếp các lời gọi `tools/*` và `resources/*` đến server của bạn qua [streamable HTTP transport](https://modelcontextprotocol.io/specification/draft/basic/transports#streamable-http).

```d2 inline=true
direction: right

assistant: AI Assistant {
  shape: image
  icon: /nx-plugin-for-aws/icons/aws/client.svg
}

ecr: ECR {
  shape: image
  icon: /nx-plugin-for-aws/icons/aws/ecr.svg
}

agentcore: MCP Server\n(AgentCore Runtime) {
  shape: image
  icon: /nx-plugin-for-aws/icons/aws/bedrock-agentcore.svg
}

cw: CloudWatch\n(Logs, Metrics) {
  shape: image
  icon: /nx-plugin-for-aws/icons/aws/cloudwatch.svg
}

assistant -> agentcore: Streamable\nHTTP
ecr -> agentcore: Container\nimage
agentcore -> cw
```
</TabItem>
<TabItem label="None (local only)" _filter={{ infra: 'none' }}>

Với `infra: None`, không có hạ tầng AWS nào được tạo ra. MCP server được cấu hình chỉ cho các giao thức truyền tải STDIO và HTTP cục bộ, và được sử dụng bởi các trợ lý AI chạy trên cùng một máy.

```d2 inline=true
direction: right

assistant: AI Assistant {
  shape: image
  icon: /nx-plugin-for-aws/icons/aws/client.svg
}

server: MCP Server\n(local process) {
  shape: rectangle
}

assistant -> server: STDIO\nor\nStreamable HTTP
```
</TabItem>
