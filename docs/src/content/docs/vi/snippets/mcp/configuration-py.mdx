### Tệp Cấu hình

Hầu hết các trợ lý AI hỗ trợ MCP sử dụng phương pháp cấu hình tương tự. Bạn sẽ cần tạo hoặc cập nhật tệp cấu hình với thông tin chi tiết về MCP server của mình:

```json
{
  "mcpServers": {
    "your-mcp-server": {
      "command": "uv",
      "args": [
        "run",
        "python",
        "-m",
        "my_module.mcp_server.stdio"
      ],
      "env": {
        "VIRTUAL_ENV": "/path/to/your/project/.venv"
      }
    }
  }
}
```

:::caution
Nếu bạn nhận được lỗi như `ENOENT uv` khi kết nối với server của mình, bạn có thể cần chỉ định đường dẫn đầy đủ đến `uv`, có thể lấy bằng cách chạy lệnh `which uv` trong terminal của bạn.
:::
