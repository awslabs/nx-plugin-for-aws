---
title: "Khả năng quan sát MCP"
---

MCP server của bạn được tự động cấu hình với khả năng quan sát bằng cách sử dụng [AWS Distro for Open Telemetry](https://aws.amazon.com/otel/) (ADOT), thông qua việc cấu hình auto-instrumentation trong `Dockerfile` của bạn.

Bạn có thể tìm thấy các trace trong CloudWatch AWS Console, bằng cách chọn "GenAI Observability" trong menu. Lưu ý rằng để các trace được hiển thị, bạn sẽ cần bật [Transaction Search](https://docs.aws.amazon.com/AmazonCloudWatch/latest/monitoring/CloudWatch-Transaction-Search.html).

Để biết thêm chi tiết, vui lòng tham khảo [tài liệu AgentCore về khả năng quan sát](https://docs.aws.amazon.com/bedrock-agentcore/latest/devguide/observability-configure.html).
