Generator này được sử dụng để kết nối các dự án với nhau, chẳng hạn như các trang web gọi API. Chỉ cần chọn dự án nguồn (ví dụ: dự án sẽ gọi API của bạn) và dự án đích (ví dụ: dự án API của bạn), và generator này sẽ xử lý việc tích hợp hai dự án đó.

### Các Kết Nối Được Hỗ Trợ

Generator Connection hỗ trợ các kết nối sau:

:::note[Cấu Hình Runtime]
Generator connection sử dụng <Link path="guides/runtime-config">Runtime Configuration</Link> để truyền các giá trị tại thời điểm triển khai (chẳng hạn như URL API, cài đặt Cognito và ARN runtime của agent) giữa các dự án và component được tạo ra tại runtime để chúng có thể khám phá và kết nối với nhau.
:::

:::tip[Phát Triển Cục Bộ]
Các dự án được kết nối có thể được chạy trên máy của bạn với các target `serve` và `serve-local`. Xem hướng dẫn <Link path="guides/local-development">Local Development</Link> để biết chi tiết.
:::
