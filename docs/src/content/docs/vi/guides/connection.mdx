Generator này được sử dụng để kết nối các dự án với API. Chỉ cần chọn dự án nguồn (dự án sẽ gọi API của bạn) và dự án đích (dự án API của bạn), và generator này sẽ xử lý việc tích hợp hai dự án đó.
