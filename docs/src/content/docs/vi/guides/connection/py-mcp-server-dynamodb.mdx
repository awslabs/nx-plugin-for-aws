Trình tạo `connection` kết nối một <Link path="guides/py-mcp-server">Python MCP Server</Link> với dự án <Link path="guides/py-dynamodb">Python DynamoDB</Link>, cấu hình phát triển cục bộ để cả hai khởi động cùng nhau tự động.

## Điều kiện tiên quyết

Trước khi sử dụng trình tạo này, hãy đảm bảo bạn có:

1. Một dự án <Link path="guides/py-mcp-server">`py#mcp-server`</Link>
2. Một dự án <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Cách sử dụng

### Chạy trình tạo

<RunGenerator generator="connection" />

Chọn dự án MCP server của bạn làm nguồn và dự án DynamoDB của bạn làm đích. Nếu dự án chứa nhiều thành phần MCP server, hãy chỉ định `sourceComponent` để phân biệt.

### Tùy chọn

<GeneratorParameters generator="connection" />

## Kết quả của trình tạo

Trình tạo cập nhật target `<mcp-server-name>-serve-local` của MCP server trong `project.json` để phụ thuộc vào target `serve-local` của dự án DynamoDB. Không có tệp nguồn nào được sửa đổi.
