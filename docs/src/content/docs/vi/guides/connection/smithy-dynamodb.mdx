Trình tạo `connection` kết nối một <Link path="guides/ts-smithy-api">Smithy API</Link> với dự án <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link>, cấu hình phát triển cục bộ để cả hai khởi động cùng nhau một cách tự động.

## Điều kiện tiên quyết

Trước khi sử dụng trình tạo này, hãy đảm bảo bạn có:

1. Một dự án <Link path="guides/ts-smithy-api">Smithy TypeScript API</Link> (được tạo bằng `ts#api` sử dụng `--framework=smithy`)
2. Một dự án <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link>

## Kết quả của trình tạo

Trình tạo cập nhật `project.json` của Smithy API để thêm phụ thuộc từ target `dev` của nó đến target `dev` của dự án DynamoDB. Không có tệp nguồn nào bị sửa đổi.

## Sử dụng DynamoDB trong các thao tác

Import các factory entity từ gói DynamoDB và sử dụng chúng bên trong các triển khai thao tác của bạn:

```ts title="packages/api/src/operations/list-examples.ts"
import { createExampleEntity } from ':my-scope/my-table';
import {
  ListExamplesOperationInput,
  ListExamplesOperationOutput,
} from '../generated/ssdk/index.js';
import { ServiceContext } from '../context.js';

export const listExamples = async (
  _input: ListExamplesOperationInput,
  _ctx: ServiceContext,
): Promise<ListExamplesOperationOutput> => {
  const entity = await createExampleEntity();
  const result = await entity.scan.go();
  return { items: result.data };
};
```
