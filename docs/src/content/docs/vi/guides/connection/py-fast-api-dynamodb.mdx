## Kết quả của Trình tạo

Trình tạo cập nhật `project.json` của FastAPI để thêm phụ thuộc từ target `serve-local` của nó đến target `serve-local` của dự án DynamoDB, và thêm gói DynamoDB làm phụ thuộc workspace. Không có tệp nguồn nào được sửa đổi.

## Sử dụng DynamoDB trong Route Handlers

Import các lớp entity từ gói DynamoDB và sử dụng chúng bên trong các route handler của bạn:

```python title="packages/my_api/my_api/api.py"
from my_scope.my_table.entities.example import ExampleModel

@app.get("/examples")
def list_examples():
    return list(ExampleModel.scan())
```
