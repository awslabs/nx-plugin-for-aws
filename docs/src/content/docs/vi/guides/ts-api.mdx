The TypeScript API generator creates a new API project with your choice of framework. It supports [tRPC](https://trpc.io/) for end-to-end type-safe APIs and [Smithy](https://smithy.io/) for model-driven API development.

## Frameworks

Choose a framework based on your needs:

- **tRPC** — End-to-end type-safe APIs with TypeScript. Ideal when your frontend and backend share a TypeScript codebase. See the <Link path="guides/trpc">tRPC guide</Link> for details.
- **Smithy** — Model-driven API development using an interface definition language. Ideal for APIs that need to be consumed by multiple languages or require formal API contracts. See the <Link path="guides/ts-smithy-api">Smithy TypeScript API guide</Link> for details.
