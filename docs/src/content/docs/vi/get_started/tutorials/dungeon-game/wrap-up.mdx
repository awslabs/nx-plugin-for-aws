## Kết thúc

Chúc mừng! Bạn đã tạo thành công một trò chơi phiêu lưu ngục tối Agentic AI sử dụng nhiều generator có sẵn trong `@aws/nx-plugin`.  🎉🎉🎉

## Tiếp theo là gì?

Chúng tôi khuyên bạn nên tự mình thử mở rộng codebase với các khả năng sau đây:

1. Thêm <Link path="guides/license">license generator</Link> vào dự án của bạn để tự động quản lý các tệp LICENSE và source code headers trong workspace của bạn.
2. <Link path="get_started/tutorials/contribute-generator">Đóng góp generator của riêng bạn</Link> để đơn giản hóa việc thêm các tRPC API mới.
3. Thêm một API `resetToAction` mới (sử dụng generator ở trên) trong GameAPI cho phép bạn quay ngược thời gian và xóa tất cả các hành động từ một điểm nhất định. Cập nhật UI để thêm một nút trên conversation bubble để reset về điểm này (và gọi API mới được tạo).
4. Khám phá các hướng dẫn về component khác nhau bằng cách sử dụng thanh điều hướng bên trái.

## Hủy tài nguyên

1. Để hủy các tài nguyên AWS đã được tạo, chạy lệnh sau:

<NxCommands commands={['destroy infra "dungeon-adventure-infra-sandbox/*"']} />

    Lệnh này sẽ nhắc bạn danh sách các stack cần xóa, bao gồm `dungeon-adventure-infra-sandbox/Application/GameUI/waf` và `dungeon-adventure-infra-sandbox/Application`.

2. Nhập `Y` để tiếp tục. CloudFormation sẽ hủy các stack của bạn.
