Khi kết thúc hướng dẫn, bạn sẽ có được kiến thức để:

- sử dụng `@aws/nx-plugin` để tạo các ứng dụng mới,
- sử dụng NX để quản lý/xây dựng codebase của họ,
- xây dựng API sử dụng tRPC,
- xây dựng Agent sử dụng Strands,
- xây dựng MCP server bằng TypeScript,
- sử dụng Tanstack router để tạo các trang mới,
- sử dụng Tanstack query để gọi các API backend, và
- tạo và triển khai cơ sở hạ tầng CDK.

### Kiến trúc ứng dụng

Trò chơi phiêu lưu hầm ngục được hỗ trợ bởi Agentic AI được xây dựng sử dụng kiến trúc sau:

<Image style="margin: auto" src={dungeonAdventureArchitecturePng} alt="dungeon-game-architecture.png" height="600" />

- Website frontend React/Vite sử dụng:
  - Amazon Cognito/Identity Pools cho các cuộc gọi API bảo mật.
  - [Tanstack router](https://tanstack.com/router/latest) hỗ trợ định tuyến dựa trên tệp với type-safe.
  - SDK được tạo tự động để gọi Game API.
- API [tRPC](https://trpc.io/) sử dụng [ElectroDB](https://electrodb.dev/)/[DynamoDB](https://aws.amazon.com/dynamodb/) để quản lý trạng thái trò chơi.
- Agent [Strands](https://strandsagents.com/latest/) được lưu trữ trên [Amazon Bedrock AgentCore](https://aws.amazon.com/bedrock/agentcore/) để chạy trò chơi.
- MCP Server TypeScript được lưu trữ trên [Amazon Bedrock AgentCore](https://aws.amazon.com/bedrock/agentcore/) để agent quản lý kho đồ của người chơi.
- Cơ sở hạ tầng [CDK](https://docs.aws.amazon.com/cdk/v2/guide/home.html) để triển khai ứng dụng.
