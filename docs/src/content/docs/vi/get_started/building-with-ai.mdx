Nx Plugin cho AWS bao gồm một [MCP Server](https://modelcontextprotocol.io/introduction) cho phép các trợ lý AI làm việc với Nx Plugin cho AWS. Bằng cách sử dụng MCP Server, bạn có thể tăng tốc quy trình phát triển của mình với AI, đồng thời hưởng lợi từ việc các dự án của bạn được tạo khung một cách xác định, tiết kiệm thời gian và ít tốn cửa sổ ngữ cảnh hơn cho việc thiết lập các thành phần chính.

Sử dụng MCP Server với Trợ lý AI mà bạn lựa chọn, chẳng hạn như Amazon Q Developer, Cline, Claude Code hoặc Cursor.

## Cấu hình MCP Server

Hầu hết các Trợ lý AI sẽ có một file JSON để cấu hình MCP server. Trong Amazon Q Developer, file này nằm ở `~/.aws/amazonq/mcp.json`. Thêm một mục cho Nx Plugin cho AWS MCP Server

## Bắt đầu Vibe Coding

Yêu cầu Trợ lý AI của bạn xây dựng thứ gì đó bằng cách sử dụng `aws-nx-mcp` MCP Server! Thông thường các Trợ lý AI sẽ bắt đầu bằng cách tạo một workspace, tạo khung với các generator phù hợp, và sau đó điền vào logic nghiệp vụ.
