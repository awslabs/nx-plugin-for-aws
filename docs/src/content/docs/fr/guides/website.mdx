Le générateur `ts#website` crée une nouvelle application de site web. Vous pouvez choisir quel framework frontend utiliser via l'option `framework` — actuellement [React](https://react.dev/) est supporté.

Pour des informations détaillées sur la structure du projet généré, la configuration et l'utilisation, consultez le guide spécifique au framework :

- <Link path="/guides/react-website">React Website</Link>
