Le générateur TypeScript API crée un nouveau projet d'API avec le framework de votre choix. Il prend en charge [tRPC](https://trpc.io/) pour les API type-safe de bout en bout et [Smithy](https://smithy.io/) pour le développement d'API piloté par modèle.

## Utilisation

### Générer une API TypeScript

Vous pouvez générer une nouvelle API TypeScript de deux façons :
