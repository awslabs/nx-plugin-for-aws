Ce générateur permet de connecter des projets entre eux, comme des sites web appelant des API. Sélectionnez simplement le projet source (par exemple le projet qui appellera votre API) et le projet cible (par exemple votre projet d'API), et ce générateur se chargera d'intégrer les deux.

### Connexions prises en charge

Le générateur de connexion prend en charge les connexions suivantes :

:::note[Configuration d'exécution]
Le générateur de connexion utilise la <Link path="guides/runtime-config">Configuration d'exécution</Link> pour transmettre des valeurs au moment du déploiement (telles que les URL d'API, les paramètres Cognito et les ARN d'exécution d'agent) entre les projets et composants générés au moment de l'exécution afin qu'ils puissent se découvrir et se connecter les uns aux autres.
:::
