Le générateur `connection` relie une <Link path="guides/ts-smithy-api">API Smithy</Link> à un projet <Link path="guides/ts-dynamodb">TypeScript DynamoDB</Link>, en configurant le développement local pour que les deux démarrent ensemble automatiquement.

## Prérequis

Avant d'utiliser ce générateur, assurez-vous d'avoir :

1. Un projet <Link path="guides/ts-smithy-api">Smithy TypeScript API</Link> (généré avec `ts#api` en utilisant `--framework=smithy`)
2. Un projet <Link path="guides/ts-dynamodb">`ts#dynamodb`</Link>

## Utilisation

### Exécuter le générateur

<RunGenerator generator="connection" />

<GeneratorParameters generator="connection" />

## Sortie du générateur

Le générateur met à jour le fichier `project.json` de votre API Smithy pour ajouter une dépendance de sa cible `serve-local` vers la cible `serve-local` du projet DynamoDB. Aucun fichier source n'est modifié.

## Utiliser DynamoDB dans les opérations

Importez les factories d'entités depuis le package DynamoDB et utilisez-les dans vos implémentations d'opérations :

```ts title="packages/api/src/operations/list-examples.ts"
import { createExampleEntity } from ':my-scope/my-table';
import {
  ListExamplesOperationInput,
  ListExamplesOperationOutput,
} from '../generated/ssdk/index.js';
import { ServiceContext } from '../context.js';

export const listExamples = async (
  _input: ListExamplesOperationInput,
  _ctx: ServiceContext,
): Promise<ListExamplesOperationOutput> => {
  const entity = await createExampleEntity();
  const result = await entity.scan.go();
  return { items: result.data };
};
```
