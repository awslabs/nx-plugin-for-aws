Le générateur `connection` relie une <Link path="guides/fastapi">FastAPI</Link> à un projet <Link path="guides/py-dynamodb">Python DynamoDB</Link>, en configurant le développement local pour que les deux démarrent ensemble automatiquement.

## Prérequis

Avant d'utiliser ce générateur, assurez-vous d'avoir :

1. Un projet <Link path="guides/fastapi">`py#api`</Link>
2. Un projet <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Utilisation

### Exécuter le générateur

<RunGenerator generator="connection" />

Sélectionnez votre projet FastAPI comme source et votre projet DynamoDB comme cible.

### Options

<GeneratorParameters generator="connection" />

## Sortie du générateur

Le générateur met à jour le fichier `project.json` de la FastAPI pour ajouter une dépendance de sa cible `dev` vers la cible `dev` du projet DynamoDB, et ajoute le package DynamoDB en tant que dépendance de workspace. Aucun fichier source n'est modifié.

## Utilisation de DynamoDB dans les gestionnaires de routes

Importez les classes d'entités du package DynamoDB et utilisez-les dans vos gestionnaires de routes :

```python title="packages/my_api/my_api/api.py"
from my_scope.my_table.entities.example import ExampleModel

@app.get("/examples")
def list_examples():
    return list(ExampleModel.scan())
```
