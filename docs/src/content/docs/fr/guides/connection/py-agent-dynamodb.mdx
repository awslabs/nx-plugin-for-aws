Le générateur `connection` relie un <Link path="guides/py-agent">Python Agent</Link> à un projet <Link path="guides/py-dynamodb">Python DynamoDB</Link>, en configurant le développement local pour que les deux démarrent ensemble automatiquement.

## Prérequis

Avant d'utiliser ce générateur, assurez-vous d'avoir :

1. Un projet <Link path="guides/py-agent">`py#agent`</Link>
2. Un projet <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Utilisation

### Exécuter le générateur

<RunGenerator generator="connection" />

Sélectionnez votre projet Agent comme source et votre projet DynamoDB comme cible. Si le projet contient plusieurs composants agent, spécifiez `sourceComponent` pour lever l'ambiguïté.

### Options

<GeneratorParameters generator="connection" />

## Sortie du générateur

Le générateur met à jour la cible `<agent-name>-serve-local` de l'agent dans `project.json` pour dépendre de la cible `serve-local` du projet DynamoDB, et ajoute le package DynamoDB comme dépendance de workspace. Aucun fichier source n'est modifié.

## Utilisation de DynamoDB dans les Agents

Importez les classes d'entités depuis le package DynamoDB et utilisez-les dans vos outils d'agent :

```python title="packages/my_project/my_project/my_agent/agent.py"
from my_scope.my_table.entities.example import ExampleModel
from strands import tool

@tool
def list_examples() -> list:
    """List all example items."""
    return [item.attribute_values for item in ExampleModel.scan()]
```

## Infrastructure

Pour permettre à la fonction Lambda de l'agent d'accéder à la table DynamoDB, accordez les permissions nécessaires dans votre infrastructure.

<Infrastructure>
<Fragment slot="cdk">

```ts title="packages/infra/src/stacks/application-stack.ts"
import { MyTable } from ':my-scope/common-constructs';

const table = new MyTable(this, 'Table');
const myAgent = new MyAgent(this, 'MyAgent');

table.grantReadWriteData(myAgent);
```

`grantReadWriteData` accorde à la fois les permissions DynamoDB et KMS au rôle d'exécution de l'agent.
</Fragment>
<Fragment slot="terraform">

```hcl title="packages/infra/src/main.tf"
module "my_table" {
  source = "../../common/terraform/src/app/dynamodb/my-table"
}

resource "aws_iam_role_policy" "dynamodb_access" {
  role = module.my_agent.lambda_role_name

  policy = jsonencode({
    Version = "2012-10-17"
    Statement = [
      {
        Effect = "Allow"
        Action = [
          "dynamodb:GetItem", "dynamodb:PutItem", "dynamodb:UpdateItem",
          "dynamodb:DeleteItem", "dynamodb:Query", "dynamodb:Scan",
          "dynamodb:BatchGetItem", "dynamodb:BatchWriteItem",
        ]
        Resource = [
          module.my_table.table_arn,
          "${module.my_table.table_arn}/index/*",
        ]
      },
      {
        Effect = "Allow"
        Action = [
          "kms:Encrypt",
          "kms:Decrypt",
          "kms:ReEncrypt*",
          "kms:GenerateDataKey*",
          "kms:DescribeKey"
        ]
        Resource = [module.my_table.kms_key_arn]
      },
    ]
  })
}
```
