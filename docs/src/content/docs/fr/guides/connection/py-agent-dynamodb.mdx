Le générateur `connection` relie un <Link path="guides/py-agent">Python Agent</Link> à un projet <Link path="guides/py-dynamodb">Python DynamoDB</Link>, en configurant le développement local pour que les deux démarrent ensemble automatiquement.

## Prérequis

Avant d'utiliser ce générateur, assurez-vous d'avoir :

1. Un projet <Link path="guides/py-agent">`py#agent`</Link>
2. Un projet <Link path="guides/py-dynamodb">`py#dynamodb`</Link>

## Utilisation

### Exécuter le générateur

<RunGenerator generator="connection" />

Sélectionnez votre projet Agent comme source et votre projet DynamoDB comme cible. Si le projet contient plusieurs composants agent, spécifiez `sourceComponent` pour lever l'ambiguïté.

### Options

<GeneratorParameters generator="connection" />

## Sortie du générateur

Le générateur met à jour la cible `<agent-name>-serve-local` de l'agent dans `project.json` pour dépendre de la cible `serve-local` du projet DynamoDB. Aucun fichier source n'est modifié.
