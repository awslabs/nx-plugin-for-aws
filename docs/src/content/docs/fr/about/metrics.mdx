---
title: "Métriques d'utilisation"
description: "Comment @aws/nx-plugin collecte les métriques"
---



Pour nous aider à améliorer le Nx Plugin pour AWS, nous collectons des métriques sur l'utilisation des générateurs. Ce processus ne nécessite aucune requête réseau supplémentaire - nous exploitons simplement les métriques d'utilisation existantes d'AWS.

Pour garantir la transparence, nous ajoutons un ID, une version et une liste d'IDs de générateurs aux descriptions des piles CloudFormation. Cette opération est effectuée dans le `MetricsAspect` présent dans `common/constructs`.
