---
title: "Métriques d'utilisation"
description: "Comment @aws/nx-plugin collecte les métriques"
---



Afin de nous aider à améliorer le Nx Plugin for AWS, nous collectons des métriques sur l'utilisation des générateurs. Ce processus n'implique aucune requête réseau supplémentaire - nous utilisons simplement les métriques d'utilisation existantes d'AWS.

Pour garantir la transparence, nous ajoutons un ID, une version et une liste d'IDs de générateurs dans les descriptions des piles CloudFormation. Cette opération est réalisée via le `MetricsAspect` dans `common/constructs` pour CDK, ou via la pile CloudFormation `metrics.tf` pour Terraform.
