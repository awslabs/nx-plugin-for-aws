## Récapitulatif

Félicitations ! Vous avez créé un jeu d'aventure dans un donjon avec IA agentique qui utilise de nombreux générateurs disponibles dans le `@aws/nx-plugin`. 🎉🎉🎉

## Prochaines étapes ?

Nous vous recommandons d'essayer d'étendre vous-même la base de code avec les fonctionnalités suivantes :

1. Ajoutez le <Link path="guides/license">générateur de licence</Link> à votre projet pour gérer automatiquement les fichiers LICENSE et les en-têtes de code source dans votre espace de travail.
2. <Link path="get_started/tutorials/contribute-generator">Contribuez à votre propre générateur</Link> pour simplifier l'ajout de nouvelles API tRPC.
3. Ajoutez une nouvelle API `resetToAction` (en utilisant votre générateur ci-dessus) dans GameAPI permettant de revenir en arrière et supprimer toutes les actions à partir d'un point donné. Mettez à jour l'UI pour ajouter un bouton sur une bulle de conversation permettant de réinitialiser à ce point (en appelant la nouvelle API créée).
4. Explorez les différents guides de composants en utilisant la navigation de gauche.

## Destruction des ressources

1. Pour détruire les ressources AWS créées, exécutez la commande suivante :

<NxCommands commands={['destroy infra "dungeon-adventure-infra-sandbox/*"']} />

    Cette commande vous demandera la liste des piles à supprimer, qui doivent inclure `dungeon-adventure-infra-sandbox/Application/GameUI/waf` et `dungeon-adventure-infra-sandbox/Application`.

2. Saisissez `Y` pour continuer. CloudFormation détruira vos piles.
