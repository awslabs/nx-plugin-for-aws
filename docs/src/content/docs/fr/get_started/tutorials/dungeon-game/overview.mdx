## Aperçu

Au cours de ce tutoriel, vous allez construire un jeu d'aventure donjon alimenté par l'IA. Ce tutoriel ne nécessite aucune connaissance préalable du `@aws/nx-plugin` ou des technologies associées. Les techniques que vous apprendrez ici sont fondamentales pour construire n'importe quelle application basée sur `@aws/nx-plugin`, et leur maîtrise vous donnera une compréhension approfondie du `@aws/nx-plugin` ainsi qu'une base solide pour utiliser [NX](https://nx.dev/).

<Aside>Ce tutoriel est conçu pour ceux qui préfèrent apprendre en pratiquant et veulent rapidement créer quelque chose de concret. Si vous préférez apprendre chaque concept étape par étape, reportez-vous aux guides des composants individuels.</Aside>

À l'issue de ce tutoriel, vous serez capable de :

- utiliser le `@aws/nx-plugin` pour créer de nouvelles applications
- utiliser NX pour gérer/construire votre base de code
- construire des APIs avec tRPC et FastAPI
- utiliser Tanstack router pour créer de nouvelles pages
- utiliser Tanstack query pour appeler des APIs backend
- modifier du code généré pour l'adapter à vos besoins
- créer et déployer une infrastructure CDK

### Composants de l'application

Le jeu sera construit avec l'architecture suivante :

<Image class="white-bg" src={dungeonAdventureArchitecturePng} alt="architecture du jeu donjon" width="800" height="600" />

- API [tRPC](https://trpc.io/) utilisant [ElectroDB](https://electrodb.dev/)/[DynamoDB](https://aws.amazon.com/dynamodb/) pour la gestion de l'état du jeu
- Fonction Lambda Python/[FastAPI](https://fastapi.tiangolo.com/) avec réponses en streaming pour la génération d'histoire (alimentée par [Amazon Bedrock](https://aws.amazon.com/bedrock/))
- Frontend React/Vite utilisant :
  - Amazon Cognito/Identity Pools pour les appels API sécurisés
  - [Tanstack router](https://tanstack.com/router/latest) avec routage type-safe basé sur les fichiers
  - SDKs générés pour appeler les APIs de jeu et d'histoire (utilisant [Tanstack query](https://tanstack.com/query/latest))
- Infrastructure [CDK](https://docs.aws.amazon.com/cdk/v2/guide/home.html) pour le déploiement
