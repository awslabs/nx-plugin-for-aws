### Fichiers de configuration

La plupart des assistants IA compatibles avec MCP utilisent une approche de configuration similaire. Vous devrez créer ou mettre à jour un fichier de configuration avec les détails de votre serveur MCP :

```json
{
  "mcpServers": {
    "your-mcp-server": {
      "command": "uv",
      "args": [
        "run",
        "python",
        "-m",
        "my_module.mcp_server.stdio"
      ],
      "env": {
        "VIRTUAL_ENV": "/path/to/your/project/.venv"
      }
    }
  }
}
```

:::caution
Si vous rencontrez une erreur comme `ENOENT uv` lors de la connexion à votre serveur, vous devrez peut-être spécifier le chemin complet vers `uv`, que vous pouvez obtenir en exécutant `which uv` dans votre terminal.
:::
