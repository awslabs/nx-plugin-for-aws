import NxCommands from '@components/nx-commands.astro';

Le générateur configure automatiquement une cible `bundle` qui utilise [Rolldown](https://rolldown.rs/) pour créer un package de déploiement :

<NxCommands commands={['run <project-name>:bundle']} />

La configuration de Rolldown se trouve dans `rolldown.config.ts`, avec une entrée par bundle à générer. Rolldown gère la création de plusieurs bundles en parallèle si définis.

:::tip
Vous pouvez ajuster la configuration de Rolldown dans `rolldown.config.ts`, par exemple si vous avez des dépendances qui reposent sur des chemins de fichiers et ne peuvent pas être regroupées, vous pouvez les ajouter à la liste des modules `external`.
:::
