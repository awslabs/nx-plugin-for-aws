Ce générateur fournit de l'infrastructure as code basée sur votre `iacProvider` choisi. Il créera un projet dans `packages/common` qui inclut les constructions CDK ou modules Terraform pertinents.

Le projet commun d'infrastructure as code est structuré comme suit :

<Infrastructure>
<Fragment slot="cdk">
<FileTree>
  - packages/common/constructs
    - src
      - app/ Constructions pour l'infrastructure spécifique à un projet/générateur
      - core/ Constructions génériques réutilisées par celles dans `app`
      - index.ts Point d'entrée exportant les constructions depuis `app`
    - project.json Cibles de build et configuration du projet
</FileTree>

:::note
Ce projet est généré avec le générateur <Link path="/guides/typescript-project">`ts#project`</Link> et configure donc les mêmes cibles de build.
:::
</Fragment>
<Fragment slot="terraform">
<FileTree>
  - packages/common/terraform
    - src
      - app/ Modules Terraform pour l'infrastructure spécifique à un projet/générateur
      - core/ Modules génériques réutilisés par ceux dans `app`
    - project.json Cibles de build et configuration du projet
</FileTree>

:::note
Ce projet est généré avec le générateur <Link path="/guides/terraform-project">`terraform#project`</Link> et configure donc les mêmes cibles de build.
:::
</Fragment>
</Infrastructure>
