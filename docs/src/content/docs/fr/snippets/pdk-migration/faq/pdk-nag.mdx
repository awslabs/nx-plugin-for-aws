[PDK Nag](https://aws.github.io/aws-pdk/developer_guides/pdk-nag/index.html) encapsule [CDK Nag](https://github.com/cdklabs/cdk-nag) et fournit un ensemble de [règles spécifiques à la création de prototypes](https://github.com/aws/aws-pdk/blob/mainline/packages/pdk-nag/src/packs/aws-prototyping-rules.ts).

Pour migrer depuis PDK Nag, utilisez CDK Nag directement. Si vous avez besoin du même ensemble de règles, vous pouvez créer votre propre "pack" en suivant la [documentation disponible ici](https://github.com/cdklabs/cdk-nag/blob/main/docs/NagPack.md).
