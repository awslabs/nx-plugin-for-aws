[AWS Arch](https://aws.github.io/aws-pdk/developer_guides/aws-arch/index.html) fournit des correspondances entre les ressources CloudFormation et leurs icônes d'architecture associées pour CDK Graph.

Consultez la [page des icônes d'architecture AWS](https://aws.amazon.com/architecture/icons/) pour les ressources relatives aux icônes. [Diagrams](https://diagrams.mingrammer.com/) propose également une méthode pour créer des diagrammes en code.

Si vous utilisiez cela directement, envisagez de forker le projet et de vous en approprier la gestion !
