---
title: "Choisir entre REST et HTTP"
---



:::note
Vous pouvez choisir entre une [REST API](https://docs.aws.amazon.com/apigateway/latest/developerguide/apigateway-rest-api.html) ou une [HTTP API](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api.html) pour déployer votre API. Veuillez consulter le [API Gateway Developer Guide](https://docs.aws.amazon.com/apigateway/latest/developerguide/http-api-vs-rest.html) pour vous aider à décider.
:::
