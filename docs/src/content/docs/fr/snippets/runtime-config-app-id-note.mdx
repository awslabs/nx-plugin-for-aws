:::note[Configuration d'exécution]
Cela repose sur la variable d'environnement `RUNTIME_CONFIG_APP_ID` pour récupérer la configuration depuis AWS AppConfig au moment de l'exécution. Les projets construits avec ce plugin (APIs, Agents, serveurs MCP) ont déjà cette variable configurée automatiquement. Pour les autres projets, assurez-vous que `RUNTIME_CONFIG_APP_ID` est définie dans l'environnement d'exécution avec l'ID d'application AppConfig provisionné par votre infrastructure. Pour plus d'informations, consultez le <Link path="guides/runtime-config">guide de configuration d'exécution</Link>.
:::
