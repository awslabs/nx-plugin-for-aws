---
title: Exigences SSL lors de la connexion sans RDS Proxy
---

Le magasin de confiance CA intégré de l'environnement d'exécution Lambda Amazon Linux 2023 inclut les CA racine Amazon utilisés par RDS, donc aucune configuration supplémentaire n'est nécessaire.

Lors de l'utilisation de RDS Proxy, vous n'avez pas besoin de configurer le bundle CA RDS dans votre fonction Lambda.
