---
title: Développement Local Python DynamoDB
---

Le générateur `connection` configure la cible `serve-local` de votre projet pour dépendre de la cible `serve-local` du projet DynamoDB. DynamoDB Local démarrera automatiquement en même temps que votre projet lors de l'exécution de `serve-local`.

La variable d'environnement `SERVE_LOCAL=true` est définie automatiquement, donc `is_local()` retourne `True` et vos entités PynamoDB se connectent à l'instance DynamoDB locale au lieu d'AWS.
