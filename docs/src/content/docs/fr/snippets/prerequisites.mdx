---
title: "Conditions préalables"
---



- Installez [Git](https://git-scm.com/book/en/v2/Getting-Started-Installing-Git).
- Installez [Node >= 22](https://nodejs.org/en/download) (Nous recommandons d'utiliser [NVM](https://github.com/nvm-sh/nvm) pour gérer vos versions de Node)
  - Pour vérifier votre version, exécutez `node --version` dans la fenêtre de terminal.
- Installez [PNPM >= 10](https://pnpm.io/installation#using-npm) (Vous pouvez également utiliser [Yarn >= 4](https://yarnpkg.com/getting-started/install), [Bun >= 1](https://bun.sh/docs/installation) ou [NPM >= 10](https://nodejs.org/en/learn/getting-started/an-introduction-to-the-npm-package-manager))
  - Pour vérifier votre version, exécutez `pnpm --version`, `yarn --version`, `bun --version` ou `npm --version` dans votre fenêtre de terminal.
- Installez [UV >= 0.5.29](https://docs.astral.sh/uv/getting-started/installation/)
  1. Pour installer Python 3.12, exécutez `uv python install 3.12.0`.
  2. Vérifiez avec `uv python list --only-installed`
- Configurez vos [AWS Credentials](https://docs.aws.amazon.com/sdkref/latest/guide/access.html) sur votre compte AWS cible (où votre application sera déployée).
- Si vous utilisez [VSCode](https://code.visualstudio.com/), nous vous recommandons d'installer le [Nx Console VSCode Plugin](https://marketplace.visualstudio.com/items?itemName=nrwl.angular-console).
